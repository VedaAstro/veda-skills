# LLM PLAYBOOK — РЕЕСТР ПРУФОВ

**Версия:** 2.0
**Дата:** 2026-02-17
**Назначение:** Каталог внешних фактов для LLM. Цель: легитимизация астрологии и закрытие барьеров аудитории на вебинарах. Каждый факт содержит скрипт, готовый к вставке в notes слайда.
**Связанные плейбуки:** `Playbook_Hormozi.md` (proof points каждые 5-7 мин), `Playbook_Product.md` (продукт), `Playbook_Cases.md` (кейсы учениц), `Playbook_Buyer.md` (барьеры, DT-1..5), `Playbook_Speaker.md` (голос спикера для скриптов), `Playbook_Slides.md` (формат вставки в TSV), `Playbook_Astro.md` (контент).

---

```xml
<role>
Ты — фактчекер-копирайтер вебинаров.
Задача: подбирать и вставлять внешние факты (пруфы) в скрипт вебинара для закрытия конкретных барьеров аудитории.
Правило: доказательства > обещания. Каждый факт — с источником, без выдумок.
При подборе пруфов ИСПОЛЬЗУЙ Playbook_Buyer.md: барьеры из DT-1..DT-5, язык ЦА.
При вставке в слайды ИСПОЛЬЗУЙ Playbook_Slides.md: формат notes (льющаяся речь, связки, 80-120 слов).
</role>
```

---

```xml
<glossary>

## ГЛОССАРИЙ

| Термин | Определение |
|--------|-------------|
| Пруф | Внешний факт (исследование, статистика, исторический пример), используемый для легитимизации метода и закрытия барьера |
| Скрипт | Готовый текст для речи спикера (notes) — произносится вслух, написан живой речью |
| Барьер | Убеждение аудитории, мешающее доверию или покупке (см. Playbook_Buyer DT-1..5) |
| Сила (★) | Оценка убедительности: ★★★★★ = peer-review журнал / Нобель / крупное исследование; ★★★★☆ = хороший источник, но с оговорками; ★★★☆☆ = слабая выборка или спорный |
| Proof point | Момент в вебинаре, когда спикер вставляет факт — каждые 5-7 мин (Playbook_Hormozi §8.4) |
| Peer-review | Экспертная рецензия научной работы перед публикацией — золотой стандарт доказательности |

</glossary>
```

## КАК ИСПОЛЬЗОВАТЬ

```
АЛГОРИТМ ПОДБОРА ПРУФА:

1. ОПРЕДЕЛИ БАРЬЕР → Таблица «Быстрый доступ по барьерам» ниже
2. НАЙДИ ФАКТ по номеру → Открой соответствующую ЧАСТЬ
3. ВОЗЬМИ СКРИПТ → Вставь в notes слайда (Playbook_Slides §0: связки, живая речь)
4. АДАПТИРУЙ при необходимости → Скрипт можно сократить или расширить, сохраняя источник
5. ПРОВЕРЬ → Факт соответствует барьеру? Источник указан? Нет слов из антисловаря?

ПРАВИЛА:
- Не более 2 пруфов подряд без смены формата (кейс, практика, интерактив)
- Proof point каждые 5-7 мин (Playbook_Hormozi §8.4)
- Скрипт = живая речь, НЕ телеграф (Playbook_Slides §0)
- Если факт спорный (★★★☆☆ или с оговоркой) — использовать мягкие формулировки
```

### №1: Закон Бенфорда — числа НЕ случайны

| Параметр | Значение |
|----------|----------|
| **Суть** | В реальных данных (высоты гор, налоги, население) цифра 1 первая в 30% случаев. Не 11% по теории вероятности — а 30. |
| **Применение** | Аудиторы ловят мошенников этим законом — выдуманные числа нарушают закон. |
| **Источник** | Nature, PNAS, Scientific American |
| **Сила** | ★★★★★ |
| **Барьер** | «Числа случайны / совпадения» |

**Скрипт:**
> «Закон Бенфорда. Если взять любые реальные данные — высоты гор, налоговые декларации, население городов — цифра один будет первой в тридцати процентах случаев. Не одиннадцать, как должно по теории вероятности — тридцать. Аудиторы ловят мошенников этим законом. Вселенная распределяет даже хаос по строгому правилу. И ваша дата рождения — тоже не случайность.»

| **Сила** | ★★★★★ |

### №5: Апофения — мозг создан искать паттерны

| Параметр | Значение |
|----------|----------|
| **Суть** | Апофения — способность мозга находить закономерности. Не баг, а фича: помогала предкам выживать. |
| **Почему работает** | Переворачивает «вы видите то, чего нет» → «мозг создан видеть паттерны — и они реальны». |
| **Источник** | Conrad K., 1958; нейронаука |
| **Сила** | ★★★★☆ |
| **Барьер** | «Это просто совпадения» |

**Скрипт:**
> «Кто-то скажет: это просто совпадения, мозг видит паттерны там, где их нет. Да, мозг ищет паттерны — это называется апофения. Но почему эволюция сохранила эту способность? Потому что паттерны РЕАЛЬНЫ. Наши предки, которые видели закономерности — выживали. Те, кто не видел — нет.»

### №73: Электромагнитное поле сердца и мозг

| Параметр | Значение |

## ЧАСТЬ 12: ВЕРА, ОЖИДАНИЯ, ПЛАЦЕБО (№76-77)

### №76: Открытое плацебо — когда «пустышка» честно подписана

| Параметр | Значение |
|----------|----------|
| **Суть** | Пациентам с синдромом раздражённого кишечника давали таблетки, на которых честно написано «placebo». Несмотря на это, симптомы улучшились значительно сильнее, чем в контрольной группе. |
| **Почему работает** | Показывает силу ожиданий и ритуала даже без обмана. |
| **Источник** | Kaptchuk T.J. et al., *PLoS ONE* (2010). DOI: 10.1371/journal.pone.0015591. https://journals.plos.org/plosone/article?id=10.1371/journal.pone.0015591 |
| **Сила** | ★★★★★ |
| **Барьер** | «Это всё плацебо и самообман, значит, не имеет ценности» |

**Цифры:**
- 80 пациентов с СРК
- 59% в группе открытого плацебо сообщили об «адекватном облегчении», против 35% в контроле
- Размер эффекта — крупный, сопоставим с настоящими лекарствами

**Скрипт:**
> «Гарвардские врачи выдали пациентам с кишечными проблемами таблетки, на которых честно написали: "плацебо, в составе — ничего". Объяснили, что сам факт приёма таблеток иногда запускает внутренние ресурсы. Через три недели почти 60% пациентов с плацебо почувствовали значимое облегчение, в контрольной группе — 35%. То есть даже когда вы знаете, что "это просто ритуал", ритуал всё равно работает.»

### №77: Ожидания меняют результат (самосбывающийся гороскоп)

| Параметр | Значение |
|----------|----------|
| **Суть** | Позитивные ожидания повышают результаты тестов и уровень уверенности; негативные — снижают. Гороскоп выступает как удобный контейнер ожиданий. |
| **Почему работает** | Объясняет, почему «хороший прогноз» часто сбывается. |
| **Источник** | Обзор эффектов ожиданий в BBC Earth (см. №69) + общая литература по expectancy effects |
| **Сила** | ★★★★☆ |
| **Барьер** | «Если это работает только через веру — это ерунда» |

**Скрипт:**
> «Эксперименты показывают: если человеку сказать, что сегодня его день, он решает задачи и берётся за сложные проекты смелее. Если сказать, что всё будет плохо, — часть людей даже в тестах показывает худшие результаты. Гороскоп — это по сути аккуратно упакованное ожидание. И если наука подтверждает, что ожидания реально меняют поведение и итоги, то в наших интересах выбирать те модели, которые поддерживают, а не обесценивают.»

### №78: Overview Effect — переживание единства с Землёй

| Параметр | Значение |

Астронавты, увидевшие Землю из космоса, описывают мощное чувство единства со всем человечеством, исчезновение границ, резкий сдвиг жизненных приоритетов. Это документировано и описано в научном журнале. По сути, астрология предлагает похожий "над-видовой" взгляд на свою жизнь — только без полёта на орбиту.»

### №79: Самоорганизация — порядок из хаоса

| Параметр | Значение |
|----------|----------|
| **Суть** | Теория сложных систем показывает: из простых локальных правил спонтанно возникает упорядоченное поведение — в атмосфере, кристаллах, стаях птиц, нейронных сетях. |
| **Почему работает** | Даёт научный язык для идеи «скрытых паттернов» в жизни. |
| **Источник** | Heylighen F., обзоры по complexity science; Simon H. «The Architecture of Complexity» (1962) |
| **Сила** | ★★★★☆ |
| **Барьер** | «Всё хаос, никакого смысла и паттернов нет» |

**Скрипт:**
> «Физика и теория сложных систем показывают: порядок может возникать сам — без внешнего дирижёра. Снежинка формирует идеальный узор, стая птиц летит как одно целое, нейроны вместе создают сознание. Это называется самоорганизацией и эмерджентностью. Если на всех уровнях от атомов до общества мир создаёт узоры из хаоса, логично предположить, что и в нашей биографии есть более глубокий паттерн. Астрология как раз и занимается его описанием.»
