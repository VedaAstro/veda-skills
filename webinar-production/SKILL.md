# Производство вебинаров (контент)

## Когда использовать

- Написание речи спикера (notes) -- тон, маркеры, метафоры, табу Ирины
- Создание оффера / value stack -- тарифы, модули, цены, опции
- Подбор кейса ученицы по барьеру или сегменту
- Вставка факта / пруфа для легитимизации утверждения
- Написание скрипта дожима

## Ключевые правила

### Голос Ирины
- Обращение: "дорогие мои", "дорогие". НИКОГДА: "девочки", "красотки", "подружки"
- Тон: теплый наставник с материнской энергией, НЕ рекламный копирайтер
- Маркеры: "вот", "на самом деле", "согласны?", "откликается?"
- Метафоры: из жизни, из кухни, из материнства

### Кейсы
- Кейс = "такая же как ты" -- совпадение по СИТУАЦИИ, не по результату
- 33 кейса учениц по формуле Hormozi (ситуация -> проблема -> решение -> результат)
- Trial close после КАЖДОГО кейса: "Кто хочет так же? +"
- НЕ придумывать -- только из плейбука

### Пруфы
- Proof point каждые 5-7 мин
- 79 внешних фактов с источниками (НЕ мнения, а данные)
- Каждый факт = готовый скрипт (notes)
- Данные только из плейбуков, НЕ придумывать

### Продукт
- 7Н (7 направлений) / 5S (5 сфер) -- два формата курса
- Точные цены и тарифы -- только из плейбука
- НейроАстролог -- AI-инструмент, снимает барьер "не справлюсь"
