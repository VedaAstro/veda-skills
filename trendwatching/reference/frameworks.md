# Analysis Frameworks — Trendwatching

## 1. Hook-Body-CTA-Offer (Creative Analysis)

### Hook Types
| Type | Description | Example |
|------|-------------|---------|
| **problem** | Highlights pain point | "Tired of...", "Struggling with..." |
| **result** | Shows outcome | "Get clear skin in 7 days" |
| **shock** | Surprising stat or claim | "97% of people don't know..." |
| **social_proof** | Testimonials, numbers | "1M+ customers", "As seen on..." |
| **question** | Engages curiosity | "Did you know...?" |
| **trend** | References current trend | "The #1 TikTok trend..." |

### Body Types
| Type | Description |
|------|-------------|
| **before-after** | Transformation story |
| **demo** | Product in action |
| **testimonial** | Customer quote/review |
| **list** | Bullet points of features/benefits |
| **story** | Narrative arc |
| **educational** | Teaches something useful |

### CTA Types
| Type | Description |
|------|-------------|
| **direct** | "Buy now", "Shop today" |
| **soft** | "Learn more", "See how" |
| **urgency** | "Limited time", "Last chance" |
| **free** | "Try free", "Free shipping" |
| **social** | "Join 10K+ customers" |

### Offer Types
| Type | Description |
|------|-------------|
| **discount** | Percentage or dollar off |
| **bundle** | Multi-product package |
| **trial** | Free trial period |
| **free_shipping** | Shipping incentive |
| **guarantee** | Money-back guarantee |
| **exclusive** | Limited edition, VIP access |

### Scoring (1-10 per element)
- **Clarity** — how clear is the message?
- **Emotional impact** — what does it make you feel?
- **Uniqueness** — differentiated from competitors?
- **Relevance** — speaks to the target audience?

**Overall score** = average of element scores. 5 = average, don't inflate.

## 2. Landing Page Framework

### Above the Fold
- **Headline** — main value proposition
- **Subheadline** — supporting text
- **Primary CTA** — button text and style
- **Hero visual** — product photo / lifestyle / video / illustration
- **Social proof** — badges, ratings, logos, customer count

### Page Structure
- **Section order** — what story does the page tell?
- **Section count** — how long?
- **Content density** — text-heavy or visual?
- **Navigation** — sticky? minimal? full?
- **Narrative** — Problem → Solution → Proof → Offer (typical)

### Conversion Triggers
- **Objection handling** — FAQ, guarantees, comparisons
- **Trust signals** — reviews, certifications, press, security
- **Urgency** — countdowns, stock, limited offers
- **Social proof density** — how many proof points on page?

### Pricing Analysis
- **Price presentation** — how shown?
- **Plans/tiers** — if applicable
- **Highlighted tier** — which pushed?
- **Anchoring** — strikethrough, "save X%"
- **Risk reversal** — free trial, money-back

### Technical
- **Platform** — Shopify / WordPress / Tilda / Custom
- **Load speed** — visual estimate (fast/medium/slow)
- **Mobile responsive** — yes/no
- **Tech stack** — visible pixels, analytics, chat widgets

## 3. Competitor Clustering

| Cluster | Definition | Analysis Focus |
|---------|-----------|----------------|
| **direct** | Same product, same audience, same price | Primary competition, detailed analysis |
| **indirect** | Adjacent product or audience | What can we borrow? |
| **leader** | Market leaders, benchmarks | Best practices to emulate |
| **emerging** | New players with growing presence | Threats and innovation signals |

## 4. A/B Test Detection

Signs of active A/B testing:
- Same brand, similar creative structure, one element differs
- Multiple active ads from same brand targeting same audience
- **Winner signal**: longer run time (30+ days vs <7 days)

What to look for:
- Hook variations (same body, different hook)
- Visual variations (same text, different images)
- Offer variations (same creative, different discount)
- CTA variations (same ad, different button text)

---

## 5. Evergreen Analysis

**Evergreen** = creative running 30+ days. These are likely proven winners.

Common traits of evergreen ads:
- Clear, simple message
- Strong social proof
- Proven offer (usually discount or guarantee)
- UGC or lifestyle visual style
- Problem → Solution narrative

## 6. Idea Generation Framework

Each idea must include:
```json
{
  "title": "Descriptive name",
  "hook": "Template: 'I used to [pain] until I found [brand]'",
  "body": "Before-after transformation, 15-sec video",
  "offer": "20% off + free shipping",
  "cta": "Try it yourself",
  "target_segment": "Young professionals 25-35",
  "type": "safe_bet|wild_card",
  "priority": "high|medium|low",
  "based_on": "What pattern/evidence supports this idea"
}
```

- **safe_bet** — proven pattern, lower risk, predictable ROI
- **wild_card** — untested angle, higher risk, potential breakthrough

## 7. Russian Market Specifics

### Channels to check (in addition to Meta/TikTok)
- **VK** — dominant ad platform in Russia. Check VK Ads, wall posts with "Реклама" label
- **Telegram** — channels, bots, ads via Telegram Ads
- **Yandex Direct** — search ads (use Yandex Wordstat for volumes)
- **GetCourse** — dominant LMS/CRM, check product pages
- **Marquiz/Tilda** — quiz funnels, landing builders

### VK Ad Analysis extras
- Satellite groups (brands use 50+ groups to post same ad)
- Promoted posts vs targeted ads
- Community size vs engagement rate

### Yandex Wordstat integration
- Brand queries → brand awareness proxy
- Niche queries → market volume
- Seasonal dynamics → timing strategy
