# VEDA Sales Case Studies

Эта папка — для разборов реальных закрытых/незакрытых сделок Академии.

⚠️ **Коммерческая тайна.** Имена клиентов и менеджеров — анонимизированы. Суммы — только категорией (`флагман`, `бюджетный`, `upsell`), не абсолютные.

## Зачем это нужно

- учить менеджеров на реальных паттернах;
- видеть, где сработал insight, а где сломался процесс;
- пополнять coaching-базу без фантазий.

## Формат каждого разбора

- `case-<число>.md`
- Ситуация: сегмент, источник лида, продукт-категория
- Время звонка: `<N>` минут
- Использованные методики: `[SPIN, Challenger, MEDDIC, Sandler, айкидо]`
- Ключевой момент: insight или переломная реплика
- Результат: `closed-won` / `closed-lost` / `on-hold`
- Уроки: что работало, что нет

## Обязательные поля

- `lead_type`: cold / warm / diagnostic / resurrection
- `main_objection`
- `decision_process`
- `next_step_quality`
- `call_review_notes`
