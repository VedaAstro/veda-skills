## Чеклист (проверять КАЖДОЕ письмо)

```
ТЕМА
[ ] <40 символов, есть "Ваш/Ваша", утверждение (не вопрос)
[ ] Без эмодзи, без CAPS, без {first_name}, без "нейросети"
[ ] Прехедер 40-90 символов, дополняет тему

ТЕЛО
[ ] От первого лица Ирины, одна мысль, 80-150 слов (напоминание 40-70)
[ ] CTA — 1 кнопка, видна без скролла на мобильном, ссылок 1-3
[ ] Нет спам-триггеров: "деньги"→"доход", "бесплатно"→"доступ открыт" (полный список в reference/data.md)
[ ] P.S. = дедлайн / соцдоказательство / незакрытый вопрос на следующее письмо

3 СЛОЯ ОДНОВРЕМЕННО
[ ] Транзакционная тема — чтобы открыли
[ ] Боль байера на её языке — зачем читать
[ ] Ценность без покупки — даже без клика, из письма узнала полезное

ОТПРАВКА
[ ] Сегмент <5K, интервал между батчами 30-60 мин
[ ] Исключены: автоворонка, текущие ученики, уже зарегистрированные
[ ] Бот через 30 мин после email, 3-5 строк + ссылка
```

## Темы: что работает (данные VEDA Funnel)

| Тип | OR | Пример |
|-----|-----|--------|
| Транзакционная ("Ваш + готов/открыт") | 14-20% | "Ваш расчёт на апрель готов" |
| Подарок от эксперта | 12-14% | "Ирина Чайка оплатила вам 3 эфира" |
| Обратный отсчёт | 10-14% | "Через 24 часа — расчёт вашей карты" |
| Боль + снятие вины | 8-12% | "Это не лень и не невезение" |
| Кейс-история | 8-10% | "Полтора года боялась начать" |

**Убивает OR:** {first_name} в теме (5-6%), абстракция без "Ваш" (CTR 0%), тема >45 символов, "нейросети" в теме (выручка x10 хуже).

## Язык байера (НЕ эксперта)

| НЕ писать | Писать |
|-----------|--------|
| Сфера роста в карте | Откуда ждать денег |
| Натальная карта | Дата рождения |
| Экзальтация | Пик силы |
| Кейс выпускницы | История обычной женщины |
| Консультировать | Помогать людям |

Голос Ирины: первое лицо, тёплый, снятие вины ("это не лень — точка в дате рождения"), бытовые метафоры ("стучитесь не в свою дверь"), "если" в призыве (не "когда").

## Карта прогрева (для серий к запуску)

### 5 смыслов — покрыть минимум в 5 из 8 писем

| # | Смысл | Суть | Чего НЕ делать |
|---|-------|------|-------------|
| 1 | **Повод/событие** | Почему именно сейчас | "Приходите на вебинар" |
| 2 | **Узнаваемая боль** | Ситуация, которую читательница узнаёт в себе | "Углубите знания" |
| 3 | **Тизер продукта** | Нарастающая интрига, без названия продукта | Жаргон в теме |
| 4 | **Ценность без покупки** | Кейс / инсайт / практика в каждом касании | Пустые напоминания |
| 5 | **Премьера/дефицит** | Первый раз, единственный, ограничено | "Открываем набор" |

### Структура серии (цепочка с нарастанием)

| # | Функция | Что делает |
|---|---------|-----------|
| 1 | **Интрига** | Зацепить, обозначить событие, первый тизер |
| 2 | **Боль** | Назвать ситуацию, снять вину |
| 3 | **Озарение через кейс** | Реальная история с цифрами |
| 4 | **Скрытые выгоды** | Конкретные результаты эфира/продукта |
| 5 | **Срочность** | Обратный отсчёт, соцдоказательство |

- P.S. каждого письма = незакрытый вопрос → ответ в следующем
- Тизер продукта нарастает: "инструмент" → "раньше не видели" → "тот самый" → "покажу вживую"
- Email И бот **каждый день**, без пауз
- Напоминания = ценность + время ("через 3 часа — инструмент за 5 минут"), НЕ пустые ("через 3 часа эфир")

### Кейсы — ТОЛЬКО реальные

Из реестра кейсов Академии. Выдумывать запрещено. Имя + возраст + цифры. Боль байера ("поверить что получится"), не эксперта ("разрешить себе брать оплату").

## Антиспам

1. Чистая кириллица, без CAPS, без !!!
2. 1-3 ссылки (кнопка лучше текстовой +28% CTR)
3. Текст 60-80%, картинки 20-40%
4. Спам-триггеры заменены (полный список → `reference/data.md §Антиспам`)
5. SPF/DKIM/DMARC + чистая база
