# Email-баннеры и визуалы — best practices

## Когда использовать

- Создание баннеров для email-рассылок
- Создание картинок для TG-бота
- Выбор: текстовое письмо vs HTML с баннером
- A/B тест визуалов

---

## 1. ГЛАВНЫЙ ИНСАЙТ: 70% ПИСЕМ — БЕЗ БАННЕРА

HubSpot (500M+ писем): каждая картинка снижает opens на 25%, GIF на 37%.
Marketo: текстовые +8% CTR, +11% CTOR.
Local Marketing Group 2026: переход на plain text = +22% opens.

**Правило для Vedantica:**
- 70% рассылок — «личные письма» от Ирины с минимумом визуала (логотип + текст)
- 30% — HTML с одним сильным баннером (открытие уроков, финальный CTA)

Для нашей воронки уроков:
- День 0 (анонс) — без баннера, «личное письмо»
- День 1, 3, 5 (уроки открыты) — с баннером
- Напоминания — без баннера, текстовые
- День 7 (закрытие) — с баннером (два CTA)

## 2. РАЗМЕРЫ

### Email баннер
- **Файл:** 1200 x 400 px (2x для Retina)
- **Отображение:** 600 x 200 px (через HTML width/height)
- **Вес:** до 80 KB (JPEG quality 60-70 при 2x)
- **Формат:** JPEG для фото, PNG для графики с текстом
- **НЕ использовать:** WebP (Outlook не поддерживает), SVG (Gmail не поддерживает)

### Telegram картинка
- **Промо:** 1280 x 1280 px (квадрат — максимум экрана телефона)
- **Баннер с текстом:** 1280 x 720 px (горизонт)
- **Формат:** PNG (текст чётче после сжатия TG)
- **Вес:** до 500 KB (TG сожмёт до JPEG q82)

### Gmail лимит
Общий вес HTML + inline = **102 KB максимум**. Gmail обрезает с кнопкой «Показать полностью» — большинство НЕ нажимают.

## 3. БАННЕРНАЯ СЛЕПОТА — КАК ПРОБИВАТЬ

### Что привлекает взгляд (eye-tracking, NNGroup)
1. **Лица** (особенно прямой взгляд в камеру)
2. **Чистый текст** (plain text побеждает дизайн)
3. **Контент, не похожий на рекламу**

### Что игнорируется
- Всё, что визуально = реклама (яркие рамки, «кричащие» цвета)
- Stock photo (рукопожатия, фальшивые улыбки)
- Перегруженные баннеры (много элементов, много текста)
- Правая колонка, верхняя полоса

### Рабочие приёмы

| Приём | Почему работает | Данные |
|-------|----------------|--------|
| Фото эксперта | Доверие + лицо пробивает слепоту | +95% конверсий (VWO A/B тест) |
| Рукописный стиль | Аутентичность в эру AI-контента | +22% open rate (LMG 2026) |
| Скриншот продукта | Выглядит как контент, не реклама | Proof-элемент |
| Минимум текста на картинке | Считывается за 3 секунды | 3-7 слов максимум (Opensense) |
| «Messy» стиль | Личное > полированное | +28% opens, +22% CTR (BHirst Media) |

### Иерархия эффективности визуалов
1. **Реальное фото лица Ирины** — максимум доверия и CTR
2. **Скриншот карты с подсветкой** — proof, не реклама
3. **Рукописный текст / маркер** — аутентичность
4. **Кастомная иллюстрация** — для разнообразия
5. **Stock photo** — минимальная эффективность, НЕ использовать

## 4. ФОРМУЛА БАННЕРА

### Структура
```
┌─────────────────────────────────┐
│  [Фото Ирины]   ЗАГОЛОВОК      │
│   слева          3-5 слов       │
│                  крупно          │
│                                  │
│       [ CTA 2-3 слова ]         │
│       контрастный цвет          │
└─────────────────────────────────┘
```

### Правила
- **Заголовок:** 3-7 слов, шрифт крупный
- **Подзаголовок:** 1 строка (опционально)
- **CTA:** 2-3 слова, контрастная кнопка 44x44 px минимум
- **Итого текста:** максимум 10-15 слов
- **Правило 3 секунд:** сообщение считывается мгновенно
- **Один CTA** — множественные CTA = -121% конверсия

### Палитра для Vedantica
- **Фон:** глубокий синий (#1D4ED8) → фиолетовый градиент, или светлый (#DBEAFE)
- **Текст:** белый на тёмном / тёмный на светлом
- **CTA кнопка:** золотистый (#FFC107) или оранжевый — контраст с синим
- **Акценты:** звёздная текстура (тонко, не перегружать)

### Для астрологии — что работает
- Космическая палитра (синий + золото + звёзды) — но минимально
- Фрагмент натальной карты как фон/элемент
- Фото Ирины на фоне звёздной текстуры
- Рукописные элементы («подписано звёздами»)

### Чего избегать
- Эзотерический китч (кристаллы, руны, мистические глаза)
- Тёмный фон без контраста (не читается в dark mode)
- Перегруженность символами
- Типичный «инфобизнес-стиль» (жёлтый текст на чёрном, обводки)

## 5. ALT-ТЕКСТ

- Обязателен для каждого изображения
- 43% удаляют email если картинки не загрузились
- Не «banner.jpg» — а «15 минут: три точки в вашей карте»
- Дублировать смысл баннера в alt
- Можно стилизовать через inline CSS (цвет, размер)

## 6. ЧЕКЛИСТ БАННЕРА

```
☐ Размер: 1200x400 (email) или 1280x1280 (TG)
☐ Вес: до 80 KB (email), до 500 KB (TG)
☐ Формат: JPEG (фото) или PNG (графика с текстом)
☐ Текст на баннере: 3-7 слов заголовок + CTA
☐ Фото Ирины (если есть)
☐ Один CTA, контрастный цвет
☐ Alt-текст описательный
☐ Считывается за 3 секунды
☐ Не выглядит как реклама
☐ Проверено на мобильном
☐ Общий вес письма < 102 KB (Gmail)
```
