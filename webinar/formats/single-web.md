# Формат А — Один веб с оффером

## Когда использовать

- **Мини-продукт** (1-15К ₽) — не нужна серия
- **Разогрев базы** перед большим запуском — один веб, одна цель
- **Регулярный эфир** в рамках Академии — для учеников, CR выше
- **Дожим после серии** — когда основная серия прошла, собираем «хвост»

**Когда НЕ использовать:** запуск флагмана (ПНА/ВА) — там нужна серия из 3 вебов (см. `3-webs-series.md`). Аналитика подтверждает: серия даёт x2 выручки.

---

## Общая архитектура

```
Длительность: ~85-90 минут
Слайдов: ~90-95
Плотность: 1 слайд/мин (темп Ирины)

[Блок 1] Программа + захват              (0-8 мин,  ~10 слайдов)
[Блок 2] Узнавание боли                  (8-20 мин, ~12 слайдов)
[Блок 3] Снятие вины                     (20-27 мин, ~7 слайдов)
[Блок 4] Астро-фрейм                     (27-40 мин, ~12 слайдов)
[Блок 5] Практика                        (40-55 мин, ~15 слайдов)
[Блок 6] Мечта + кейсы                   (55-65 мин, ~10 слайдов)
[Блок 7] Миссионерский переход           (65-70 мин, ~5 слайдов)
[Блок 8] Оффер + закрытие барьеров       (70-85 мин, ~15 слайдов)
[Блок 9] Urgency + CTA                   (85-90 мин, ~8 слайдов)
```

## Skeleton сценария — чёткий порядок блоков

Ниже — **готовый каркас**, который заполняется под тему конкретного веба. Заменяй `[X]`, `[Y]`, `[Z]` на реальный контент.

### Блок 1. Программа + захват (0-8 мин, 10 слайдов)

**Задача:** расслабить зрителя + дать первую петлю + перечислить бонусы + сегментация.

**Шаблоны слайдов:** см. `../playbooks/P3-warmup-flow.md §Фаза 1`.

**Критично в этом блоке:**
- Слайд 1-2: приветствие + мост (30 сек)
- Слайд 3 (180 сек): **первая открытая петля** (P2)
- Слайд 4-5: программа + бонусы
- Слайд 6: сегментация (активность зрителя)
- Слайды 7-10: регалии (30 сек — 1 мин, не больше) + дисклеймер

### Блок 2. Узнавание боли (8-20 мин, 12 слайдов)

**Задача:** Вывод 1 «это про меня».

**Шаблоны:** `P3 §Фаза 2`.

**Критично:**
- Слайды 11-14: **3-4 якоря 🔵** из `../lang/anchors-life-layer.md` (разные сегменты)
- Слайды 15-17: метафора (12 дверей / распредщит)
- Слайды 18-19: пример-контраст «две женщины»
- Слайды 20-22: вопросы в чат «плюс, если узнали» (микро-активность, P2)

### Блок 3. Снятие вины (20-27 мин, 7 слайдов)

**Задача:** Вывод 3 «я не виновата». **Критический блок — без него не работает практика.**

**Шаблоны:** `P3 §Фаза 3`.

**Критично:**
- Слайды 23-24: снятие вины через Нарратив 1 (внешний локус)
- Слайды 25-26: цикл застревания (визуализация)
- Слайд 27: разрешение «посмотреть, не купить»
- Слайды 28-29: эмоциональная пауза + мостик в фрейм

### Блок 4. Астро-фрейм (27-40 мин, 12 слайдов)

**Задача:** Вывод 2 «вот почему».

**Шаблоны:** `P3 §Фаза 4`.

**Критично:**
- Слайды 30-33: объяснение механики через метафору (4 главных дома / 5 планет / 7 слоёв)
- Слайды 34-35: **пруф метода** (5000 лет, 0.3% — см. `../library/proofs.md`)
- Слайды 36-39: **кейс «такая же как ты»** с внутренним сдвигом (см. `../library/cases.md`)
- Слайды 40-41: переход к практике + открытая петля 2 «через X минут вы сами увидите»

### Блок 5. Практика (40-55 мин, 15 слайдов)

**Задача:** Вывод 4 «я смогла».

**Шаблоны:** `P3 §Фаза 5`.

**Критично:**
- Слайды 42-43: подготовка + пауза 2 мин
- Слайды 44-49: таблица вариантов / схема «ваш X → ваш Y»
- Слайды 50-52: калибровка 1-10, реакция на чат
- Слайды 53-54: **фиксация «вы сделали сами»** (это ключ к Выводу 4)
- Слайды 55-56: тизер глубины «это один слой из семи» (Нарратив 7)

### Блок 6. Мечта + кейсы (55-65 мин, 10 слайдов)

**Задача:** Вывод 5 «это может стать моей жизнью».

**Шаблоны:** `P3 §Фаза 6`.

**Критично:**
- Слайды 57-60: **3 кейса разных сегментов** с внутренним сдвигом в центре (`../library/cases.md`)
- Слайды 61-63: личная история Ирины (`../core/bio.md §История 1. Ночь в больнице`)
- Слайды 64-65: переход к миссии + напоминание бонусов (P2)

### Блок 7. Миссионерский переход (65-70 мин, 5 слайдов)

**Задача:** перейти к офферу **через миссию**, а не через маркетинг.

**Критично:**
- Слайды 66-67: миссия Ирины (`../core/mission.md`)
- Слайды 68-69: триада «помочь себе → поделиться → карма рода» (`../core/narratives.md §Нарратив 4 Пассаж В`)
- Слайд 70: «мне нужны проводники»

**Это единственный правильный переход к офферу.** Никаких «а теперь расскажу про программу» / «у меня есть для вас предложение».

### Блок 8. Оффер + закрытие барьеров (70-85 мин, 15 слайдов)

**Задача:** Вывод 6 «одна не справлюсь» + закрытие 5 барьеров.

**🔴 Критический порядок (см. `P5-objections-aikido.md`):**

1. **Барьер №1 «Для себя, если получится»** — 5 слайдов (слайды 71-75). **ДО цены!** Это самый важный блок — без него цена не работает.
2. Поддержка и структура (триггер #1 — 39.2%) — 2 слайда (76-77): «куратор 5 мес, пишет на 3-й день, 92% доходимость»
3. Value stack — 2 слайда (78-79): 4 блока (обучение / поддержка / практика / статус)
4. Цена + рассрочка — 1 слайд (80)
5. **Барьер №2 «Дорого»** (22%) — 2 слайда (81-82): 3 ветки (нет денег / «мне нельзя» / страх потратить зря)
6. **Барьер №3 «Не справлюсь»** (17.5%) — 1 слайд (83): через уже-сделанное в практике
7. **Барьер №5 «Опять брошу»** (2.5% явно, но важно) — 1 слайд (84)
8. **Барьер №4 «Обман»** (5.2%) — 1 слайд (85): гарантия 14 дней

Все скрипты — в `../playbooks/P5-objections-aikido.md`.

### Блок 9. Urgency + CTA (85-90 мин, 8 слайдов)

**Задача:** Выводы 6, 7 «без этого не справлюсь» + «покупаю сейчас».

**Шаблоны:** `P6-now-this-way.md`.

**Критично:**
- Слайд 86: astro-urgency через реальное окно (`../library/astronomy.md`)
- Слайд 87: 3 фазы окна или конкретные даты
- Слайд 88: JTBD-позиционирование («именно это решает вашу главную задачу»)
- Слайд 89: реальный дедлайн с физической причиной (Пуджа / 20 мест / транзит)
- Слайд 90: финальный CTA + гарантия + практика намерения (3 вдоха)
- Слайды 91-93: закрытие эфира + выдача оставшихся бонусов + «до встречи»

## Критические правила формата А

### 1. Оффер только после 65-й минуты

Если оффер появляется раньше — зритель не успел пройти выводы 1-5, амигдала ждёт «опять продают», закрывается.

### 2. Барьер №1 «Для себя» — до цены, не после

Иначе цена не работает: «для себя» снимает давление результата, и рассрочка/окупаемость не цепляют.

### 3. Миссионерский переход — ДО value stack

Иначе value stack = коммерческая рамка, ломает голос Ирины.

### 4. Практика обязательна

Без практики не вызывается Вывод 4 («я смогла»), без него не работает оффер.

### 5. Первая петля в первые 180 секунд

Иначе 15-20% теряются на минуте 3-5.

### 6. Бонусы выдаются до или одновременно с оффером, не после

Иначе часть уходит после объявления цены.

### 7. Все астро-факты — верифицированы

Через `../library/astronomy.md`. Иначе риск повтора ошибки с «Сатурном в Овне» / «Венерой и Марсом в экзальтации раз в 12 лет» (эти формулировки были ошибочны).
