# Библиотека Ирины Чайки

## Что здесь лежит

| Файл | Что внутри | Когда открывать |
|---|---|---|
| `cases.md` | 33 кейса учениц с формулой Hormozi, по сегментам, барьерам, фазам веба | Нужен кейс для слайда / блока / закрытия барьера |
| `proofs.md` | 79 внешних фактов с источниками: статистика, наука, Wordstat, кейсы | Нужен пруф для легитимизации метода / утверждения |
| `products.md` | 7Н / 5S программы, тарифы, модули, опции, НейроАстролог | Писать оффер-блок, value stack, тарифы, FAQ |
| `visuals.md` | Разрешённые img_tag, примеры использования, правило разнообразия | Писать TSV, выбирать визуалы |
| `astronomy.md` | Верифицированные астро-факты, запрещённые утверждения | Вставлять астро-факт в сценарий, проверять ложные |

## Правила использования

1. **Не читать целиком** — открывать только нужный файл по задаче
2. **Каждый файл самодостаточен** — не требует чтения других
3. **Все факты — из реальных источников**. Если нужен новый — добавить в файл с источником
4. **При конфликте** с `../SKILL.md` — **побеждает webinar/SKILL.md**
