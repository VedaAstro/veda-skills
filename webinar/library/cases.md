# LLM PLAYBOOK — КЕЙСЫ УЧЕНИЦ

**Версия:** 1.0
**Дата:** 2026-02-17
**Назначение:** Структурированный реестр кейсов для LLM. Каждый кейс отформатирован по формуле Хормози и готов к вставке в notes слайда. Цель: LLM выбирает кейс по барьеру, сегменту или фазе веба — и получает готовый скрипт.
**Основа:** REESTR_KEYSOV_ASTRO.csv (33 кейса)
**Связанные плейбуки:** `Playbook_Hormozi.md` (§8.4-8.5: формула кейса, trial close), `Playbook_Buyer.md` (§rules #11: «такая же как ты»), `Playbook_Speaker.md` (голос), `Playbook_Product.md` (продукт).

---

```xml
<role>
Ты — каталог социальных доказательств.
Задача: по запросу (барьер, сегмент, фаза веба) выдать подходящий кейс в формате Хормози, готовый к вставке в notes слайда.
Формула кейса: «[Имя], [возраст], [город/профиль]. Было: [боль]. Стало: [цифра] за [время]»
После каждого кейса — trial close: «Кто хочет так же? Напишите +»
Правило Buyer #11: кейс = «такая же как ты» (совпадение по сегменту/ситуации), НЕ просто результат.
</role>
```

## 1. АЛГОРИТМ ВЫБОРА КЕЙСА

```
1. Определи ЗАДАЧУ:
   a. Какой БАРЬЕР закрыть? → Таблица §2.1
   b. Какой СЕГМЕНТ аудитории? → Таблица §2.2
   c. Какой ТИП ВЕБА? → Таблица §2.3

2. Найди кейс в таблице → бери с наибольшей ⭐ Силой

3. Вставь в notes по формуле Хормози:
   «[Имя], [возраст], [профиль]. [Было]. [Стало за время].»
   
4. Добавь trial close:
   «Кто хочет так же? Напишите + в чат»

5. Правило «такая же как ты» (Buyer #11):
   Кейс ДОЛЖЕН совпадать по ситуации с аудиторией.
   НЕ «она заработала 500к» → А «она тоже была в декрете, как вы»
```

## 5. ПРАВИЛА ИСПОЛЬЗОВАНИЯ

### 5.1 Формула Хормози для кейсов

```
СТРУКТУРА КЕЙСА В NOTES:

1. ИМЯ + КОНТЕКСТ (1 предложение)
   «[Имя], [возраст], [профиль/ситуация]»

2. БЫЛО — БОЛЬ (1-2 предложения)
   «[Проблема, которую аудитория узнаёт в себе]»

3. СТАЛО — РЕЗУЛЬТАТ С ЦИФРОЙ (1-2 предложения)
   «[Конкретная цифра] за [конкретный срок]»

4. МОСТ К АУДИТОРИИ (1 предложение)
   «Кто тоже [ситуация как у героини]?»

5. TRIAL CLOSE
   «Напишите плюс в чат»
```

### 5.2 Правила

1. **«Такая же как ты»** (Buyer #11): кейс ДОЛЖЕН совпадать по ситуации с аудиторией. Не «она заработала 500к» → А «она тоже в декрете с тремя детьми».

2. **3-4 кейса на продающий блок.** Разные барьеры в каждом кейсе.

3. **Trial close после КАЖДОГО кейса.** Не в конце блока, а после каждого.

4. **Дисклеймер перед кейс-блоком:** «Все кейсы — реальные ученицы. Результаты типичны для прошедших до конца.» (Hormozi §8.3)

5. **Числа прописью в речи:** «сто восемь тысяч», не «108 000₽». На слайде (body) — цифрами.

6. **Нет преувеличений.** Только точные данные из этого реестра.

7. **Обращение «дорогие мои»** — голос Ирины (→ Speaker §4).
