# Прогревочный веб (Веб-1) — старая версия, не читать

## Что это и зачем

**Веб-1 = первый эфир в серии 3.** Его задача — **контент + доверие**, не продажа.

По аналитике запусков:
- CR 3-го эфира в серии всегда **выше** 1-го (аудитория разогревается постепенно)
- Серия из 3 эфиров → выручка x2 vs одиночный эфир
- Задача Веб-1 — **сдвинуть зрительницу от «кто это?» к «это моё»** через практику и узнавание
- Оффер на Веб-1 = **критическая ошибка**, которая ломает всю серию

**Формула Веб-1:**
> Узнавание + фрейм + практика с «я смогла» + мост к Веб-2, где будет оффер.

## Архитектура Веб-1

### 7 блоков, ~85 мин, ~85-95 слайдов

```
БЛОК 1: Программа + захват       (0-8 мин,  ~10 слайдов)
БЛОК 2: Узнавание + снятие вины  (8-20 мин, ~12 слайдов)
БЛОК 3: Астро-фрейм               (20-33 мин, ~13 слайдов)
БЛОК 4: Практика                  (33-50 мин, ~15 слайдов)
БЛОК 5: WOW-момент (опционально)  (50-65 мин, ~15 слайдов)
БЛОК 6: Астрособытие + 25-дн.окно (65-78 мин, ~11 слайдов)
БЛОК 7: Лид-магнит + мост на Веб-2 (78-85 мин, ~10 слайдов)
```

### Блок 1. Программа + захват (0-8 мин)

**Цель:** ориентация — что будет, зачем оставаться.

**Обязательные элементы:**
1. Приветствие в голосе Ирины («Дорогие мои, здравствуйте»)
2. Программа первой, **НЕ боль** (5 пунктов с обещаниями)
3. Регалии коротко (30 сек — 1 мин) — из `../core/bio.md`
4. **3 бонуса досидевшим** (досиживание критично — без него CR падает)
5. Дисклеймер («результаты зависят от действий каждой» — 3+ раз за эфир)
6. Сегментация («кто здесь — для себя / для практики / для отношений / для ребёнка»)

**Антипаттерн:** долгие регалии (>2 мин на холодной базе, >1 мин на тёплой). Аудитория уходит.

**Голос:** см. `../core/voice.md` §«Эмоциональные режимы. 1. Тёплый наставник».

### Блок 2. Узнавание + снятие вины (8-20 мин)

**Цель:** сдвиги #1-3 (см. `../buyer/_index.md` §«5 моментов переключения»).

**Обязательные элементы:**
1. **Якори узнавания из 2-3 сегментов** — дословные фразы из `../buyer/segments.md`. Не менее 3 штук в разных слайдах.
2. Вопрос «напишите плюс, если узнали себя»
3. Метафора объяснения (например, «дом с 12 дверями» для ПНА-серии)
4. **Цикл застревания** — «симптом → неверное объяснение → действие не туда → вина → круг»
5. **Снятие вины через внешний локус** — `../core/narratives.md` §«Нарратив 1. Вам не показали карту»
6. Разрешение — «разрешите себе сегодня посмотреть»

### ⭐ Правило: каждый якорь узнавания = астро-привязка (микро-фраза)

**Критически важно** (выявлено при сверке Web1 v3): блок боли не должен быть чистой психологией. Иначе он неотличим от коуча/психолога и ломает принцип #1 «астрология — основа смысла».

**Каждый слайд узнавания должен содержать микро-привязку к карте — 1 фразу внутри notes:**

| ❌ Чистая психология | ✅ С астро-привязкой |
|---|---|
| «День сурка. Вы стараетесь, а доход не двигается. Напишите плюс, если узнали» | «День сурка. Вы стараетесь, а доход не двигается. **И это не "со мной что-то не так" — это конкретный паттерн в карте: ваши усилия идут не в ту дверь**. Напишите плюс, если узнали» |
| «Загораюсь и бросаю — знакомо?» | «"Загораюсь и бросаю" — это не характер. **Это рисунок в карте: сильное Солнце без поддержки Сатурна**. Так тоже можно объяснить» |
| «Кому я нужна в 44?» | «"Кому я нужна" — не про возраст. **Венера в вашей карте показывает, где ваш канал связи открыт именно сейчас**. Возраст не имеет отношения — карта имеет» |
| «И это всё?» | «"И это всё?" — это **Сатурн в определённом доме, сигнал души "я не там, где должна быть"**. Это можно увидеть. И можно изменить» |

**Формат микро-привязки:**
- 1 фраза, не целый блок
- В середине или конце notes, не в начале (сначала узнавание, потом привязка)
- Без терминов, которые зритель не поймёт (не «Сатурн в Сагиттарии по 10-му дому», а «Сатурн в определённом доме»)
- Обещание: «это видно, и это можно увидеть сегодня»

**Проверка:** если убрать все фразы про астро из блока узнавания — останется ли психологический коуч? Если да — блок сломан, добавить привязки.

**Антипаттерн:** боль без снятия вины дольше 3 минут. Активируется амигдала, зритель закрывается.

**Переход к следующему блоку:** «А теперь — фундамент. Без него практика не сработает.»

### Блок 3. Астро-фрейм (20-33 мин)

**Цель:** сдвиг #4 «Вот почему у меня так» через **астрологическое объяснение**.

**Обязательные элементы:**
1. **Метафора**, которая объясняет механику (например, «натальная карта = распределительный щит квартиры, 12 линий»)
2. **Астрологическое понятие**, которое лежит в основе (дома, планеты, накшатры, транзиты)
3. **Ключ к применению** — что смотреть в своей карте (лагна, планета, период)
4. **Пруф метода** — `../library/proofs.md` §«5000 лет джйотиш», «точность 0.3%»
5. **Кейс «такая же»** — одна история ученицы из `../library/cases.md` с внутренним сдвигом в центре (не с цифрой дохода)

**Принцип фрейма:** **движение через астро-объяснение, не через «как зарабатывать»**. Деньги в кейсе — последний штрих, не главный. Главное — внутренний сдвиг («перестала бояться», «увидела свою силу», «разрешила себе»).

**См. также:** `../SKILL.md` §«Как Ирина соединяет астро с нужным движением» — цепочка факт→образ→узнавание→локус→практика→фиксация.

### Блок 4. Практика (33-50 мин)

**Цель:** сдвиг #5 «Я смогла!» — через **выполненное действие в своей карте**.

**Обязательные элементы:**
1. Подготовка («нужна лагна», «откройте app.astroguru.ru если не знаете»)
2. **Пауза 2 минуты** на построение карты (в это время можно комментировать чат, пить воду)
3. **Таблица или схема** — как применить фрейм к своей карте (например, «какая лагна → какой дом → какая сфера»)
4. **Мини-практика калибровки** — «оцените от 1 до 10, насколько откликается»
5. **Вторая мини-практика** — эмоциональная или через ощущения («когда было легко заработать — через что?»)
6. **Фиксация «вы это сделали»** — обязательно, `../core/narratives.md` §«Нарратив 7. Два слоя» Пассаж А

**Антипаттерн практики:**
- ❌ Практика «на бумаге» — «запишите в блокнот». Аудитория не запишет, не выполнит, не почувствует «смогла».
- ✅ Практика «в чат» — зритель пишет действие, Ирина реагирует, есть подтверждение.

**Ветка «не получилось»:** всегда предусмотреть. Если зритель не нашёл, не понял — что делает? Ответ: «если пятёрка — значит, нужны следующие слои, это нормально».

### Блок 5. WOW-момент (50-65 мин) — опционально, если есть

**Цель:** продемонстрировать инструмент / технологию / способ, который даёт зрительнице ощущение «такого я не видела».

**Для ПНА-серии это демо НейроАстролога:**
1. Личная история: «2 года + 10 млн в инструмент. Почему»
2. Проблема, которую инструмент решает («подготовка к консультации 3 часа → 5 минут»)
3. Живое демо: выбрать участницу, ввести данные, построить карту, показать PDF
4. Разбор страницы «Деньги» этой участницы
5. Пруф-кейс: Ирина Душина — «с инструментом страх ушёл, за 14 дней 84 500₽»

**Ключевое правило для WOW-демо:**
- Это НЕ продажа инструмента («купите доступ»)
- Это **демонстрация возможности** («так можно»)
- Ирина Чайка даёт увидеть, что **инструмент — это снятие страха «я не эксперт»**, не «нажать кнопку и заработать»

**Подстраховка:** иметь 1-2 заготовленные карты на случай, если инструмент зависнет. Показать их как «ещё один пример».

### Блок 6. Астрособытие + окно (65-78 мин)

**Цель:** создать **контекст времени** — почему именно сейчас важно действовать, через **астрологию**, а не через «не упусти».

**Обязательные элементы:**
1. **Астрономический факт** — что происходит в небе в ближайшие 25 дней (проверенный через `../library/astronomy.md`)
2. **Смысл этого факта** — бытовая метафора
3. **Что это значит для зрительницы лично** — связка с её домом/картой
4. **3 фазы окна** — закладка / углубление / пиковые дни (с конкретными датами)
5. **Тизер на Веб-2** — «завтра расскажу про...»

**Принцип astro-urgency:** urgency создаётся **через реальное астро-событие**, не через маркетинг.
- ✅ «Венера с 19.04 в Тельце на 25 дней — окно для финансовых и отношенческих действий»
- ❌ «Только сегодня / осталось 2 часа»

**Миссионерский контекст (опционально):** если есть вчерашняя тема про «помощь другим», дать **присоединение**: «нам всем сейчас тяжело, и самые нужные — те, кто сам прошёл и может провести другую». НЕ коммерческую рамку «рынок растёт».

### Блок 7. Лид-магнит + мост на Веб-2 (78-85 мин)

**Цель:** зритель уходит с **реальной ценностью** + **ясным намерением прийти завтра**.

**Обязательные элементы:**
1. **Лид-магнит для всех** — PDF / чек-лист / инструмент (например, `astro.myveda.ru` для ПНА)
2. **3 бонуса досидевшим** (обещаны в блоке 1 — выполнить)
3. **Практика намерения** — соматическая (рука на сердце) + одно действие в чат
4. **Крючок на Веб-2** (см. ниже — НЕ просто анонс темы)
5. **Благодарность** — «до завтра»

### ⭐ Крючки на Веб-2 — обязательный элемент Веб-1

**Крючок на Веб-2 ставится МИНИМУМ 3 раза:**
- **В Блоке 1** (программа): «завтра покажу [X] — и без него сегодняшняя дверь не сработает»
- **В Блоке 4 или 5** (после практики): «вы нашли один слой. Завтра — второй, он отвечает, почему "делаю правильно, но не идёт"»
- **В Блоке 7** (финал): полный мост на Веб-2 с конкретным обещанием инсайта

**3 формы крючка (НЕ анонс темы):**

1. **Зависимость результата** — «то, что вы сегодня нашли — работает только если знаете [X], про который завтра»
   - ✅ «Сегодня вы увидели свою дверь. А завтра — увидите, почему у одних через эту дверь течёт, а у других застревает. Без этого дверь — только половина пути»
   - ❌ «Завтра поговорим про карму дохода»

2. **Обещание инсайта** — конкретная фраза, которую услышат только пришедшие
   - ✅ «Завтра покажу то, чего обычно не говорят: почему у женщины с одинаковым опытом и картой одна идёт, а другая топчется. Это один конкретный цикл, и его видно»
   - ❌ «Завтра будет ещё интереснее»

3. **Незавершённая история** — кейс, результат которого узнают только завтра
   - ✅ «Одна ученица сделала то, что я покажу завтра — и за 14 дней её жизнь изменилась. Цифры и детали — на Веб-2»
   - ❌ «Завтра будут кейсы»

**Принцип:** крючок создаёт **незавершённость**. Зритель должен уйти с ощущением «я что-то недополучил, и только завтра смогу дополучить». Это не манипуляция — это честное обещание реального инсайта, который НЕ влезает в Веб-1 без перегрузки.

**Проверка крючка:**
- [ ] В крючке есть **конкретная фраза** или **тип инсайта**, не просто «тема»?
- [ ] Зритель может сказать, **что именно** он узнает завтра?
- [ ] Крючок создаёт **зависимость** сегодняшнего результата от завтрашнего?

### ⭐ Вера «это для меня» — обязательно в Веб-1

**Три линии, каждая с конкретным слайдом:**

1. **«Такая же как ты» в кейсе** — в Блоке 3 или 4
   - Кейс ученицы с совпадением по ситуации (не по результату): 5/2 график, мама троих, после декрета, с нуля
   - Не «вау, заработала 500К». А: «работала как ты, боялась как ты, и вот что произошло»

2. **«Именно такая женщина нужна сейчас особенно»** — в Блоке 6 (миссионерский контекст)
   - Не «рынок растёт» (коммерческая рамка)
   - А «нам всем сейчас тяжело — и именно такие, как вы, обычные, прошедшие своё, становятся опорой для других. Это и есть служение»
   - Готовый нарратив — см. `../core/narratives.md` §«Нарратив 5. Обычная женщина ценнее звезды»

3. **«Астрология — лучший путь именно для тебя»** — в Блоке 5 (после демо) или 6
   - Почему астрология, а не психология / коучинг / блогинг:
     - Работает с КАРТОЙ (уже есть, не надо «изобретать себя»)
     - Инструмент снимает «я не эксперт» (НейроАстролог)
     - Деньги через служение (снимает «мне нельзя на себя»)
     - Карма рода — не только своё (сакральный уровень)
     - Один раз разобралась → на всю жизнь

**Без этих трёх линий Вера #3 («это для меня») не закрывается, и на Веб-2 зритель приходит с защитой.**

### ⭐ Предутилизация возражений на Веб-1 — минимум 2 штуки

На Веб-1 полной карты возражений ещё не нужно — нет оффера. Но **минимум 2 предутилизации обязательны**, чтобы снять защиту к Веб-2.

**Какие именно возражения предутилизировать на Веб-1:**

1. **«Это не ваша вина»** (страх самообвинения) — в Блоке 2-3
   - Нарратив 1 из `../core/narratives.md`
   - «В школе учили химию, а карту не показали — это не вы "неправильная"»
   - Снимает Барьер «Не справлюсь» (24% базы) на корневом уровне

2. **«Загораюсь и бросаю»** (страх надежды) — в Блоке 2 или 6
   - Названо вслух: «Знаю, многие думают "я уже пробовала и бросала"»
   - Присоединение, не оспаривание: «Вы правы. Я не обещаю. Просто предлагаю посмотреть»
   - Снимает главный скрытый блок прогрева

**Опционально (если аудитория подходит):**

3. **«Мне нельзя на себя»** — в Блоке 2 (разрешение)
   - «Сначала дети, потом муж, потом ипотека. На себя — если останется»
   - Переворот: «Когда мама на своём месте — семья счастлива. Это не эгоизм, это инвестиция в семью»

4. **«У меня не получится, потому что я не астролог»** — в Блоке 5 (WOW-демо)
   - Снимается инструментом: «Инструмент делает расчёт за вас. Вы — проводник, не эксперт»

**Формула айкидо (применяется ко всем):**
```
1. Назвать возражение вслух: «Знаю, что некоторые сейчас думают: [X]»
2. Присоединиться: «Это нормально. Это защитная реакция»
3. Переформулировать через внешний локус / смысл / кейс
4. НЕ оспаривать напрямую
5. Оставить зрителю пространство решать самой
```

**Полная карта предутилизации 5-7 возражений — только для Веб-2 (оффер-веб).** См. `offer.md §Блок 5. Карта предутилизации`.

**Антипаттерн финала:**
- ❌ CTA «кнопка под видео, покупайте»
- ❌ «Сегодня специальные условия»
- ❌ «Если не решитесь — завтра будет дороже»

**Правильный финал:** зритель уходит с ощущением «я прикоснулась к большему, и завтра узнаю больше».

## Пример архитектуры Веб-1 (для ПНА-серии, 15 апреля 2026)

Полный сценарий `Web1-scenario-15-04-v3.md` (следующая итерация) должен следовать этой схеме:

```
Блок 1: Программа + захват (12 слайдов, ~12 мин)
  Приветствие → мост из прогрева → регалии (30 сек) → 
  программа 5 пунктов → 3 бонуса → дисклеймер → сегментация

Блок 2: Узнавание + снятие вины (12 слайдов, ~12 мин)
  Якори A1+A2+B2 → «напишите плюс» → метафора «12 дверей» → 
  пример «две женщины» → снятие вины → разрешение → цикл застревания

Блок 3: Астро-фрейм (13 слайдов, ~13 мин)
  Распредщит → 12 домов → 4 главных денежных → лагна → 
  «я Телец — мало» → 5000 лет proof → кейс Ольги (10 дом → 108К)

Блок 4: Практика «Найди свою дверь» (14 слайдов, ~14 мин)
  Подготовка → пауза на astroguru → таблица 12 лагн → калибровка → 
  «вы сами сделали» → тизер «1 из 7 слоёв» → мини-практика «когда было легко»

Блок 5: WOW-демо НейроАстролога (15 слайдов, ~15 мин)
  Личная история → 2 года + 10 млн → НейроАстролог → 
  выбор участницы → ввод → ожидание → PDF → страница «Деньги» → 
  кейс Ирины Душиной → «выполнила обещание?»

Блок 6: Окно Акшая + контекст через миссию (11 слайдов, ~11 мин)
  Акшая Трития — что это → 2026 особенная → присоединение «нам всем тяжело» →
  «помощь другим — нужна сейчас особенно» (без противопоставления) →
  3 фазы окна → тизер Веб-2

Блок 7: Лид-магнит + финал (10 слайдов, ~9 мин)
  PDF всем → 3 бонуса досидевшим → тизер программы + грант →
  практика намерения → одно слово в чат → благодарность → до завтра
```

**Итого:** 87 слайдов, ~86 мин, плотность ~1 слайд/мин — в ритме Ирины.
