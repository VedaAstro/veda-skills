## Быстрое правило

**Ирина Чайка работает с 4 сегментами женщин 35-55, доход 30-70К, регионы.** Каждый сегмент имеет свой внутренний монолог, свой язык, свой главный барьер, свой момент переключения. **Без знания сегмента любая речь = абстракция и отскакивает.**

### 5 моментов переключения (последовательность важна)

```
1. «Это про меня!»       (узнавание через язык жизни)
       ↓
2. «Вот почему у меня так!» (понимание причины через карту)
       ↓
3. «Я не виновата»        (снятие вины через внешний локус)
       ↓
4. «Я смогла!»            (практика с микро-успехом)
       ↓
5. «Это может стать моей жизнью» (зажигание мечты → действие)
```

**Нельзя прыгать через шаги.** Если практика не дала «я смогла» — не работает разрешение. Если разрешение не снято — не работают ни миссия, ни оффер.

## 4 сегмента — краткая матрица

| Сегмент | Доля | Главный JTBD | Главный барьер | Триггер |
|---------|------|---|---|---|
| **A1: Потерявшая себя** | 40% | Найти предназначение, себя, смысл | Не справлюсь | «Это про меня!» + практика |
| **A2: В сложных отношениях** | 20%+ | Понять причину, принять решение | Стыд + зависимость | Надежда + кейс разрешения |
| **B1: Тревожная мама** | 10% | Понять ребёнка, направить | Не справлюсь + нет времени | Знать точно, а не гадать |
| **B2: После развода** | 5%+ | Финансовая независимость, начать заново | Дорого | Реальная экономика + новый старт |

**Остальные 25% — мелкие сегменты** (карьера, одинокие, выгорание). Работают по тем же правилам, но лидирующих — эти 4.

## 5 барьеров покупки (сводная)

| Барьер | Доля | Глубинный смысл | Где решается |
|--------|------|-----------------|--------------|
| **Дорого** | 27% | «Потрачу последнее» + «мне нельзя на себя» | Разрешение + блок «мне можно», не скидка |
| **Не справлюсь** | 24% | «Никогда не получалось» | Микро-успех в практике + кейс «такая же» |
| **Нет времени** | 16% | «Я последняя в очереди» | Переворот: «это не про время — про разрешение» |
| **Не найду клиентов** | 7% | «Кому я нужна» | «Ваша первая пятёрка уже в телефоне» |
| **Муж против** | 6% | «Перестанут любить» | Кейс «муж сам попросил карту» + «это ваша карта» |

**Страх надежды** — ключевой скрытый блок (не отдельный барьер, работает поверх всех). Женщина, уже обжигавшаяся на курсах, защищается. Снимается **НЕ обещаниями**, а **предложением попробовать без риска**.

**Режим «мне нельзя на себя»** — невидимый блок. Пока не снят разрешением — любая цена кажется «слишком много». Работа: не снижать цену → снимать боль разрешения.

## Триггеры покупки (что включает)

| Триггер | Доля | Механика |
|---------|------|----------|
| **Личность Ирины** | 27% | Окситоцин, доверие к человеку |
| **Понятность подачи** | 22% | Снижение кортизола, «наконец кто-то объяснил просто» |
| **Практика / применение** | 12% | Серотонин, «я смогла» |
| **Объём / полнота** | 9% | Дофамин, «это целая система» |
| **Заработок** | 7% | Дофамин → мечта (осторожно, чтобы не ломать миссию) |
| **Поддержка** | 6% | Окситоцин, «не бросят» |
| **Инсайт про себя** | 4% | Дофамин узнавания |

**Самый важный вывод:** триггер #1 — **личность Ирины**. Это значит: любая речь, которая ломает её голос → ломает триггер #1 → ломает всю конверсию. См. `../core/voice.md` и `../SKILL.md` (метаправило).

## Главный запрос: «Не бросьте меня» (32%)

**Поддержка важнее контента.** Это то, что реально вылечивает «загораюсь и бросаю».

| Ожидание | Доля |
|----------|------|
| **Обратная связь / поддержка** | **32%** |
| Практика | 16% |
| Понятность | 15% |

**Как отражать в веб-сценариях:**
- В прогреве: косвенно, через тон Ирины («я вижу вас», «не бросим»)
- В кейсах: показывать куратора в истории ученицы
- В оффер-вебе: явно — «92% доходимость», «куратор пишет на третий день», «не бросим»
- В follow-up: через реальные кейсы «мне написали, помогли, не дали сойти»

## 5 скрытых вопросов перед покупкой

Закрываются в порядке: доверие → вера в метод → вера в себя → финансы → семья.

| Вопрос | За чем стоит | Чем закрывать |
|---|---|---|
| **Это вообще реально?** | «Не развод?» | Практика с результатом на вебе + 5000 лет традиции |
| **Это реально для МЕНЯ?** | «Я особенная в плохом» | Кейсы «такая же как ты» |
| **Этот человек доведёт?** | «Не бросит?» | Регалии + кураторы + «92% доходимость» |
| **Я потяну по деньгам?** | «Не останусь в долгах?» | Рассрочка + калькулятор окупаемости (ТОЛЬКО в оффер-вебе) |
| **Мне разрешат?** | «Муж не осудит?» | Кейсы «муж сам попросил карту» + «это ВАША карта» |

**Правило:** в прогревочном Веб-1 закрываются первые 3 вопроса (через контент). В оффер-Веб-2 — оставшиеся 2 (через техники оффер-веба).

## Цикл «почему застряла»

Это корневая модель, которую Ирина вскрывает в каждом узнавании боли.

```
СИМПТОМ: «Нет денег» / «Устала» / «Муж не понимает»
    ↓
НЕВЕРНОЕ ОБЪЯСНЕНИЕ: «Мало стараюсь»
    ↓
ДЕЙСТВИЕ НЕ ТУДА: Берёт подработку / терпит / угождает
    ↓
НЕ РАБОТАЕТ: Денег не стало больше, сил меньше
    ↓
ВИНА: «Со мной что-то не так»
    ↓
ПОТЕРЯ ВЕРЫ: «Я уже всё пробовала»
    ↓
КОМПЕНСАЦИЯ: Терплю / откладываю жизнь
    ↓
ВОЗВРАТ К СИМПТОМУ (но хуже)
```

**Формулировка в речи Ирины:** см. `../core/narratives.md` §«Нарратив 1. Вам не показали карту» Пассаж Б.

**Важно:** выход из цикла — **не «больше стараться»**, а **изменить направление через карту**. Это связка с нарративом «Вы — проводник, не эксперт».

## Антипаттерны при работе с аудиторией

### В речи

| ❌ Нельзя | ✅ Вместо |
|---|---|
| «Это не депрессия» | «Это может быть похоже на то, как если внутри пусто, хотя снаружи всё есть» |
| «У вас абьюзивные отношения» | «Когда вас не слышат и не уважают» |
| «Это токсичное окружение» | «Когда рядом те, кто не верит» |
| «Выйти из зоны комфорта» | «Попробовать новое» |
| «Привлечь изобилие» | «Улучшить финансы» |
| «Негативные установки» | «То, что мешает» |
| «Проработка» | «Понять и изменить» |
| «Самоидентификация» | «Понять кто ты» |
| «Экзистенциальный кризис» | «Потеря себя» |

**Правило:** все академические термины — заменять на бытовые формулировки из словаря ЦА. См. `vocabulary.md`.

### В тоне

- ❌ Давление («последний шанс», «успей», «только сегодня»)
- ❌ Обесценивание страха («не бойся!», «это легко!», «ты справишься!»)
- ❌ Обвинение («ты выбираешь не тех», «ты не умеешь копить»)
- ✅ Присоединение («нам всем сейчас тяжело», «я это вижу»)
- ✅ Внешний локус («карта показывает», «это кармическая задача»)
- ✅ Дать попробовать без давления («просто посмотри», «просто попробуй»)

## Быстрый маршрут при написании сценария

1. **Определить состав зала** через сегментацию («для себя / консультирование / отношения / ребёнок»)
2. **Определить главный барьер каждого сегмента** → адресовать в речи
3. **Использовать язык сегмента** из `segments.md` (внутренние монологи)
4. **Проверить словарь** через `vocabulary.md` — нет ли запрещённых слов
5. **Проверить барьеры** через `barriers.md` — закрыт ли DT для каждого из 5 барьеров в оффер-части
6. **Проверить 5 моментов переключения** — пройдена ли последовательность?
7. **Пройти чеклист** `../SKILL.md §Чеклист` — миссия, не расчёт
