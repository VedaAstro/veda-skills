## 🔴 НАРРАТИВ 0 — СМЫСЛОВАЯ ОСЬ ВСЕХ ВЕБОВ (ЧИТАТЬ ПЕРВЫМ)

Любой веб Vedantica / Ирины Чайки строится на **одном родительском нарративе**. Без него веб = вода / продажа через боль → техники. С ним веб = контент про жизнь через астрологию, оффер ложится органично.

### 4 слоя Н0

**Слой 1. «Делается не то. Нужно делать другое».**
У вас проблемы, задачи. Вы очень много делаете. И не получается. Потому что **делается не то**. Нужно делать **другое**. Чтобы понять что — нужна астрология. Потому что есть **факторы (кармические, в карте)**, которые этим «другим» управляют.

**Слой 2. «Астрология видит причину и влияет на неё напрямую».**
С помощью астрологии можно **диагностировать прямые причины** (не переживание, как психолог; не цель, как коуч; не «да/нет», как таро). И **напрямую воздействовать** через **упаи** (ведические практики, мантры, камни, действия в правильные периоды). Инженерное решение на уровне причины. **Точка.**

**Слой 3. «Это именно для ВАС. Не сбоку. Не кому-то везёт».**
Не история про «кто-то зарабатывает». Именно для вас. Сложность была в том, что раньше для доступа к инструментам надо было учиться год+ и практиковаться. Мы создали **НейроАстролог** — инструмент даёт доступ сразу. Таким как вы. Без страха «я не эксперт».

**Слой 4. «Лучшая кармическая профессия — особенно сейчас».**
Одновременно помогаете другим и зарабатываете. По закону ведической традиции и по закону рынка. **Именно сейчас** — внешнее давление, неопределённость, всё блокируют → у всех обостряется проблематика → люди ищут опору → спрос растёт. Плюс астрономически — Акшая Трития = момент переписать карму.

## 🛑 КРИТИЧЕСКИЕ ПРАВИЛА (6 запретов)

### 1. Веб = астрология + drip, не блок-pipeline

Body/title слайдов = **астрология** (карма рода, 2-й дом, Юпитер, Акшая, упаи). Продающие смыслы (узнавание боли / снятие вины / кейсы / оффер) **капают drip в notes**, не образуют блоков «Блок боли / Блок оффера / Блок возражений».

Нарушение: pipeline T1→T7 под продажу = провал drip-принципа.

### 2. Не фабриковать биографические детали

Только то что в `core/bio.md` или подтверждено заказчиком явно.
**Пример анти-паттерна:** «родители Ирины на АвтоВАЗе / 17 дней конвейерного отпуска» — **галлюцинация** (факт был только «Ирина из Тольятти» + «АвтоВАЗ встал», остальное я выдумал для эмоционального якоря).

### 3. Не выдумывать цифры без первички

- Wordstat динамику — через реальный MCP `yandex-wordstat__dynamics`
- **Не использовать сезонные запросы** («гороскоп на 2026») для динамики — в Jan 2024 искали «на 2024»
- Общую статистику рынка эзотерики 2023 не выдавать за актуальную для ниши 2026
- Каждая цифра = с источником

### 4. Title / названия — через потребительский язык, не астро-термин

- ✅ «Где твои деньги», «Почему нет сил», «Карма рода: как переписать»
- ❌ «Ведический астролог за 60 дней», «Натальная карта с нуля», «Джйотиш», «Нейросети в астрологии», «15 лет практики»

Факт Vedantica analytics: title с финзвуком (Деньги/Карма) даёт 3.9M ср.выручку vs «Нейросети» 384K — **×10 разница**.

Полные правила → `context/market-language-shift.md` + `standards/naming-rules.md`

### 5. Не давить «успейте / последний шанс» как угроза

Дедлайны = факт календаря (Акшая Трития 19 апреля — астрономический факт), не угроза. Не «доверяете или живёте как сейчас» (обвинение). Не «до свидания неверующим» (на холодной базе риск потери).

### 6. Запрещённые штампы

«15 лет практики», «нейросети» в title, длинные тире, детские аналогии, «трейлер vs фильм», «меньше чашки кофе», «лучший момент войти», «недвижимость как бетон».
