# Производство вебинаров (контент)

## Когда использовать

- Написание речи спикера (notes) -- тон, маркеры, метафоры, табу Ирины
- Создание оффера / value stack -- тарифы, модули, цены, опции
- Подбор кейса ученицы по барьеру или сегменту
- Вставка факта / пруфа для легитимизации утверждения
- Написание скрипта дожима
- Подбор изображения / иконки / визуала -- 3D иконки, сферы, стикмены, планеты, знаки

## Ключевые правила

### Голос Ирины
- Обращение: "дорогие мои", "дорогие". НИКОГДА: "девочки", "красотки", "подружки"
- Тон: теплый наставник с материнской энергией, НЕ рекламный копирайтер
- Маркеры: "вот", "на самом деле", "согласны?", "откликается?"
- Метафоры: из жизни, из кухни, из материнства

### Кейсы (Формула Хормози, 6 шагов)

Источник: HORMOZI_WEBINAR.md, Фаза 4 — структура КАЖДОГО кейса:

| Шаг | Содержание | Пример |
|-----|-----------|--------|
| 1. Герой | Кто, контекст (имя, возраст, профессия, город) | "Ольга, 42 года, начальник отдела Соцфонда, 5/2" |
| 2. Проблема | Что было плохо (узнаваемая боль аудитории) | "Самозванец, уходило много времени, тяжело совмещать" |
| 3. Решение | Что конкретно сделали (инструмент/метод) | "Взяла НейроАстролога — подготовка сократилась" |
| 4. Результат | Цифры: деньги, срок, количество | "108 000 рублей за 10 дней, чек 5 000" |
| 5. Контраст | Было X -> Стало Y (яркое противопоставление) | "Было: нет времени, страх -> Стало: очередь клиентов" |
| 6. Вывод | Мост к аудитории + trial close | "Кто тоже работает 5/2? Напишите плюс" |

Правила:
- Кейс = "такая же как ты" -- совпадение по СИТУАЦИИ, не по результату (Buyer #11)
- 32 кейса учениц в реестре (TSV + Playbook_Cases.md)
- Trial close после КАЖДОГО кейса: "Кто хочет так же? Напишите +"
- Дисклеймер перед кейс-блоком: "Все кейсы — реальные ученицы"
- Числа прописью в речи ("сто восемь тысяч"), цифрами на слайде
- НЕ придумывать -- только из плейбука

### Пруфы
- Proof point каждые 5-7 мин
- 79 внешних фактов с источниками (НЕ мнения, а данные)
- Каждый факт = готовый скрипт (notes)
- Данные только из плейбуков, НЕ придумывать

### Продукт
- 7Н (7 направлений) / 5S (5 сфер) -- два формата курса
- Точные цены и тарифы -- только из плейбука
- НейроАстролог -- AI-инструмент, снимает барьер "не справлюсь"

### Как использовать

1. **Поиск**: открой каталог TSV, найди по тегам нужное изображение
2. **Путь**: `{GDRIVE_PATH}/{filename}` — файл доступен локально через Google Drive sync
3. **Конвертация для веба**: `cwebp -q 85 input.png -o output.webp` (макс 300KB)
4. **Важно**: icon_/sfera_/bonus_/module_ — PNG с прозрачным фоном (RGBA)
5. **image* файлы** — ч/б стикмены, использовать для эмоциональных слайдов и бот-сообщений
