## 1. ФУНДАМЕНТ: УРАВНЕНИЕ ЦЕННОСТИ И ПСИХОЛОГИЯ БАЙЕРА

### 1.1 Формула

**ЦЕННОСТЬ = (Результат мечты × Воспринимаемая вероятность) / (Время до результата × Усилия)**

Покупка происходит, когда ЦЕННОСТЬ >> ЦЕНА. Задача веба — поднять числитель и опустить знаменатель.

**НО:** У нашего байера есть дополнительное условие — **блок разрешения** (→ Playbook_Buyer §5.5). Даже при бесконечной ценности покупка не произойдёт, если не снят внутренний запрет «мне нельзя на себя». Это не про цену — островковая кора активирует физическую боль при мысли «потратить на себя» у женщины, 20 лет ставившей себя последней.

```
ПОКУПКА = (ЦЕННОСТЬ >> ЦЕНА) × РАЗРЕШЕНИЕ
Если РАЗРЕШЕНИЕ = 0, покупка = 0 при любой ценности.
```

### 1.2 Четыре переменных

| # | Переменная | Цель | Как влиять на вебе | Через какой астро-инструмент |
|---|-----------|------|-------------------|----------------------------|
| 1 | Результат мечты | ↑ | Конкретика в терминах статуса | Атмакарака → «ваше призвание записано в карте» |
| 2 | Вероятность | ↑ | Практика «ты уже сделала» + кейсы + калибровка | Практика на вебе → 87% совпадение |
| 3 | Время | ↓ | «Сколько ещё терпеть текущую ситуацию?» | Периоды планет → «окно открыто сейчас» |
| 4 | Усилия | ↓ | «Не больше работать, а правильно» + НейроАстролог | Карта → «маршрут уже есть, не надо искать» |

### 1.3 Результат мечты — через призму сегментов байера

**Не абстракция, а конкретика. Не процесс, а пункт назначения. Формулировать в терминах статуса в ЕЁ группе.**

Мечту формулировать ПО СЕГМЕНТУ (→ Playbook_Buyer §2.2):

| Сегмент (Buyer) | Мечта (её слова 🟢) | Статусная формулировка |
|-----------------|---------------------|----------------------|
| **Потеря себя** (40%) | «Найти своё дело» | Подруги спрашивают «как ты это делаешь?» |
| **Отношения** (20%) | «Наладить с мужем» | Муж гордится, дети видят сильную маму |
| **Дети** (10%) | «Понять ребёнка» | Ребёнок открылся, доверяет |
| **После развода** (5%+) | «Начать заново» | Перестать зависеть и считать дни до зарплаты |
| **Выгорание** (5%) | «Уйти из найма» | Поехать куда давно хотела, не считая |

**Скрытые мечты (→ Buyer §4.2):** Разрыв слово↔желание огромен: путешествия — в ВЦИОМ 46%, в анкетах 0.6% (77×). Формулируйте мечту через конкретику быта, не через абстракции.

### 1.4 Три веры (закрыть ДО оффера)

| Вера | Вопрос в голове | Чем закрывать на вебе |
|------|-----------------|----------------------|
| Метод работает | «Это не гадание?» | Практика + калибровка + «5000 лет, разница с наукой 0.3%» |
| Спикер доведёт | «Почему ей верить?» | Регалии + история + кейсы учеников + цифры + миссия (→ Playbook_Speaker §1-3) |
| Я справлюсь | «А вдруг не смогу?» | Практика «ты уже сделала» + кейсы 50+ + НейроАстролог |

**Условия (время, деньги) — часть веры №3, не отдельный барьер.**

### 1.5 Снятие вины — обязательный элемент

**После каждого блока боли — снять вину. Не «ты виновата», а «тебе не показали».**

| Не так | А так |
|--------|-------|
| «Вы делаете неправильно» | «Вам не показали, как» |
| «Вы работаете на чужой работе» | «Не знали, где ваше место — в карте записано» |
| «Вы повторяете мамин сценарий» | «Не знали, что он не ваш — Сатурн повторяет, пока не услышите» |

**Астрологический фрейм:** Вина снимается через внешнюю причину (планета, период, карма) → «Не ваш характер. Это Сатурн даёт урок» → Решение = осознанная работа с картой.

### 1.6 Сломанная модель → Новая модель

Паттерн, присутствующий в КАЖДОМ продающем вебе:

| Сломанная модель (старый путь) | Новая модель (наш путь) |
|-------------------------------|------------------------|
| Купи → Учись → Попробуй → (может быть) заработай | Попробуй → Заработай → Убедись → Купи |
| «Сколько дипломов у вас в ящике?» | «Сначала результат — потом решение» |
| Риск на ученике | Риск на Академии |

**Астрологический фрейм:** Как дипломы в ящике — это как Сатурн, который накопил долги: вы вложили, но не получили отдачу. Новая модель = прожить метод на себе, увидеть результат, и только потом решить.

## 2. АСТРОЛОГИЯ КАК ЯДРО ВЕБА

### 2.1 Принцип: астрология — носитель, не надстройка

Все продающие смыслы проходят ЧЕРЕЗ астрологический инструментарий. Планета = продающий аргумент. Дом = сфера боли. Период = причина срочности.

### 2.2 Карта «планета → продающая функция»

| Планета | Сфера | Продающая функция на вебе | Пример формулировки |
|---------|-------|--------------------------|---------------------|
| **Атмакарака** | Душа, призвание | ЗАХВАТ: «Найдёте свою миссию» | «Атмакарака — планета с наибольшим градусом. Она показывает, для чего вы здесь» |
| **Юпитер** | Деньги, мудрость, рост | УГЛУБЛЕНИЕ: «Откуда приходят деньги» | «Юпитер в вашей карте показывает, через ЧТО к вам идут деньги. Через людей? Через знания? Через риск?» |
| **Сатурн** | Карма, уроки, дисциплина | БОЛЬ + СРОЧНОСТЬ: «Почему деньги уходят» | «Сатурн — строгий судья по долгам. Он забирает то, что вы должны по карме. Не наказывает — взыскивает. Пока урок не пройден — будет приходить снова, и каждый раз строже» |
| **Венера** | Отношения, женственность | РАЗРЕШЕНИЕ: «Кто запретил быть счастливой» | «Венера в вашей карте показывает, как вы принимаете любовь и изобилие. Слабая Венера — запрет на "хотеть для себя"» |
| **Раху / Кету** | Направление / Прошлое | ФИНАЛЬНЫЙ ТОЛЧОК: «Куда идти» | «Раху показывает вектор будущего. Кету — что вы уже умеете из прошлых жизней. Вместе = компас» |
| **Солнце** | Эго, отец, лидерство | УВЕРЕННОСТЬ: «Право быть собой» | «Солнце — ваше Я. Сильное Солнце = знаете, кто вы. Слабое = ищете одобрения» |
| **Луна** | Ум, эмоции, мать | ЭМПАТИЯ: «Почему чувствуете так» | «Луна — ваш ум. Когда Луна ослаблена — тревога, неуверенность, перепады» |
| **Марс** | Энергия, действие | АКТИВАЦИЯ: «Сила действовать» | «Марс даёт энергию. Если в карте он ослаблен — чувствуете, что сил нет, хотя вроде здоровы» |

### 2.3 Карта «астро-концепция → продающий смысл»

| Астро-концепция | Продающий смысл | Как используется |
|----------------|-----------------|-----------------|
| Натальная карта | «Инструкция к себе» | Основной фрейм всего интенсива |
| Периоды планет | «Окно открыто сейчас» | Естественная срочность (не таймер, а космос) |
| Дома | «Где искать» | Конкретизация: «ваш 10-й дом показывает сферу карьеры» |
| Накшатры | «Точная настройка» | Аргумент глубины: «3 человека со Львом — 3 разные инструкции» |
| Аспекты | «Кто влияет» | Объяснение конфликтов: «Сатурн смотрит на Юпитер — деньги через труд» |
| Йоги | «Встроенные формулы» | Кейс: «У неё была йога богатства — нашли и активировали» |
| Ретроградность | «Недоделанные уроки» | Снятие вины: «Ретро-Сатурн = карма из прошлой жизни, не ваша вина» |
| 7 уровней карты | «Трейлер vs фильм» | Аргумент для покупки: «На интенсиве — 3 планеты × 1 уровень. На курсе — 9 × 7» |
| Атмакарака | «Планета души» | Первая практика: найти и интерпретировать |
| Упайи (коррекции) | «Можно изменить» | Надежда: «50% записано, 50% в ваших руках» |

### 2.4 Логика «трейлер — фильм»

На вебе зритель получает ТРЕЙЛЕР: одну планету, один уровень. Этого достаточно, чтобы:
1. Убедиться что метод работает (калибровка 8+/10)
2. Получить инсайт о себе (вау-эффект)
3. Понять, сколько ещё не раскрыто (9 планет × 7 уровней = 63 параметра)

**Формула:** «Мы посмотрели 3 планеты по 1 уровню. Это 3 из 63. Трейлер — 2 минуты из двухчасового фильма.»

### 2.5 Астрологическая срочность (вместо искусственной)

| Тип | Пример | Почему работает |
|-----|--------|-----------------|
| Планетарный транзит | «Венера в Тельце с 19 мая — раз в 12 лет» | Объективное событие, не маркетинг |
| Смена периода | «Период Юпитера начинается — время расширяться» | Личное, по карте |
| Нумерологический цикл | «2026 — год единицы, начало 9-летнего цикла» | Начинай сейчас = задаёшь вектор |
| Созревание планеты | «Сатурн созревает в 36. Если вам 35-37 — это ВАШЕ время» | Привязано к возрасту зрителя |

### 2.6 Запрещённые метафоры

| Нельзя | Почему | Замена |
|--------|--------|--------|
| «Кран» (свой / чужой) | Примитивизирует систему | «Ваш источник» / «Ваш канал» |
| «Дырявое ведро» | Бытовая, не ведическая | «Сатурн — строгий судья по долгам. Забирает то, что вы должны по карме» |
| «Гадание» | Обесценивает метод | «Система возрастом 5000 лет» |
| «Предсказание» без оговорки | Юридический риск | «Карта показывает возможности и тенденции» |

## 3. АРХИТЕКТУРА СЕРИИ ВЕБИНАРОВ

### 3.1 Три типа вебов

| Тип | Цель | Продажа? | Длительность | Формат |
|-----|------|----------|-------------|--------|
| **Прогрев** | Доверие + практика + закрытие барьеров | Нет | 60 мин | Контент + практика + петля |
| **Оффер** | Продажа | Да | 90-120 мин | Контент + практика + оффер |
| **Дожим** | Допродажа тем, кто не купил | Да | 60-80 мин | Честность + кейсы + оффер |

### 3.2 Три закона серии

1. **Каждый веб заканчивается открытой петлёй** — «Завтра покажу, почему деньги утекают»
2. **Практика даёт вкус, но не полную картину** — «Трейлер, не фильм»
3. **Барьеры закрываются ДО того, как зритель их озвучит** — проактивно

### 3.3 Формат 5 вебов с астро-ядром

| Веб | Тип | Астро-тема | Продающая функция | Эмоция | Барьеры |
|-----|-----|-----------|-------------------|--------|---------|
| 1 | Прогрев | **Атмакарака** (предназначение) | Захват: «найдёшь миссию» | Удивление: «работает!» | скучно, не про меня |
| 2 | Прогрев | **Юпитер** (деньги, рост) | Углубление: «откуда деньги» | Понимание: «вот почему так» | не справлюсь, почему спикер |
| 3 | Оффер | **Сатурн** (уроки, карма денег) | Продажа: «почему утекают + решение» | Желание: «хочу всю систему» | дорого, время, клиенты |

### 4.3 Маршрут переключения байера (→ Playbook_Buyer §3.3, §task)

Каждая фаза веба решает конкретную задачу на **маршруте покупки** из Playbook_Buyer. Последовательность нарушать нельзя — каждый шаг опирается на предыдущий:

```
ФАЗА ВЕБА          →  МАРШРУТ БАЙЕРА           →  МЕХАНИКА BUYER
─────────────────────────────────────────────────────────────────
ЗАХВАТ              →  Узнавание «это про меня» →  Язык 🔵 ЖИЗНЬ, маркеры боли 3-4
БОЛЬ + СНЯТИЕ ВИНЫ  →  «Вот почему так»         →  Внешний локус (карта/планета) + цикл §2.3
ПРАКТИКА            →  «Я смогла!»              →  Микро-успех (Buyer §5.3) → серотонин
КЕЙСЫ               →  «Это возможно ДЛЯ МЕНЯ» →  Кейсы «такая же как ты» (Buyer §rules #11)
ОФФЕР               →  РАЗРЕШЕНИЕ «мне можно»   →  Блок разрешения (Buyer §5.5) + айкидо
ВОЗРАЖЕНИЯ          →  Закрытие конкретного DT   →  DT-1..DT-5 из Buyer §logic
```

🔴 **Если практика не дала «я смогла» — не переходить к офферу.** Амигдала активна = любое предложение воспринимается как угроза (→ Buyer §5.1).

🔴 **Если блок разрешения не снят — каскад цен бесполезен.** «Дорого» ≠ нет денег. «Дорого» = островковая кора (→ Buyer §5.2). Сначала разрешение — потом цена.

## 5. ПРОЕКТИРОВАНИЕ ПРАКТИКИ

### 5.1 Зачем нужна практика

Практика — центральный элемент, отличающий наши вебы от стандартного Хормози. Она одновременно:
- Доказывает что метод работает (вера №1)
- Даёт зрителю чувство «я уже сделала» (вера №3)
- Создаёт социальное доказательство через калибровку
- Создаёт ощущение «трейлера» — хочется больше

**Связка с Buyer:** Практика — **главный триггер покупки** (→ Buyer §3.2: «Вебинар/практика» = 14.2% из того, что помогло преодолеть страхи). Это **микро-успех** (→ Buyer §5.3): простое действие с гарантированным результатом, которое перезаписывает выученную беспомощность.

Для байера со страхом надежды (→ Buyer §3.1) практика — единственный способ обойти защиту. Не обещать → дать попробовать → зафиксировать: «ТЫ сделала».

### 5.2 Типы практик

| Тип | Описание | Пример | Когда |
|-----|----------|--------|-------|
| **Когнитивная** | Зритель находит что-то в своей карте | Найти Атмакараку, Юпитер, Сатурн | Прогрев, оффер |
| **Соматическая** | Ощущение в теле | Скан тела: где напряжение? | Дожим, спец.вебы |
| **Визуализация** | Управляемое воображение | «Золотые ножницы» — отрезать нить прошлого | Спец.вебы |
| **Реальное действие** | Физическое действие прямо сейчас | «Отправьте сообщение подруге прямо сейчас» | Спец.вебы |

### 5.3 Структура когнитивной практики (основная)

| Шаг | Что делает спикер | Что делает зритель | Время |
|-----|-------------------|-------------------|-------|
| 1 | «Сейчас вы сами найдёте свою [планету]» | Берёт телефон/калькулятор | 1 мин |
| 2 | «Вот инструкция — покажу на примере» | Смотрит демо | 2 мин |
| 3 | «Теперь ваша очередь. Считайте» | Считает | 3 мин |
| 4 | «Напишите в чат, что получилось» | Пишет результат | 1 мин |
| 5 | «У кого [X] — это значит [Y]» | Получает интерпретацию | 5 мин |
| 6 | **КАЛИБРОВКА:** «Оцените от 1 до 10, насколько совпало» | Ставит оценку | 1 мин |
| 7 | **«ВЫ САМИ СДЕЛАЛИ»:** «Только что нашли. Без подготовки. Значит справитесь» | Осознаёт | 1 мин |

### 5.4 Астро-практики по вебам (пример)

| Веб | Планета | На вебе (общее, всем) | В боте после (персональное) | Калибровочный вопрос |
|-----|---------|----------------------|---------------------------|---------------------|
| 1 | Атмакарака | Найти планету + дом | Трактовка по знаку Атмакараки | «Совпало с ощущением? 1-10» |
| 2 | Юпитер | Найти Юпитер + дом | Трактовка по знаку Юпитера + источник дохода | «Узнали себя? 1-10» |
| 3 | Сатурн | Найти Сатурн + дом | Трактовка по знаку Сатурна + конкретный урок + упайя | «Есть ощущение, что это про вас? 1-10» |

### 5.5 Правила практики

1. **Простая инструкция** — 2-3 шага, не больше
2. **Демо на примере** — сначала спикер показывает на себе
3. **Время на выполнение** — пауза минимум 2 мин, не торопить
4. **Чат читается вслух** — «Вижу Юпитера в Раке... О, Юпитер в Овне...»
5. **Калибровка обязательна** — «87% ставят 8+» = социальное доказательство
6. **«Вы уже сделали»** — закрывает веру №3 на месте
7. **НЕ давать полную интерпретацию** — «На курсе разберём все 7 уровней. Сейчас — первый»
8. **Проблема 12 знаков** — решается по правилам §5.6

### 5.6 Проблема 12 знаков и бот-связка

#### Суть проблемы

Если на вебинаре давать трактовки по знакам (например: «Юпитер в Овне означает... Юпитер в Тельце означает...»), зритель услышит СВОЙ знак за 30 секунд, а потом сидит 11 × 30 сек = 5.5 мин, слушая чужие. Внимание падает, вовлечённость теряется.

#### Решение: два уровня разборов

| Уровень | Где | Что | Почему работает |
|---------|-----|-----|-----------------|
| **Общий** | НА ВЕБЕ | Разборы по планетам, домам, аспектам — то, что интересно ВСЕМ одновременно | Все вовлечены, никто не ждёт |
| **Персональный** | В БОТЕ / ДЗ | Трактовки по конкретным знакам, накшатрам, комбинациям | Каждый получает своё, в своём темпе |

#### Что делать НА ВЕБЕ (интересно всем):

- **Найти планету** (Атмакараку, Юпитер, Сатурн) — КАКАЯ планета, в каком ДОМЕ
- **Общая интерпретация по дому** (12 домов — но здесь зритель узнаёт свой дом, а чужие тоже полезны как контекст)
- **Калибровка** — «совпало? 1-10»
- **Общие паттерны** — «если Сатурн в кендре — одна история, если в трикона — другая» (крупные блоки, не 12 штук)

#### Что выносить В БОТА / ДЗ (персональное):

- Детальные трактовки по знакам (12 вариантов)
- Накшатра-трактовки
- Персональные прогнозы (транзиты, периоды)
- Совместимость
- Индивидуальные упайи (коррекции)

#### Механика бот-связки (TaniBot / Bot V2)

Боты проекта (TaniBot, Bot V2) имеют доступ к LLM-базе знаний и могут персонально трактовать карту. Это используется в трёх функциях:

**1. Бонус за досиживание (крючок удержания)**

```
СПИКЕР: «Все, кто досидит до конца, получат персональный 
разбор своей Атмакараки от нашего астро-бота. 
Он знает всё, что знаю я — только отвечает лично вам.»

→ После веба зритель идёт в бот → получает разбор
→ В разборе — крючок: «А завтра на вебе мы посмотрим, 
   как это влияет на ваши деньги. Приходите!»
```

**2. Домашнее задание (углубление вовлечения)**

```
СПИКЕР: «Сегодня мы нашли вашу Атмакараку и дом. 
Теперь идите в бот — он расскажет, что означает 
ваш конкретный ЗНАК Атмакараки. 
Это то, что я бы рассказывала каждому отдельно час.
Бот сделает это за 2 минуты.»
```

**3. Крючок на следующий веб (петля через бот)**

```
БОТ (после разбора): «Ваша Атмакарака — Солнце в 10 доме. 
Это значит, что ваше призвание связано с лидерством и публичностью.

Но есть важный нюанс: Сатурн может блокировать этот потенциал.
Завтра на вебе Ирина покажет, КАК найти и снять этот блок.
Не пропустите — [ссылка].»
```

#### Правила бот-связки

1. **Бот дополняет, не заменяет** — на вебе должен быть полноценный опыт, бот = бонус сверху
2. **Персональное = ценность** — «то, что обычно стоит 5 000₽ за консультацию — бесплатно для участников»
3. **Крючок в каждом разборе** — бот ВСЕГДА заканчивает интригой на следующий веб или на оффер
4. **Не перегружать** — 1 бот-задание на веб, не больше. Иначе = домашка = кортизол

## 7. ПРОАКТИВНОЕ ЗАКРЫТИЕ БАРЬЕРОВ

### 7.1 Принцип

- Проактивно: закрыть ДО того, как зритель озвучит
- К моменту оффера все основные барьеры уже закрыты
- На оффере — только напоминание + кейсы

**Связка с Buyer:** Каждый барьер имеет **глубинный страх** за поверхностным (→ Buyer §3.1). При закрытии работать с глубиной, не с поверхностью. Алгоритмы закрытия — decision trees DT-1..DT-5 из Buyer §logic.

| Поверхностный барьер | Глубинный страх (Buyer) | Что работает |
|---------------------|------------------------|-------------|
| «Дорого» | Страх потери + блок разрешения «мне нельзя на себя» | Не снижать цену → снимать боль разрешения (Buyer DT-1) |
| «Не справлюсь» | Страх некомпетентности: «я снова увижу что я тупая» | Микро-успех → зафиксировать «ты сделала» (Buyer DT-2) |
| «Нет времени» | «Я последняя в своей жизни» | Переворот: «на детей время есть?» (Buyer DT-3) |
| «Муж против» | Страх отвержения: «меня перестанут любить» | Кейс + «это вложение в профессию» (Buyer DT-4) |
| «Надо подумать» | Паралич / вежливый отказ / перегруз | Упростить: «подходит или нет?» (Buyer DT-5) |
| *(молчит)* | Страх надежды: «если поверю и не выйдет — боль невыносима» | НЕ обещать → дать попробовать (→ Buyer §3.1) |

🔴 **Принцип айкидо (→ Buyer §5.3):** Не бороться с сопротивлением — присоединиться и перенаправить. «Да, серьёзные деньги. Давай посмотрим, подходит ли тебе вообще» — НЕ «Это инвестиция!»

## 8. РЕЧЕВЫЕ ФОРМУЛЫ И РИТМ

🔴 **Словарь:** Все формулы — на языке ЦА (→ Buyer §vocabulary). **Запрещено:** самоидентификация, экзистенциальный кризис, токсичные отношения, абьюз, проработка, негативные установки, привлечь изобилие, выйти из зоны комфорта, энергия денег. **Замены:** «найти себя», «потеряла себя», «день сурка», «пустота внутри», «сложные отношения», «понять и изменить», «улучшить финансы».

🔴 **Внешний локус (→ Buyer §vocabulary):** Никогда обвинять. Вместо «ты выбираешь не тех мужчин» → «Венера в 12 доме даёт сложные связи». Вместо «ты не умеешь копить» → «Раху во 2 доме создаёт утечку ресурсов». Астрология = внешний локус = снятие вины = открытая дверь к покупке.

### 8.1 Сквозные формулы (5-7 штук, повторять 3+ раз)

| # | Формула | Когда |
|---|---------|-------|
| 1 | «Реальные кейсы — не выдуманные истории» | Перед каждым блоком кейсов |
| 2 | «Та же карта — глубокий анализ — другая жизнь» | После каждой практики |
| 3 | «Это не обещание результата. Результаты зависят от ваших действий» | 3+ раза через весь веб |
| 4 | «Я выполнила обещание?» | После каждого пункта плана |
| 5 | «Только пока идёт эфир» | Каждые 5-7 мин после оффера |
| 6 | «Помоги людям понять себя» | Переход к продаже + финал |
| 7 | «Вы только что сделали это сами. Без подготовки» | После каждой практики |

### 8.2 Netflix-механика: крючки между блоками

🔴 **Принцип:** Зритель уходит не «после веба», а **между блоками** — когда текущая тема закончилась, а следующая ещё не зацепила. Каждый переход между блоками = точка потери. Netflix решает это тизером следующего эпизода ДО титров. На вебе — то же самое.

**Структура:** Каждый блок заканчивается **двумя элементами:**
1. **Закрытие** — резюме того, что получили («Итак, вы только что нашли свою планету души»)
2. **Крючок на следующий блок** — интрига, которая создаёт незавершённость и НЕОБХОДИМОСТЬ досидеть

**Формулы крючков по переходам:**

| Переход | Крючок (что говорить В КОНЦЕ текущего блока) |
|---------|----------------------------------------------|
| Программа → Боль | «Но прежде чем мы найдём ваше призвание — покажу кое-что. Неприятное. Но без этого дальше не пойдём» |
| Боль → Снятие вины | «И знаете, что самое важное? Это НЕ ваша вина. Через минуту покажу, почему» |
| Снятие вины → Фреймворк | «А теперь — то, ради чего вы здесь. Через 2 минуты вы увидите систему, которая объясняет ВСЁ это» |
| Фреймворк → Практика | «Хватит теории. Сейчас вы сами найдёте это в СВОЕЙ карте. Через 5 минут у вас будет ответ, за который платят 5 000₽» |
| Практика → Кейсы | «Вы только что это сделали. А теперь покажу, что случилось с теми, кто пошёл дальше. Первая история вас удивит» |
| Кейсы → Система | «Вы видели результаты. Но я ещё не показала ГЛАВНОЕ — систему, которая за ними стоит. Это следующие 10 минут» |
| Система → Оффер | «Всё это я показала вам за 60 минут. А теперь — то, что я обещала в начале. И это не то, что вы думаете» |
| Оффер → Возражения | «Знаю, о чём сейчас думаете. Сейчас отвечу на то, что крутится в голове — даже на то, что вы не спросите вслух» |
| Возражения → Живая часть | «А теперь — разберу карту кого-то из вас прямо в эфире. Кто хочет? Пишите дату рождения» |

**Типы крючков:**

| Тип | Механика | Пример |
|-----|----------|--------|
| **Обещание ценности** | «Через N минут вы получите X» | «Через 5 мин у вас будет ответ за 5 000₽» |
| **Интрига** | «Незавершённая история / неожиданный поворот» | «И самое важное — это НЕ то, что вы думаете» |
| **Контринтуитивное** | «Удивление, парадокс» | «Причина не в деньгах. Причина — в 5-м доме» |
| **Персональное обещание** | «Вы лично что-то получите» | «В следующем блоке найдёте ЭТО в своей карте» |
| **Социальное доказательство» | «Результат другого → хотите узнать как?» | «Татьяна заработала 100К за неделю. Сейчас покажу как» |

🔴 **Правило Netflix:** Крючок ставится ЗА 30 СЕКУНД ДО конца блока, не после. Как в сериале — самое интересное показывают ДО рекламы, не после.

🔴 **Правило выполнения:** Каждый крючок ОБЯЗАН быть закрыт в следующем блоке. Невыполненное обещание = потеря доверия. Формула проверки: «Я обещала показать [X] — сейчас покажу» (→ сквозная формула №4: «Я выполнила обещание?»).

### 9.2 Типы контентных блоков

| Тип | Функция | Частота |
|-----|---------|---------|
| Программа | Ориентация: что будет, зачем сидеть | Начало каждого веба |
| Титул | Открытие блока | 1 на блок |
| Боль | Эмоциональный крюк | 3-5 подряд с нарастанием |
| Мини-практика / интерактив | Активное действие зрителя (см. §9.4) | Каждые 3-5 слайдов |
| Пруф | Статистика, скрин, факт | После каждого утверждения |
| Кейс | Имя + возраст + город + было/стало | 3-4 на продающий блок |
| Практика (большая) | Пошаговая работа с картой | 1-2 на веб (→ §5) |
| CTA | «Кнопка под видео» | После каждого кейс-блока |
| Возражение | Барьер → кейс-ответ | 1 слайд = 1 возражение |
| Навигация | «Где мы сейчас» | Между блоками |

### 9.3 Принцип: доказательства > обещания

| Плохо | Хорошо |
|-------|--------|
| «Я эксперт» | «10 000 учеников, 4.9 на Отзовике, 1% крупнейших школ» |
| «Вы справитесь» | «Галина, 58 лет. Первая консультация — через 2 месяца» |
| «Метод работает» | «87% совпадение по калибровке» |
| «Это изменит жизнь» | «Татьяна: с 0 до 100 400₽ за 7 дней» |

### 9.4 Интерактив ≠ формальный вопрос

🔴 **«Напишите +» — это НЕ интерактив.** Это формальный жест. Настоящий интерактив — когда зритель ДЕЛАЕТ что-то и получает персональный результат. Именно это ценит наша аудитория и что отличает практический веб от лекции.

**Три уровня интерактива:**

| Уровень | Что | Примеры | Эффект |
|---------|-----|---------|--------|
| ❌ Формальный | Вопрос в чат | «Напишите +», «Кто слышит?» | Вовлечение 1/10 |
| ✅ Мини-практика | Быстрое действие (30 сек — 2 мин) | Наблюдение за телом, подсчёт, мысленное упражнение | Вовлечение 5/10 |
| ✅✅ Практика с картой | Работа со своей картой (5-15 мин) | Найти планету, интерпретировать, калибровка | Вовлечение 10/10 |

**Мини-практики (быстрые, между блоками):**

| Тип | Пример | Когда использовать |
|-----|--------|-------------------|
| **Телесное наблюдение** | «Закройте глаза. Где в теле напряжение? Назовите место в чат» | После блока боли — переход к снятию вины |
| **Мысленный эксперимент** | «Представьте: утро через год. Проснулись. Что первое чувствуете?» | Рисование мечты / точка Б |
| **Быстрый подсчёт** | «Посчитайте: сколько часов в неделю тратите на то, что не любите?» | Перед блоком «цена бездействия» |
| **Выбор из двух** | «Что ближе: "я устала терпеть" или "я не знаю чего хочу"? Напишите 1 или 2» | Сегментация внутри веба |
| **Психологическая мини-техника** | «Вспомните момент, когда последний раз чувствовали "я на своём месте". Сколько лет назад?» | Перед фреймворком |
| **Дыхание / пауза** | «Три вдоха вместе. Вдох... выдох... Сейчас — про вас» | Перед практикой с картой |

🔴 **Правило:** На каждые 10 мин контента — минимум 1 мини-практика (не считая формальных вопросов). Это делает веб ПРАКТИЧЕСКИМ, а не теоретическим.

> **Полный каталог из 30 упражнений** с механикой, психологией, мостами к продаже, готовыми наборами по типам вебов и фильтрами по фазам/типам/сегментам: → `Playbook_Interactive.md`

## 10. ЧЕКЛИСТ — ПРОВЕРКА ГОТОВНОСТИ

### Астро-ядро
- [ ] Каждый веб имеет свою планету / астро-тему?
- [ ] Все продающие смыслы идут ЧЕРЕЗ астрологию, не рядом?

- [ ] Аргумент «трейлер vs фильм» перед оффером?
- [ ] Астрологическая срочность (транзит / период / цикл)?
- [ ] Нет запрещённых метафор?

### Уравнение ценности
- [ ] Мечта конкретная (статус, не абстракция)?
- [ ] Все три веры закрываются до оффера?
- [ ] Время фреймируется как «сколько ещё терпеть»?
- [ ] Усилия: «активация, не напряжение» + НейроАстролог?

### Структура
- [ ] Веб начинается с программы (что будет, зачем сидеть), НЕ с боли?
- [ ] Снятие вины после каждого блока боли?
- [ ] Сломанная модель → новая модель (перед оффером)?
- [ ] Trial close после каждого кейса?
- [ ] Анти-отбор + 3 вопроса самоквалификации?
- [ ] Дисклеймер 3+ раз?

### Value stack
- [ ] 4 блока (Обучение / Поддержка / Заработок / Статус)?
- [ ] Каждый компонент имеет цену отдельно?
- [ ] Общая ценность >> цены (минимум 5-10×)?

### Каскад цен
- [ ] Минимум 5 ступеней (ценность → цена → грант → финал → рассрочка)?
- [ ] «Дешевле чем [бытовая аналогия]»?
- [ ] Калькулятор окупаемости?
- [ ] НДФЛ вычет?

### Усиление
- [ ] Дефицит с математикой (не просто «мало мест»)?
- [ ] Дефицит объяснён (грант / юбилей / событие)?
- [ ] Астро-срочность (транзит / период)?
- [ ] Минимум 3 бонуса разных типов с ценами?
- [ ] Гарантия сформулирована (14 дней + 3 клиента)?

### Серия вебов
- [ ] Каждый веб заканчивается открытой петлёй?
- [ ] Каждый веб (кроме 1) начинается с программы + моста?
- [ ] Барьеры распределены проактивно по матрице?
- [ ] К моменту оффера все основные барьеры закрыты?
- [ ] Практика на каждом вебе?
- [ ] Бот-связка: бонус за досиживание + крючок на следующий веб?

### Практика и интерактив
- [ ] На вебе — ОБЩИЕ разборы (по планетам/домам), не 12 знаков подряд?
- [ ] Персональные трактовки по знакам вынесены в бот / ДЗ?
- [ ] Интерактив = реальные мини-практики (тело, подсчёт, визуализация), не только «+»?
- [ ] Минимум 1 мини-практика на каждые 10 мин контента?

### Ритм и удержание
- [ ] Proof каждые 5-7 мин?
- [ ] Смена формата каждые 10-15 мин?
- [ ] Эмоциональные качели (не застреваем)?
- [ ] Сквозные формулы повторяются 3+ раз?
- [ ] Боль не дольше 3 мин без снятия вины?
- [ ] **Netflix-крючок на КАЖДОМ переходе между блоками?** (§8.2)
- [ ] Каждый крючок закрыт в следующем блоке? (невыполненное обещание = потеря доверия)
