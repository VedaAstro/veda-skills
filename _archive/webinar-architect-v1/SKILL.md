# Архитектор вебинаров

## Ядро: технология соединения

**Весь контент Ирины строится на одном принципе:**

> Через конкретную астро-технику (мухурта, накшатры, дробные карты, транзиты, ректификация) можно получить желаемое (доход, ясность, уверенность, отношения) или избавиться от проблемы (застой, неуверенность, непонимание) — точнее и легче, чем любым другим способом.

Если элемент (заголовок, слайд, notes, кейс, крючок) не соединяет технику с результатом — он не работает.

**Формула:**

```
[астро-техника] × [активный глагол] → [результат в жизни]

Эталон: "Мухурта богатства" — 2 слова, техника + результат

✅ "Как активировать транзит удачи на 25 дней"
✅ "Накшатры покажут, почему с одним легко, а с другим — война"
✅ "Дробные карты: как увидеть скрытую причину застоя в деньгах"
❌ "Ваша карта в период окна" — нет результата, нет глагола
❌ "5 планет в силе" — описание неба, не влияние на жизнь
❌ "Откуда взять решительность" — нет техники
❌ "Транзиты удачи: 25 дней" — нет глагола
```

**Правила формулирования:**

| # | Правило | ✅ | ❌ |
|---|---------|---|---|
| 1 | Три компонента: техника + глагол + результат | "активировать транзит для дохода" | "транзит в силе" |
| 2 | Глагол простой, планеты делают работу | активировать, включить, открыть, направить | действовать, работать, стараться |
| 3 | Результат конкретный | доход, удача, ясность, уверенность | поток, энергия, сила |
| 4 | Цифры цифрами | 5 планет, 25 дней | пять планет |
| 5 | Коротко: 3-5 слов идеал, макс 12 | "Мухурта богатства" | "Как использовать транзиты для того чтобы..." |
| 6 | Юрид.: потенциал/возможности | "раскрыть потенциал дохода" | "гарантия заработка", "вы заработаете X₽" |
| 7 | Тест: убери астро — есть результат? Убери результат — есть астро? | оба есть | одного нет = сломано |

## 10 правил (нерушимых)

1. **Каждый смысл = соединение.** Техника × результат. Формула → "Ядро" выше
2. **Голос = Ирина.** "Дорогие мои". Тёплый наставник. НЕ копирайтер
3. **Внешний локус.** Причина в карте/планете, не в человеке
4. **Программа первой.** Первые слайды = что будет + бонус за досиживание
5. **Речь льётся.** notes = связная речь, 80-120 слов. Нет обрубков
6. **Netflix-крючок** между КАЖДЫМИ блоками
7. **Trial close** после каждого кейса + маршрут (плюс → менеджер → помощь)
8. **Proof каждые 5-7 мин.** Кейс / статистика / калибровка
9. **Кейс = "такая же как ты."** Совпадение по ситуации, не по результату
10. **Данные из плейбуков.** НЕ придумывать кейсы, цифры, цены

## Чеклист — ОБЩИЙ

### Соединение: техника × результат
- [ ] КАЖДЫЙ заголовок = [техника] × [глагол] × [результат]?
- [ ] Нет голой астрологии без результата?
- [ ] Нет голой жизни без техники?
- [ ] Notes объясняют ЧЕРЕЗ астро, почему результат достижим?
- [ ] Юрид.: "потенциал/возможности", НЕ "гарантия"?

### Структура
- [ ] Программа первой (не боль)?
- [ ] Netflix-крючок на каждом переходе? Каждый закрыт?
- [ ] Мини-практики каждые 10 мин (тело, подсчёт, визуализация — не только "+")?
- [ ] Proof каждые 5-7 мин?
- [ ] Эмоциональные качели (не одна нота > 5 мин)?
- [ ] Снятие вины после боли?

### Цепочка сдвигов
- [ ] Узнавание → Понимание → "Я смогла" → Разрешение → Решение → Действие?
- [ ] Блок РАЗРЕШЕНИЯ до оффера?
- [ ] 3 веры до оффера (метод + спикер + я справлюсь)?
- [ ] Контраст подходов (старый не работал → новый работает)?
- [ ] Миссионерский переход (миссия, не "купите")?
- [ ] Анти-отбор + 3 вопроса самоквалификации?
- [ ] Страх надежды закрыт ("не обещаю — проверьте, 14 дней")?

### Речь
- [ ] "Дорогие мои", маркеры Ирины?
- [ ] Нет табу-слов, нет абстракций, нет директивных claims?
- [ ] Личная история Ирины?
- [ ] Дисклеймер 3+ раз?
- [ ] Сквозные формулы (5 шт, каждая 3+ раз)?

### Данные
- [ ] Кейсы из плейбуков? Trial close после каждого?
- [ ] Цены/тарифы из плейбуков?
- [ ] Предутилизация каждого исхода практики?

## Чеклист — LTV-вебы (на учеников)

Дополнительно к общему.

- [ ] Байер знает продукт — не объяснять с нуля?
- [ ] Оффер НЕ повторяет предыдущий веб? Новые аргументы?
- [ ] Регалии коротко (30 сек)?
- [ ] Контраст адаптирован: "общий подход → персональный", не "купи→учись→заработай"?
- [ ] Барьеры LTV: "лежат курсы" (айкидо), "для себя" (конкретика), "уже есть курс" (зачёт)?
- [ ] Продающие смыслы нативно в контенте, не отдельными блоками?
- [ ] Контент ценен сам по себе (без оффера)?
- [ ] Кейсы НОВЫЕ (не с прошлого веба)?
- [ ] FAQ по реальным вопросам после прошлого веба?

## Чеклист — холодная база

Дополнительно к общему.

- [ ] Полные регалии (2 мин)?
- [ ] Сегментация: "кто для себя / для практики / отношения"?
- [ ] Proof "5000 лет, разница с наукой 0.3%"?
- [ ] Полная сломанная модель ("купи→учись" → "попробуй→заработай")?
- [ ] Практика = "трейлер" ("3 из 63 параметров")?
- [ ] Value stack: 4 блока, каждый с ценой, каскад 5 ступеней?
- [ ] "Дешевле чем [бытовое]"? Калькулятор окупаемости?
- [ ] Дефицит с математикой?
- [ ] Бот-связка?
