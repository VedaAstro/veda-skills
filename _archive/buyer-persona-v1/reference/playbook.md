```xml
## ⚠️ КРИТИЧЕСКОЕ ПРЕДУПРЕЖДЕНИЕ О ДАННЫХ

### Источник и его ограничения

Все данные получены из **анкет людей, которые УЖЕ КУПИЛИ** курс. Это создаёт систематические искажения:

| Искажение | Суть | Как влияет | Как учитывать |
|-----------|------|------------|---------------|
| **Рационализация после покупки** | Человек заплатил 30-100К и склонен оправдывать решение | «Хочу стать астрологом» может быть оправданием, а не реальной болью | Фильтровать жизненные маркеры от курсовых целей |
| **Выборка только покупателей** | Кто НЕ купил — не попал в данные | Мы не видим тех, кого воронка не убедила | Помнить о «тёмной стороне» аудитории |
| **Социальная желательность** | Интимные проблемы занижены, одобряемые цели завышены | «Помогать людям» завышено, «одиночество» занижено в 20 раз | Декодировать эвфемизмы |
| **Смешение слоёв** | В ответах перемешаны жизненные боли и курсовые цели | «Стать астрологом» + «наладить отношения» в одном ответе | Разделять на слои (см. ниже) |

### Три временных слоя данных

Каждый фрагмент данных в этом документе относится к одному из трёх слоёв. **LLM ОБЯЗАНА различать слои при генерации контента.**

| Слой | Что это | Когда | Bias | Маркер |
|------|---------|-------|------|--------|
| **🔵 ЖИЗНЬ** | Реальная ситуация, боли, контекст ДО знакомства с продуктом | До первого веба | Искажён рационализацией, но жизненные маркеры (развод, кредит, дети) надёжны | 🔵 |
| **🟡 ОФФЕР** | Что происходит при встрече с продуктом: возражения, триггеры, что зацепило | Во время веба и принятия решения | Наиболее надёжный слой — прямой вопрос про момент покупки | 🟡 |
| **🟢 ПОСЛЕ** | Что говорят после покупки: мотивации, планы, ожидания | После оплаты | Максимальная рационализация: оправдывают решение | 🟢 |

### Как использовать слои

```
IF генерируешь РИЛС / ПРОГРЕВ / КОНТЕНТ для привлечения:
    → Используй только 🔵 ЖИЗНЬ (реальные боли, их язык)
    → НЕ используй 🟢 ПОСЛЕ (курсовые цели — это не то, чем живёт холодная аудитория)

IF генерируешь ВЕБИНАР / ОФФЕР:
    → Используй 🔵 ЖИЗНЬ для узнавания и обострения
    → Используй 🟡 ОФФЕР для закрытия барьеров
    → Используй 🟢 ПОСЛЕ для рисования картины будущего (мечта)

IF генерируешь ДОЖИМ / ДИАГНОСТИКА / БОТ:
    → Определи, в каком слое находится человек
    → Если ещё не был на вебе → 🔵 ЖИЗНЬ
    → Если был на вебе, не купил → 🟡 ОФФЕР (работай с конкретным барьером)
    → Если купил, нужна поддержка → 🟢 ПОСЛЕ
```

### Что надёжно, а что нет

| Тип данных | Надёжность | Почему |
|-----------|------------|--------|
| Демография (возраст, доход, город, профессия) | ✅ ФАКТ | Объективные данные |
| Частотность тем | ✅ ФАКТ | Прямой подсчёт |
| Язык ЦА / формулировки | ✅ ФАКТ | Прямые цитаты |
| Возражения при покупке (fears_raw) | ✅ ФАКТ | Прямой вопрос |
| Что зацепило на вебе | ✅ ФАКТ | Прямой вопрос |
| Что помогло решиться | ✅ ФАКТ | Прямой вопрос |
| Жизненные боли с маркерами (развод, кредит) | ⚠️ НАДЁЖНО | Конкретные факты, но занижены |
| Глубинные мотивации | ⚠️ ГИПОТЕЗА | Реконструкция, не прямые данные |
| «Помогать людям», «стать астрологом» | ❌ РАЦИОНАЛИЗАЦИЯ | Post-purchase bias |
| Приоритет болей | ❌ ИСКАЖЁН | Выборка только купивших |
```

```xml
### Паттерн «загораюсь и бросаю»

Многие байеры прошли через цикл:
1. Нашла «решение» (курс/марафон/психолог)
2. Загорелась
3. Через 2-4 недели бросила
4. Самобичевание: «опять не довела»
5. Вывод: «со мной что-то не так»

**Цитаты:**

> [46, Более 100К] «Я с 7 лет увлекаюсь западной астрологией, в последние годы училась у Павла Андреева, но почувствовала, что его манера преподавания мне чужда, а знания не соответствуют потраченному времени и финансам. В голове всё смешалось. Разочаровалась»

> [32, Более 100К] «Искала себя через курс психологии, медитации. Хочу теперь изучить астрологию, в надежде, что в ней найду ответы»

> [42, Нет дохода] «Подхожу к изучению себя со всех сторон — сама изучала матрицу судьбы, ходила на консультацию к тарологам. По всем картам у меня выходит обучение и помощь людям»

> [48, 51-70К] «Прошла много психологических консультаций»

**Для контента:** Это создаёт СТРАХ НАДЕЖДЫ (см. раздел 5.2). Нельзя обещать «на этот раз точно получится». Нужно дать попробовать.

## 2.1 Глубинный портрет

### Три слоя идентичности

| Слой | Сейчас | Хочет |
|------|--------|-------|
| Роль (что делаю) | Бухгалтер / учитель / мама | Консультант / эксперт |
| Статус (как видят) | «Та, которая тянет» | «К ней идут за советом» |
| Идентичность (кто я) | «Я терплю» | «Я знаю свой путь» |

### «Тихое отчаяние»

Не депрессия. Не острый кризис. Хроническое состояние:
- Внешне всё нормально (работа, семья, квартира)
- Внутри — пустота и вопрос «и это всё?»
- Нет острой боли, есть тупая хроническая
- Привыкла терпеть настолько, что не замечает

**Для продаж:** Острую боль легко продать. Тихое отчаяние — человек привык, не ищет. Задача — вытащить на поверхность.

### Социальное давление

| Кто | Что говорит |
|-----|-------------|
| Муж | «Опять ты со своими курсами», «Лучше бы работу нормальную» |
| Родители | «В наше время все работали», «Какие звёзды, иди работай» |
| Подруги | «Я тоже хотела, но...», «А ты уверена?» |
| Общество | «Женщина должна быть хорошей матерью», «Поздно менять» |

### Режим «мне нельзя»

- Нельзя хотеть для себя
- Нельзя тратить на себя
- Нельзя быть счастливой когда другим плохо

**Пока не снято это «нельзя» — покупка невозможна.** Даже если хочет и деньги есть.

### Точка А → Точка Б

**ТОЧКА А (где сейчас):**
- Работает на износ, доход 30-50К
- Кредит 400К+
- Видит детей 30 мин/день
- «День сурка»
- Зависит от мужа финансово и эмоционально

**ЧТО ДЕЛАЕТ (не работает):**
- Терпит и ждёт
- Берёт подработки
- Пробует курсы → бросает
- «Ещё больше старается» в том же направлении

**ТОЧКА Б (куда хочет):**
- Кредит закрыт, доход 150К+/мес
- Работает из дома, свой график
- Путешествует 2 раза/год
- Муж уважает, дети гордятся

### Сегмент A1: «Потерявшая себя» (40%) 🔵

**Демография:** 35-45 лет, доход 31-70К, часто после декрета или в кризисе

**Внутренний монолог (как она ДУМАЕТ, не как говорит на анкете):**

> *«Мне 38, а я не понимаю кто я. Всю жизнь для кого-то — для мужа, для детей, для работы. А сама? Где я во всём этом?*
>
> *Пробовала разное — курсы, книги, психологов. Ничего не цепляет. Загораюсь — и через месяц всё бросаю. Может, я просто такая? Может, у меня нет никакого призвания?*
>
> *Но внутри что-то ноет. Как будто я не там, где должна быть. Как будто жизнь проходит мимо.*
>
> *Муж не понимает. Говорит — у тебя всё есть, чего тебе не хватает? А я не могу объяснить.*
>
> *Иногда накрывает — паникую, плачу без причины. Потом стыдно.»*

**Язык (их слова, не наши):**
«потеряла себя», «не знаю куда себя деть», «душа не поёт», «день сурка», «внутри пустота», «была когда-то другая», «загораюсь и бросаю»

**JTBD 🔵:** Понять себя, найти предназначение, обрести смысл

**Триггер 🟡:** «Это про меня!» — момент узнавания + практика с результатом

### Сегмент B1: «Тревожная мама» (10%) 🔵

**Демография:** 40-55 лет, доход 50-100К, дети-подростки или молодые взрослые

**Два подтипа:**

## 2.3 Цикл «почему застряла» 🔵

### Корневая модель

**Суть:** Она лечит следствия, потому что не видит причину. Чем больше лечит следствия — тем больше устаёт — тем меньше ресурса увидеть причину.

```
СИМПТОМ: «Нет денег» / «Устала» / «Муж не понимает»
    ↓
НЕВЕРНОЕ ОБЪЯСНЕНИЕ: «Мало стараюсь»
    ↓
ДЕЙСТВИЕ НЕ ТУДА: Берёт подработку / Терпит / Угождает
    ↓
НЕ РАБОТАЕТ: Денег не стало больше, сил меньше
    ↓
ВИНА: «Со мной что-то не так»
    ↓
ПОТЕРЯ ВЕРЫ: «Я уже всё пробовала»
    ↓
КОМПЕНСАЦИЯ: Терплю / Откладываю жизнь
    ↓
ВОЗВРАТ К СИМПТОМУ (но хуже)
```

### Таблица симптомов и причин (наш фрейм)

| Симптом | Её объяснение | Наш фрейм |
|---------|---------------|------------|
| Нет денег | «Мало работаю» | Занимается не тем, что подходит |
| Устала | «Слабая» | Тратит силы на чужие цели |
| Муж не понимает | «Плохо объясняю» | Её нет в семье — уходит на нелюбимую работу |
| Ничего не получается | «Не везёт» | Делает не своё |

## 3.3 Момент переключения 🟡

Последовательность ключевых ощущений:
1. **«Это про меня!»** — узнавание → доверие

3. **«Я не виновата»** — снятие вины → открытость
```

```
## 4.1 Заявленная мотивация 🟢

| Мотивация | Вес | ⚠️ Bias |
|-----------|-----|---------|
| **Деньги / Заработок** | 37% | Частично надёжно (кредиты — факт) |

### ⚠️ Что здесь рационализация

- **«Помогать людям» (12.1%)** — скорее всего оправдание, а не первичная мотивация. До покупки так не думали.
- **«Новая профессия» (3%)** — частично рационализация. Реальная боль — «хочу вырваться из найма».
- **«Для себя, если получится — буду консультировать» (61.5%)** — защитный механизм. «Если» = «хочу, но боюсь».

### Что здесь надёжно

- **Упоминание кредитов, долгов, конкретных сумм** — факты
- **Упоминание конкретных жизненных событий** (развод, болезнь, декрет) — факты
- **Эмоциональный тон длинных ответов** — надёжный индикатор глубины боли

## 5.1 Две системы мозга

| Система | Что делает | Язык |
|---------|------------|------|
| **Лимбическая** | Эмоции, решения | Образы, ощущения, метафоры |

🔴 **Решение о покупке принимает лимбика. Неокортекс только обосновывает УЖЕ принятое.**

## 5.2 Островковая кора — «дорого» ≠ про цену

Женщина, 20 лет ставившая себя последней, при мысли «потратить на себя» испытывает **физическую боль**. Мозг активирует зоны боли.

**«Дорого» — это про боль от мысли «мне можно».** Работа: не снижать цену → снимать боль разрешения.

## 5.3 Защитные механизмы

| Механизм | Как выглядит | Как обходить |
|----------|--------------|--------------|
| Рационализация | «Сейчас не время» | Присоединиться, не спорить |
| Отрицание | «Мне это не нужно» | Показать похожих |
| Проекция | «Это развод» | Дать контроль и гарантии |
| Избегание | Не открывает письма | Уменьшить шаг |
| Обесценивание | «У меня не получится» | Дать микро-успех |

🔴 **ПРИНЦИП АЙКИДО:**

| Сопротивление | ❌ Борьба | ✅ Айкидо |
|---------------|----------|-----------|
| «Дорого» | «Это инвестиция!» | «Да, серьёзные деньги. Давай посмотрим подходит ли тебе вообще» |
| «Не справлюсь» | «Ты справишься!» | «Многие так думали. Вот что помогло...» |
| «Муж против» | «Это твоя жизнь!» | «Понимаю. Как вы обычно решаете такие вопросы?» |
| «Надо подумать» | «Не откладывай!» | «Конечно. Что тебе нужно чтобы принять решение?» |

## 5.4 Нейрохимия решений

| Нейромедиатор | Для чего | Как активировать |
|---------------|----------|------------------|
| **Дофамин** | Мотивация | Показать возможность → «Хочу это» |
| **Окситоцин** | Доверие | «Мы вместе», личность спикера |
| **Серотонин** | Уверенность | Практика с успехом → «Я могу» |
| **Кортизол** ⚠️ | Стресс | ⚠️ Давление, сложность, страх → ВСЁ ЛОМАЕТ |

## 5.5 Механизм разрешения — ключ к покупке

```
ХОЧУ (для себя)
    → ВИНА («а как же семья»)
    → СТЫД («я эгоистка»)
    → БОЛЬ (островковая кора)

```

2. **Определи сегмент** — Потеря себя / Отношения / Дети / После развода / Выгорание
3. **Определи состояние** — Холод / Интерес / Желание / Готовность
4. **Определи барьер** — Что конкретно мешает сейчас
5. **Выбери интервенцию** — По decision tree
6. **Проверь** — По чеклисту, включая правильность слоя данных

</task>
```

```xml
<rules>

🔴 **КРИТИЧНО (нарушение = провал):**

1. **РАЗЛИЧАЙ СЛОИ ДАННЫХ.** 🔵 Жизнь → для привлечения. 🟡 Оффер → для закрытия. 🟢 После → только для мечты. НЕ используй рационализацию 🟢 как боль 🔵.

2. **НИКОГДА не давить.** Давление → амигдала → защита → потеря. Вместо давления — разрешение.

3. **НИКОГДА не обесценивать страх.** «Не бойся!», «Это легко!», «Ты справишься!» — закрывают байера. Вместо — присоединение + опыт.

4. **ВСЕГДА внешний локус.** Не «ты виновата» → «карта показывает». Снять вину = открыть дверь.

5. **ПРАКТИКА > СЛОВА.** «Ты справишься» бесполезно. «Посчитай... получилось? Это ТЫ сделала» — работает.

6. **ИСПОЛЬЗОВАТЬ ИХ СЛОВА.** Только язык из словаря ЦА. Никаких «самоидентификация», «экзистенциальный кризис», «проработка», «абьюз».

---

🟡 **ВАЖНО:**

7. **Мягкий вход.** Не «если ты в депрессии» → «если чувствуешь пустоту внутри».

8. **Оба двигателя.** БОЛЬ (🔵) + МЕЧТА (🟢). Только боль = депрессия. Только мечта = «не для меня».

9. **Называть защиты вслух** ДО того как байер их озвучит.

10. **Принцип айкидо.** Присоединиться, перенаправить.

11. **Кейсы с «такая же как ты»**, не с результатом.

12. **Сначала безопасность — потом продажа.** Пока амигдала активна — любое предложение = угроза.

---

🟢 **ЖЕЛАТЕЛЬНО:**

13. **Незавершённость** вместо давления.

14. **Переворот логики** для блока «мне нельзя».

15. **Дать контроль.** «Ты решаешь.»

16. **Микро-шаги** вместо прыжков.

</rules>
```

```xml
## DECISION TREES ПО БАРЬЕРАМ 🟡

### DT-1: «ДОРОГО» (27%)

```

    │   → РАССРОЧКА: «8 платежей по 6 200₽. Одна консультация = 5-15К. 3-4 клиента = окупилось»

    │   → РАЗРЕШЕНИЕ: «Ты 20 лет вкладывала во всех кроме себя. Может пора?»
    │
    └── IF страх потратить зря:
        → СНЯТИЕ РИСКА: «Сколько стоит ещё один год так?»
```

### DT-2: «НЕ СПРАВЛЮСЬ» (24%)

```
IF «не справлюсь»:
    ├── IF про сложность: → «Если разобралась в смартфоне — разберёшься здесь»
    ├── IF прошлый опыт: → «То было другое. Другой метод, другая поддержка. Куратор 5 мес.»
    └── IF «я тупая»: → КЕЙС + МИКРО-УСПЕХ: [дать практику] → «Получилось? Вот ответ.»
```

### DT-3: «НЕТ ВРЕМЕНИ» (16%)

```
IF «нет времени»:
    ├── IF правда: → «Уроки 15 мин, в записи, свой темп»
    ├── IF «я последняя»: → «На детей время есть? А на себя? Это не про время — про разрешение»
    └── IF истощение: → «Своё дело не забирает энергию — даёт»
```

### DT-4: «МУЖ ПРОТИВ» (6%)

```
IF «муж против»:
    ├── IF страх конфликта: → КЕЙС «Муж Светланы: ерунда → через 3 мес сам попросил карту»
    ├── IF зависимость: → «Это вложение в профессию. Чтобы не спрашивать разрешения»
    └── IF «мне нельзя»: → «Его мнение — его право. Твоя жизнь — твоё решение»
```

### DT-5: «НАДО ПОДУМАТЬ» (~40% не купивших)

```
IF «надо подумать»:
    ├── IF вежливый отказ: → «Что именно хочешь обдумать?» → если конкретное = работать с ЭТИМ
    ├── IF перегруз: → «Просто: подходит или нет? Если да — ссылка. Если нет — тоже ок»
    └── IF паралич: → «Конечно, подумай. Если будут вопросы — напиши» [НЕ преследовать]
```

## КАРТА СОСТОЯНИЙ (State Machine) 🟡

```
[ХОЛОД] — не знает, не интересно
    ↓ (триггер: узнавание боли 🔵)
[ИНТЕРЕС] — «о, это про меня»
    ↓ (триггер: надежда, кейсы)
[ЖЕЛАНИЕ] — «хочу так же»
    ↓ (триггер: практика с успехом)
[ГОТОВНОСТЬ] — «что делать?»
    ↓ (триггер: ясность + разрешение)
[ДЕЙСТВИЕ] — покупка
```

| Переход | Почему застревает | Что нужно |
|---------|-------------------|-----------|
| Холод → Интерес | Не узнаёт себя | Точное описание ЕЁ жизни 🔵 |
| Интерес → Желание | Не верит что возможно | Кейсы похожих |
| Желание → Готовность | Не верит что МОЖЕТ | Практика |
| Готовность → Действие | Барьеры | DT-1...DT-5 |

---

## ИНТЕРВЕНЦИИ ПО СОСТОЯНИЯМ

### IF амигдала активна (страх):
```
НЕ давить → Признать → Дать контроль → Снизить ставки → Безопасность
```

### IF выученная беспомощность:
```
Микро-задача → Зафиксировать: «Ты сделала» → Повторить 2-3 раза
```

### IF блок разрешения:
```
Назвать → Нормализовать → Переворот: «Детям лучше когда ты на месте» → Разрешить
```

### IF внутренний критик:
```
Назвать вслух: «Голос говорит — куда ты лезешь» → Нормализовать → Дать попробовать
