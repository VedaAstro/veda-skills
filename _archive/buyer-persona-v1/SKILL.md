# Байер-персона Veda

## Когда использовать

- Работа с болями и возражениями клиентов (вебинар, диагностика, бот)
- Создание контента под конкретный сегмент аудитории
- Закрытие возражений (дорого, не справлюсь, нет времени, муж против)
- Квалификация лидов -- определение сегмента
- Выбор языка и формулировок для ЦА
- Написание скриптов дожима

## Ключевые правила

### Предупреждение о данных (3 bias)

Все данные из анкет КУПИВШИХ (9 520 анкет). Три систематических искажения:

| Bias | Суть |
|------|------|
| Рационализация после покупки | "Хочу стать астрологом" -- может быть оправданием, не реальной болью |
| Выборка только покупателей | Кто НЕ купил -- не попал в данные |
| Социальная желательность | "Помогать людям" завышено, "одиночество" занижено |

### 3 временных слоя данных

| Слой | Что это | Когда использовать |
|------|---------|--------------------|
| ЖИЗНЬ | Реальная ситуация ДО продукта | Рилсы, прогрев, привлечение |
| ОФФЕР | Возражения и триггеры при покупке | Вебинар, оффер, закрытие барьеров |
| ПОСЛЕ | Мотивации после покупки | Картина будущего (мечта), но осторожно с рационализацией |

### Сегменты аудитории

| Сегмент | Доля | Ядро |
|---------|------|------|
| A1: "Потерявшая себя" | 40% | Кризис идентичности, "кто я вне ролей" |
| A2: "В сложных отношениях" | 20% | Отношения / развод / одиночество |
| B1: "Тревожная мама" | 10% | Проблемы с детьми, контроль |
| B2: "После развода" | 5%+ | Перезапуск жизни, доход |

### 5 decision trees по барьерам

| DT | Барьер | Доля | Ключ |
|----|--------|------|------|
| DT-1 | "Дорого" | 27% | Рассрочка / блок разрешения / снятие риска |
| DT-2 | "Не справлюсь" | 24% | Кейс + микро-успех |
| DT-3 | "Нет времени" | 16% | 15 мин в записи / "это не про время" |
| DT-4 | "Муж против" | 6% | Кейс "муж сам попросил карту" |
| DT-5 | "Это не наука" | 5% | Пруфы + снятие ярлыка |

### Формула покупки (момент переключения)

```
Покупка = Боль(узнала себя) + Снятие вины(внешний локус) + Разрешение(мне можно) + Снятие риска(гарантия/рассрочка)
```

Главный запрос: "Не бросьте меня" (32%) -- поддержка важнее контента.

## Быстрый маршрут

1. Определи сегмент клиента (A1/A2/B1/B2)
2. Найди его боль в слое ЖИЗНЬ
3. Определи барьер (DT-1..DT-5)
4. Пройди по decision tree для этого барьера
5. Используй язык ЦА (декодируй эвфемизмы)
