# Чеклисты конверсии лендингов

## Регистрационный лендинг (opt-in)

### Структура
- [ ] Headline содержит конкретный РЕЗУЛЬТАТ, не тему
- [ ] Subheadline: кто, для кого, формат
- [ ] Форма: максимум 2-3 поля
- [ ] CTA видна без скролла на десктопе И мобайле
- [ ] CTA-текст = активный глагол ("Занять место" > "Зарегистрироваться")
- [ ] Фото спикера есть
- [ ] Минимум 2 отзыва с фото и именем
- [ ] Urgency реальная, не искусственная
- [ ] Нет навигации и внешних ссылок
- [ ] Нет автоплей видео со звуком

### Контент
- [ ] "Что ты получишь" -- результаты, не темы (3-5 пунктов)
- [ ] "Для кого" -- боли/ситуации из buyer-persona
- [ ] Credentials спикера -- конкретные числа (X учеников, Y консультаций)
- [ ] Все данные из плейбуков (не придуманные)
- [ ] Голос = Ирина ("дорогие мои", теплый наставник)

### Технические
- [ ] Mobile-first: кнопки >= 44px, шрифт >= 16px
- [ ] Загрузка < 2 секунд (PageSpeed)
- [ ] Изображения в WebP, lazy load
- [ ] Форма работает (тест отправки)
- [ ] UTM-метки прокидываются
- [ ] Аналитика настроена (визиты, регистрации, CR)

## Офферный лендинг (sales page)

### Структура (14 блоков)
- [ ] Pre-headline квалифицирует аудиторию
- [ ] Hero: headline = трансформация, не продукт
- [ ] Problem: человек узнает СЕБЯ (конкретные ситуации)
- [ ] Dream: яркий образ ПОСЛЕ (конкретные детали, не "счастье")
- [ ] Solution bridge: уникальный механизм, почему это работает
- [ ] Product: название, visual, promise statement
- [ ] Modules: benefit-заголовки (не "Модуль 1: Основы")
- [ ] Social proof: минимум 5 отзывов с конкретными результатами
- [ ] Value stack: сумма бонусов > стоимость продукта
- [ ] Pricing: якорение + рассрочка как основная цена
- [ ] Guarantee: конкретное условие и срок
- [ ] About: история трансформации эксперта, не CV
- [ ] "Для кого / Не для кого"
- [ ] FAQ: закрывает топ-5 возражений (DT-1..DT-5)

### CTA
- [ ] CTA повторяется 3-4 раза по длине страницы
- [ ] Текст CTA -- активный глагол
- [ ] Кнопка контрастного цвета
- [ ] Под CTA -- снятие риска ("14 дней возврат" / "рассрочка без %")

### Social Proof
- [ ] Кейсы = "такая же как ты" (совпадение по ситуации)
- [ ] Формула отзыва: [Скептицизм до] + [Результат] + [Жизнь после]
- [ ] Есть количественный proof (X учеников)
- [ ] Есть скриншоты из мессенджеров
- [ ] Нет придуманных кейсов (только из плейбука)

### Оффер
- [ ] Hormozi value stack: каждый бонус закрывает возражение
- [ ] Якорение цены (сравнение с альтернативами)
- [ ] Рассрочка как основная цена
- [ ] Окупаемость посчитана и показана
- [ ] Urgency в конце -- реальная (дедлайн набора / повышение цены)

### Копирайтинг
- [ ] Фреймворк выбран (PAS / BAB / Hormozi)
- [ ] Голос = Ирина (теплый наставник)
- [ ] Внешний локус вины ("не твоя ошибка")
- [ ] JTBD: emotional + identity уровни адресованы
- [ ] Формула покупки: боль + снятие вины + разрешение + снятие риска
- [ ] Нет длинных тире

### Технические
- [ ] Mobile-first: все кнопки, формы, таблицы адаптивны
- [ ] Загрузка < 2 секунд
- [ ] Accordion/collapse для длинных списков
- [ ] Sticky CTA на мобайле (опционально)
- [ ] UTM-метки + аналитика сквозная
- [ ] Проверено на реальном iPhone

## Красные флаги (если есть хоть один -- НЕ запускать)

- [ ] Headline = название продукта или тема, не результат
- [ ] 0 отзывов с конкретными результатами
- [ ] Нет гарантии на офферном лендинге с чеком > 50К
- [ ] CTA не видна без скролла на мобайле
- [ ] Фейковый таймер или скарсити
- [ ] Кейсы/данные придуманные (не из плейбука)
- [ ] Нет мобильной версии
- [ ] Загрузка > 4 секунд

## EXIT CHECKLIST — проверка перед деплоем

Прогнать **каждый** пункт. Если хоть один — красный: **не деплоить**.

### Функциональность
- [ ] Форма регистрации/лида работает (тест-сабмит, получить ответ от API)
- [ ] После сабмита — редирект или сообщение (нет "белого экрана")
- [ ] Кнопки CTA ведут на нужный якорь/страницу
- [ ] Sticky CTA (мобайл) не перекрывает форму

### Скорость (открыть DevTools → Network)
- [ ] **0 console errors** (React hydration, 404, CORS)
- [ ] Нет JPG/PNG — только WebP (кроме SVG-лого)
- [ ] Иконки до 50KB, hero-фото до 100KB
- [ ] Нет кириллических имён файлов в URL (проверить в Network tab)
- [ ] Hero-изображение: `fetchPriority="high"` или `priority`
- [ ] Изображения ниже fold: `loading="lazy"`
- [ ] Таймер обратного отсчёта: нет мигания при загрузке (hydration fix)

### Core Web Vitals (проверить через PageSpeed Insights или DevTools Lighthouse)
- [ ] LCP < 2.5 сек
- [ ] CLS < 0.1 (нет прыжков контента при загрузке)
- [ ] INP < 200 мс (кнопки реагируют мгновенно)

### Мобайл (проверить на реальном iPhone или DevTools 390px)
- [ ] Текст не обрезается, нет горизонтального скролла
- [ ] Кнопки >= 44px по высоте
- [ ] Шрифт >= 16px (иначе iOS зумирует форму)
- [ ] Фото спикера не деформировано (object-fit: cover)
- [ ] Форма помещается на экране без скролла

### SEO / метаданные
- [ ] `<title>` и `<description>` заполнены в layout.tsx
- [ ] OG-теги для соцсетей (title, description, image)
- [ ] `/wheel` редирект настроен (если нужен)

### Юридика (обязательно для лендингов с формой)
- [ ] Чекбокс согласия ПД в форме
- [ ] Ссылки: оферта, политика, согласие в футере
- [ ] Дисклеймер под кейсами: "Результаты индивидуальны..."

### Финальный визуал
- [ ] Скриншот на desktop 1440px — нет пустых блоков, нет overflow
- [ ] Скриншот на mobile 390px — нет деформаций
- [ ] Фото Ирины не приплюснуто (aspect ratio сохранён)
