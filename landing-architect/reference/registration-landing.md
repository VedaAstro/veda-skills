# Регистрационный лендинг (opt-in)

Цель: одно действие -- зарегистрироваться на бесплатное мероприятие.
CR бенчмарк: warm 30-50%, cold 15-25%, топ до 59%.

## 8 обязательных блоков (в этом порядке)

### БЛОК 1. Hero (Above the fold)

5 обязательных элементов -- все видны без скролла:

1. **Benefit-driven headline** -- конкретный результат, НЕ тема
   - Плохо: "Вебинар по ведической астрологии"
   - Хорошо: "Узнай, какие 3 планеты сейчас управляют твоей судьбой"
   - Хорошо: "За 90 минут ты поймешь, почему жизнь идет не туда -- и что с этим делать"

2. **Subheadline** -- кто, для кого, формат, почему сейчас
   - "Бесплатный мастер-класс Ирины Чайки для тех, кто хочет разобраться в себе через натальную карту"

3. **Дата / время / формат**
   - Для живого: конкретная дата + часовой пояс
   - Для автовебинара: "Ближайший эфир через X минут" или "Выбери удобное время"

4. **CTA-кнопка** -- контрастный цвет, активный глагол
   - "Занять место бесплатно" > "Зарегистрироваться"
   - "Хочу на мастер-класс" > "Отправить"

5. **Hero visual** -- фото Ирины (профессиональное, теплое) или тематическое изображение

### БЛОК 2. Что ты получишь (3-5 bullet points)

НЕ "темы вебинара", а конкретные результаты. Каждый пункт = мини-трансформация.

Формат: "Ты узнаешь / получишь / поймешь..."

Пример для прогревочного вебинара:
- Узнаешь, в каком кармическом периоде ты сейчас -- и почему одни годы легко, другие тормозят
- Поймешь, какая сфера жизни сейчас требует внимания (и что будет, если продолжать игнорировать)
- Получишь конкретный инструмент для чтения своей карты -- без базового образования

Максимум 5 пунктов (когнитивная перегрузка при 6+).

### БЛОК 3. Для кого этот вебинар

"Этот мастер-класс для тебя, если..." + 3-4 боли/ситуации из buyer-persona:

Сегмент A1 (Потерявшая себя, 40%):
- Чувствуешь, что живешь не свою жизнь
- Выполняешь роли (мама, жена, сотрудник), но не понимаешь, кто ты без них

Сегмент A2 (Сложные отношения, 20%):
- Отношения зашли в тупик или закончились
- Повторяются одни и те же сценарии с разными партнерами

Квалифицирует аудиторию + человек узнает себя = эмоциональное вовлечение.

### БЛОК 4. Спикер (Authority)

- Профессиональное фото Ирины
- 3-5 конкретных credential:
  - Основатель Академии ведической астрологии
  - 5000+ учеников за X лет
  - X+ консультаций
  - Создатель НейроАстролога (AI-инструмент)
- Короткая личная история трансформации (1-2 предложения)
- В астро-нише личная история работает ЛУЧШЕ сухих регалий

### БЛОК 5. Social Proof

- 2-3 отзыва участников прошлых вебинаров (с фото и именем)
- Количественный proof: "Более X человек уже посмотрели"
- Скриншоты из чата / мессенджера (высокое доверие)

Формула сильного отзыва: [Скептицизм до] + [Что получила на вебинаре] + [Что изменилось]

### БЛОК 6. Urgency / Scarcity

Только РЕАЛЬНОЕ:
- Countdown timer для живого мероприятия
- "Осталось X мест" -- только если правда ограничено
- "Запись НЕ будет" -- если правда
- "Бонус за регистрацию до [дата]"

Фейковые таймеры и скарсити = токсичное доверие, разрушает бренд.

### БЛОК 7. Форма регистрации (повторный CTA)

- Минимум полей: имя + email (или имя + телефон)
- Каждое дополнительное поле = -5-10% конверсии
- CTA повторяет текст из Hero
- Под формой: "Бесплатно. Без обязательств."

### БЛОК 8. FAQ (для скептиков, опционально)

Не для информации -- для закрытия возражений:
- "Это бесплатно?" -- Да, полностью
- "Будет запись?" -- [да/нет с условием]
- "Нужны знания астрологии?" -- Нет, все объясню с нуля
- "Сколько длится?" -- 60-90 минут

## Что убивает конверсию

- Форма с 5+ полями
- Размытый заголовок ("Вебинар по астрологии")
- Нет конкретики -- что именно узнаю
- Нет фото спикера
- Нет urgency
- Навигация и внешние ссылки (уводят с регистрации)
- Плохая мобильная версия
- Автоплей видео со звуком

## Пример текста Hero для Академии экспертов (прогрев-вебинар)

```
PRE-HEADLINE: Бесплатный мастер-класс Ирины Чайки
HEADLINE: Как астрологи выходят на доход 80-150 тысяч в месяц -- без холодных продаж и самозванца
SUBHEADLINE: За 90 минут ты увидишь систему, которая превращает знания в стабильную практику. Даже если сейчас у тебя 0 клиентов.
CTA: Занять место бесплатно
DATE: 15 апреля, 19:00 МСК | Онлайн
```

## Пример текста Hero для НейроАстролога (прогрев-вебинар)

```
PRE-HEADLINE: Бесплатная демонстрация
HEADLINE: AI-помощник, который готовит астрологический разбор за 5 минут вместо 8 часов
SUBHEADLINE: Посмотри, как НейроАстролог на базе ведической традиции создает 15-страничный PDF с персональной интерпретацией -- в реальном времени.
CTA: Хочу увидеть демонстрацию
```
