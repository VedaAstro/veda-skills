# Копирайтинговые фреймворки для лендингов

## PAS (Problem - Agitate - Solution)

Лучший для болевого входа. Основной фреймворк для астро/эзо ниши.

### Формула
```
Problem:  Конкретная боль ("не понимаю, почему жизнь стоит на месте")
Agitate:  Усиление ("каждый год одно и то же, отношения не строятся, деньги не идут")
Solution: Продукт как выход ("натальная карта покажет причину -- и даст маршрут")
```

### Пример для сегмента A1 (Потерявшая себя, 40%)
```
P: Ты выполняешь роли -- мама, жена, сотрудник.
   Но когда остаешься одна, не понимаешь: кто я без всего этого?

A: С каждым годом ощущение пустоты нарастает.
   Ты пробуешь новое -- курсы, терапию, медитации.
   Но ничего не цепляет, потому что ты ищешь ответ снаружи.
   А он -- в твоей карте. В задачах, с которыми ты пришла.

S: Натальная карта показывает твое предназначение -- конкретно, без гаданий.
   За 90 минут ты увидишь, зачем ты здесь и что делать дальше.
```

### Пример для сегмента A2 (Сложные отношения, 20%)
```
P: Отношения зашли в тупик. Или закончились.
   И ты не понимаешь -- почему повторяется одно и то же с разными людьми.

A: Психолог говорит "проработай детские травмы".
   Подруги говорят "забей, найдешь нового".
   Но ты чувствуешь, что тут что-то глубже. Какой-то сценарий, который ты не выбирала.

S: В карте видно: какие кармические задачи ты решаешь через отношения.
   Не "терпи" и не "уходи" -- а конкретный маршрут. Кто ты в паре и что с этим делать.
```

## BAB (Before - After - Bridge)

Идеален для трансформационных продуктов. Покупают "новую версию себя".

### Формула
```
Before:  Текущая реальность (конкретные детали повседневности)
After:   Желаемое состояние (яркое и конкретное, НЕ абстрактное "счастье")
Bridge:  Продукт = путь между Before и After
```

### Пример для Академии экспертов
```
BEFORE:
Ты знаешь астрологию. Можешь разобрать карту.
Но когда знакомая просит "посмотри мне" -- делаешь бесплатно.
Назвать цену -- неловко. Брать деньги за "хобби" -- стыдно.
3 клиента в месяц, если повезет. И все -- через знакомых.

AFTER:
Ты просыпаешься и видишь 3 заявки.
НейроАстролог за 5 минут готовит персональный разбор.
Ты проводишь консультацию уверенно -- система 7 направлений дает ответ на любой запрос.
Клиент уходит с планом. Возвращается через месяц. Рекомендует подругам.
80-150 тысяч в месяц -- стабильно, без холодных продаж.

BRIDGE:
Академия экспертов Ирины Чайки -- это не "еще один курс".
Это система: 7 направлений + AI-инструмент + 3 реальных клиента от Академии.
За 5 месяцев ты переходишь из "знаю астрологию" в "мне платят за экспертизу".
```

## Hormozi Value Equation

Для построения оффера и value stack.

### Формула
```
Ценность = (Желаемый результат x Вероятность достижения)
         / (Временная задержка x Усилие/Жертва)

Задача: ПОДНЯТЬ числитель, СНИЗИТЬ знаменатель.
```

### Как применять на лендинге

| Переменная | Что делать на лендинге | Пример |
|-----------|----------------------|--------|
| Желаемый результат ↑ | Headline = яркая трансформация | "Стабильный доход 80-150К" |
| Вероятность ↑ | Social proof, кейсы, гарантия | "347 выпускников, 78% зарабатывают" |
| Время ↓ | Показать скорость результата | "Первые клиенты через 30 дней" |
| Усилие ↓ | Показать простоту процесса | "15 минут в день, все в записи" |

### Value Stack (Hormozi Offer Stack)

Каждый элемент стека закрывает конкретный барьер:

```
ОСНОВНОЙ ПРОДУКТ: Академия 7Н (5 мес, 7 направлений)
  -> Закрывает: "Хочу знания"
  -> Ценность: курс обучения

БОНУС 1: НейроАстролог (50 консультаций)
  -> Закрывает: "Долго готовить разбор" (время ↓)
  -> Ценность: экономия 400+ часов

БОНУС 2: 3 клиента от Академии
  -> Закрывает: "Где взять клиентов" (вероятность ↑)
  -> Ценность: первый заработок

БОНУС 3: Практики с бадди
  -> Закрывает: "Не справлюсь одна" (усилие ↓)
  -> Ценность: поддержка

БОНУС 4: Шаблоны интерпретаций
  -> Закрывает: "Не хватает знаний" (усилие ↓)
  -> Ценность: готовые формулировки

ИТОГО: сумма ценностей >> цена продукта
ЦЕНА: 79 900 (рассрочка 4 439/мес)
```

Правило: сумма отдельных ценностей бонусов > стоимость основного продукта.

### Якорение цены

Сравнение с альтернативами:
- "Одна консультация стоит 5000. Окупаемость = 16 консультаций = 5 недель"
- "Без системы ты потратишь на курсы 200-300К за 2 года. И все равно не выйдешь на доход"
- "Рассрочка 4 439/мес -- дешевле кофе с десертом каждый день"

## StoryBrand (Donald Miller)

Нарратив лендинга: покупатель = герой, эксперт = проводник.

### 7 элементов StoryBrand на лендинге

1. **Герой** (ученица) хочет X
   -> Headline: "Ты хочешь стать экспертом, которому платят"

2. **Препятствие** (внешнее + внутреннее)
   -> Problem section: "Но не знаешь как, и кажется что ты еще не готова"

3. **Проводник** (Ирина) с эмпатией + авторитетом
   -> Speaker section: "Я прошла тот же путь + 5000 учеников"

4. **План** (простой, 3 шага)
   -> Product section: "Шаг 1: Диагностика. Шаг 2: Обучение 7Н. Шаг 3: Первые клиенты"

5. **Призыв к действию** (прямой)
   -> CTA: "Записаться в Академию"

6. **Успех** (если действует)
   -> Dream section: "80-150К, стабильная практика, признание"

7. **Неудача** (если не действует)
   -> Urgency: "Через год -- в том же месте. Только старше и с синдромом самозванца"

### Голос Ирины в StoryBrand

Проводник = НЕ "я крутая, слушай меня".
Проводник = "я понимаю тебя, я была там же, вот план".

Теплый наставник: "Дорогие мои", "на самом деле", "согласны?"
НЕ: "девочки", "красотки", копирайтерский тон.

## 4P (Picture - Promise - Proof - Push)

Для блока social proof и testimonials.

### Формула
```
Picture:  Нарисуй образ результата ("Представь: ты открываешь телефон и видишь 3 заявки")
Promise:  Что конкретно получит ("Первые клиенты через 30 дней")
Proof:    Доказательства ("Вот Маша -- 6 недель назад была на твоем месте...")
Push:     CTA с urgency ("Набор закрывается 20 апреля. Записаться сейчас")
```

## Выбор фреймворка по ситуации

| Ситуация | Основной фреймворк | Дополнительный |
|----------|-------------------|---------------|
| Холодный трафик, высокая боль | PAS | + StoryBrand |
| Теплый трафик, знают эксперта | BAB | + 4P |
| Высокий чек (VIP 139 900) | Hormozi Value + BAB | + StoryBrand |
| Инструмент (НейроАстролог) | 4P | + Hormozi |
| Бесплатный вебинар (opt-in) | PAS короткий | + 4P micro |
