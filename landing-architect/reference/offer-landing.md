# Офферный лендинг (Sales Page)

Цель: продать продукт или получить заявку на покупку.
CR бенчмарк: warm 3-8%, cold 0.5-2%.
Длина зависит от чека и температуры: холодный + высокий чек = длинная страница.

## 14 блоков (в этом порядке)

### БЛОК 1. Pre-headline (крючок-квалификатор)

Одна фраза НАД главным заголовком. Квалифицирует аудиторию:

Академия экспертов:
- "Для действующих и начинающих астрологов, которые хотят выйти на стабильный доход"

НейроАстролог:
- "Для астрологов, которые устали тратить часы на подготовку одной консультации"

### БЛОК 2. Hero Section

5 элементов (все видны до скролла):

1. **Headline** -- трансформация или конкретный результат
2. **Subheadline** -- уточнение: для кого, за какое время, через что
3. **Hero visual** -- мокап продукта / фото Ирины / скриншот результата
4. **CTA** -- первая кнопка (с ценой или без -- зависит от стратегии)
5. **Micro social proof** -- "Уже прошли X человек" или рейтинг

Формулы заголовков:

```
[Результат] без [страх] за [срок]
"Первые 3 клиента через 30 дней -- без холодных продаж"

Как [желаемое] даже если [возражение]
"Как стать астрологом-экспертом, даже если ты интроверт"

Из [текущее] в [желаемое] через [механизм]
"Из увлеченного астролога -- в эксперта, которому платят"
```

### БЛОК 3. Problem / Nightmare Section

Зеркало болей -- человек УЗНАЕТ СЕБЯ. Конкретные ситуации, не абстракции.

PAS: Problem -> Agitate -> (Solution в следующем блоке)

Академия экспертов:
```
Ты прошла 5 курсов по астрологии. Умеешь строить карты. Но клиентов нет.
Каждый раз думаешь: "Может, мне нужен еще один курс по прогрессиям?"
Знакомые просят разобрать карту бесплатно -- а заплатить никто не готов.
И в какой-то момент начинает казаться, что астрология -- это хобби. Навсегда.
```

НейроАстролог:
```
Интерпретация одной карты занимает 3-4 часа.
Клиент ждет. Ты копаешься в конспектах.
Берешь максимум 3 консультации в неделю -- потому что больше физически не потянуть.
А на онлайн-площадках уже появляются "астрологи", которые делают разборы за 20 минут с помощью ChatGPT.
```

Agitate (усиление):
- "Каждый месяц без системы = потеря X клиентов и Y денег"
- "Через год ты будешь в том же месте -- только еще больше разочарована"

### БЛОК 4. Dream / Vision Section

BAB: яркий образ жизни ПОСЛЕ. Конкретные детали, не абстракции.

"А что если..." (переход)

```
Ты просыпаешься и видишь 3 заявки от клиентов.
Открываешь НейроАстролог -- за 5 минут готова персональная интерпретация.
Проводишь консультацию уверенно, потому что видишь всю карту системно.
Клиент уходит с конкретным планом -- и возвращается через месяц с результатом.
И рекомендует тебя подругам.
```

### БЛОК 5. Solution Bridge

Почему именно этот подход работает:
- Представление уникального механизма (7Н система, НейроАстролог на базе ведической традиции)
- Что пробовали раньше и почему не сработало
- "Secret sauce": что отличает от 100 других курсов

Для Академии:
- Не "еще один курс по астрологии"
- Система 7 направлений + AI-инструмент + реальные клиенты от Академии
- Единственная программа, которая дает и знания, и инструмент, и клиентов

### БЛОК 6. Product Presentation

- Название программы + tagline
- Visual mockup (обложка курса, скриншоты платформы, пример PDF НейроАстролога)
- Promise statement: одна фраза = суть трансформации
- "Академия экспертов -- система, которая превращает астрологические знания в стабильную практику и доход"

### БЛОК 7. What's Inside (модули/содержание)

Список модулей с BENEFIT-заголовками (не "Модуль 1: Основы"):

| Модуль | Benefit-заголовок | Результат |
|--------|------------------|-----------|
| М1 Предназначение | "Как увидеть свой путь в карте" | Понимаешь зачем ты здесь |
| М2 Карма | "Почему повторяется одно и то же" | Выходишь из сценариев |
| М3 Финансы | "Где в карте твой доход" | Новый канал заработка |
| М4 Здоровье | "Откуда взять энергию" | Сила и устойчивость |
| М5 Отношения | "Почему с ним так" | Зрелые отношения |
| М6 Семья | "Как не повторить сценарий мамы" | Своя линия + помощь детям |
| М7 Прогнозы | "Когда действовать, когда ждать" | Движение в потоке |

Accordion / раскрывающийся список -- чтобы не перегружать.

### БЛОК 8. Social Proof (основной блок)

Порядок расстановки по силе доверия:

1. **Количественное:** X учеников, X кейсов
2. **Кейс-трансформации** (до/после, конкретный результат):
   - Формула: [Скептицизм до] + [Конкретный результат] + [Жизнь после]
   - Плохо: "Очень понравился курс, рекомендую!"
   - Хорошо: "Пришла с 0 клиентов. Через 6 недель -- 4 платных консультации в неделю, 68 000 за месяц"
3. **Текстовые отзывы** с фото + имя
4. **Скриншоты из мессенджеров** (высокое доверие)
5. **Видео-отзывы** (максимальное доверие)

Кейс = "такая же как ты". Совпадение по СИТУАЦИИ целевого сегмента, не по результату.

### БЛОК 9. Value Stack / Offer Stack (Hormozi)

Принцип: сумма бонусов > стоимость основного продукта.
Каждый бонус закрывает конкретное возражение.

Структура:
```
Основной продукт (Академия 7Н)          -- ценность X руб.
+ Бонус 1: НейроАстролог (50 конс.)     -- ценность Y руб. (закрывает "долго готовить")
+ Бонус 2: 3 клиента от Академии        -- ценность Z руб. (закрывает "где взять клиентов")
+ Бонус 3: Практики с бадди             -- ценность W руб. (закрывает "не справлюсь одна")
+ Бонус 4: Шаблоны интерпретаций        -- ценность V руб. (закрывает "не хватает знаний")

ИТОГО ценность: [сумма] руб.
Ваша цена сегодня: 79 900 руб. (рассрочка 4 439/мес)
```

Hormozi Value Equation:
```
Ценность = (Желаемый результат x Вероятность достижения)
         / (Временная задержка x Усилие/Жертва)
```
Задача лендинга: поднять числитель (яркий результат + proof), снизить знаменатель (быстро + без боли).

### БЛОК 10. Pricing & CTA

- Тарифная таблица (из `veda-offer`): 3Н / 7Н / VIP
- Целевой тариф выделен визуально (7Н, 79 900)
- Якорение: "Стоимость одной консультации = 5000. Окупаемость = 16 консультаций = 5 недель"
- Рассрочка как основная цена: "4 439 руб./мес"
- CTA: "Записаться в Академию" / "Начать обучение"

### БЛОК 11. Guarantee (Гарантия)

Отдельный визуальный блок (печать / значок):

Формулировка Hormozi: "Если через [срок] вы не получите [конкретный результат], я [действие]"

Варианты для Академии:
- "14 дней на то, чтобы убедиться. Не подойдет -- верну деньги без вопросов"
- "Гарантия результата: если после М1 вы не увидите свое предназначение в карте -- полный возврат"

Гарантия снимает ПОСЛЕДНИЙ барьер перед покупкой. Без нее "Дорого" (DT-1, 27%) закрыть сложнее.

### БЛОК 12. About (Спикер/Эксперт)

НЕ CV, а история трансформации:
- "Я сама прошла путь от [точка А] до [точка Б]"
- Фото Ирины
- Конкретные credentials (учеников, лет, консультаций)
- Связь с нишей: почему именно ведическая, а не западная

### БЛОК 13. "Для кого / Не для кого"

Четкая квалификация -- снижает возвраты, увеличивает конверсию:

Для кого:
- Знаешь основы астрологии и хочешь монетизировать
- Хочешь помогать людям через карту
- Готова уделять 15-30 мин в день

НЕ для кого:
- Ищешь "волшебную таблетку" без усилий
- Не готова практиковаться
- Хочешь только для себя (тогда тариф 3Н)

### БЛОК 14. FAQ + Urgency + Финальный CTA

FAQ -- закрытие 5 decision trees из buyer-persona:

| Возражение | FAQ вопрос | Ответ-техника |
|-----------|-----------|---------------|
| DT-1 "Дорого" (27%) | "Есть рассрочка?" | Да, 4 439/мес. Окупаемость за 5 недель |
| DT-2 "Не справлюсь" (24%) | "Нужны ли знания?" | С нуля. Бадди поддержит |
| DT-3 "Нет времени" (16%) | "Сколько времени?" | 15-30 мин/день, все в записи |
| DT-4 "Муж против" (6%) | "Как объяснить семье?" | Кейс "муж сам попросил карту" |
| DT-5 "Не наука" (5%) | "Это гадание?" | 5000 лет джйотиша, 30 астрологов, AI-система |

Urgency в конце (только реальная):
- "Набор закрывается [дата]"
- "Цена повышается [дата]"
- "Осталось X мест в потоке"

Финальный CTA с повтором гарантии + ключевого бонуса.

## CTA повторяется 3-4 раза

По длине страницы CTA-кнопка появляется:
1. Hero (блок 2)
2. После Social Proof (блок 8)
3. После Pricing (блок 10)
4. Финальный (блок 14)

Текст может варьироваться, но действие одно.

## Специфика для НейроАстролога (инструментальный оффер)

НейроАстролог -- инструмент, не трансформационный продукт.
Упор на конкретные функции + ROI:

- Сколько часов экономит в неделю (8-16 часов -> 5 минут)
- Сколько дополнительных клиентов можно взять
- Во сколько обходится час работы астролога
- Demo: показать PDF, который генерирует AI

НЕ продается отдельно от обучения (этика Ирины).
На лендинге = часть value stack Академии, не отдельный продукт.
