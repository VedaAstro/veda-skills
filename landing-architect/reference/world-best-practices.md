# Мировые практики: лендинги для инфопродуктов

Источники: Unbounce (44K лендингов), Copyhackers, DigitalMarketer, данные 2024-2026.

## Топ-практики и что они делают на лендингах

### Alex Hormozi -- структура оффера на лендинге

**Proof-Promise-Plan** -- последовательность блоков критична:
1. **Proof** -- кейсы и доказательства ПЕРВЫМИ (создают восприимчивость к promise)
2. **Promise** -- обещание результата (после proof воспринимается как реальное)
3. **Plan** -- план достижения (снижает perceived effort)

**Value Stack на лендинге:**
- Каждый элемент оффера показывается по одному
- Для каждого -- отдельная ценность в рублях
- Каждый бонус закрывает конкретное возражение
- Суммарная ценность >> реальная цена
- Price Anchor: сначала высокая цена (сумма), потом реальная

**Минимализм:** лендинг acquisition.com -- заголовок с Dream Outcome, инфо, одна кнопка. Ноль отвлекающих элементов.

### Russell Brunson -- Hook-Story-Offer

**Hook-Story-Offer** -- скелет ЛЮБОГО лендинга:
- **Hook** = заголовок + подзаголовок, вызывающий любопытство или узнавание
- **Story** = история трансформации (спикера или ученика)
- **Offer** = оффер с таким соотношением цена/ценность, что невозможно отказаться

Когда лендинг не конвертит -- проблема ВСЕГДА в одном из трех. Диагностика: что слабее всего?

**Epiphany Bridge** -- структура истории на лендинге:
Цель: привести читателя к "моменту прозрения" через историю, а не логические аргументы.
Структура: Background (кто был) -> Journey (что случилось) -> New Opportunity (что открыл) -> Framework (почему это работает) -> Achievement (результат).

Применение: блок "О спикере", кейсы учениц, Problem -> Solution bridge.

### Joanna Wiebe (Copyhackers) -- Voice of Customer

**Message Mining** -- метод написания текстов для лендинга:

Копирайтер НЕ придумывает формулировки. Он НАХОДИТ их в словах реальных клиентов.

**Где брать:**
- Анкеты купивших (у Vedantica 9520 анкет -- золото)
- Отзывы на своих и конкурентных продуктах
- Форумы, чаты, комментарии ЦА
- Звонки менеджеров (транскрипции)

**Как применять:**
1. Собрать 50-100 цитат клиентов о проблеме и результате
2. Найти повторяющиеся формулировки
3. Использовать их КАК ЕСТЬ в заголовках и тексте лендинга
4. A/B тестировать: VoC-заголовок vs придуманный

Результат: 3.5x рост конверсий у клиентов (Canva, Intuit, Sprout Social).

**Для Vedantica:** вместо придумывания "Узнай свое предназначение" -- найти в анкетах 9520 клиенток как ОНИ формулируют свою боль и желание. Это и есть заголовок.

### Jeff Walker (Product Launch Formula) -- последовательность

**Ключевой инсайт для лендингов:**
Лендинг конвертит ЛУЧШЕ, если перед ним была серия касаний (прогрев).
К моменту sales page человек уже:
- Знает эксперта (видео, вебинар, марафон)
- Получил ценность бесплатно
- Увидел кейсы

Следствие: офферный лендинг для ТЕПЛОЙ аудитории (после вебинара/марафона) и для ХОЛОДНОЙ -- разные по длине и структуре:
- Теплая: можно короче, меньше proof, больше оффер
- Холодная: длинный, много proof, подробное объяснение механизма

### Frank Kern -- персонализация

**Behavioral Dynamic Response:**
Разные люди видят разные элементы лендинга в зависимости от поведения:
- Пришел из VK -> один заголовок
- Пришел из email -> другой
- Уже был на вебинаре -> третий (без блока "О спикере", сразу к офферу)

Даже простая сегментация (UTM -> разный headline) дает +40% к CR.

### Sam Ovens -- радикальный минимализм

**Принципы дизайна лендинга:**
- Первое что видишь = цель, без двусмысленности
- Нет навигации (уводит с конверсии)
- Максимум 2 шрифта
- Минимум 14px
- Кастомные фото (не стоковые)
- Каждая страница = ОДНО действие
- Убрать навигацию = +27% конверсия

## Доказанные факторы конверсии (данные, не мнения)

### Unbounce (44K лендингов, 2024-2026)

| Фактор | Эффект | Комментарий |
|--------|--------|-------------|
| Простой язык (5-7 класс) | **+109%** (11.1% vs 5.3%) | Короткие предложения, бытовые слова |
| Персонализированные CTA | **+202%** | "Хочу на мастер-класс" vs "Отправить" |
| Отзывы с результатами | до **+270%** | Конкретные цифры > "мне понравилось" |
| Видео на лендинге | **+86%** | Видео спикера / отзыв ученицы |
| Меньше полей формы | **+160%** | 11 полей -> 4 поля |
| Видео-отзывы vs текстовые | **+80-86%** | Максимальное доверие |
| 1 секунда задержки | **-7%** | WebP, lazy load, минимум JS |
| Загрузка 1 сек vs 5 сек | **3x разница** | Цель < 2 сек |
| Убрать навигацию | **+27%** | Нет меню = нет выхода с лендинга |
| 40+ лендингов vs 1-5 | **+500%** | Сегментация = разные лендинги для разных ЦА |

### Конверсия по источникам трафика
| Источник | CR |
|----------|-----|
| Email | 19.3% |
| Instagram | 17.9% |
| Facebook | 13% |
| Google Paid | 11.3% |

### Длина лендинга по чеку и температуре

| Чек + температура | Длина | Почему |
|-------------------|-------|--------|
| Бесплатное (регистрация) | Короткий, 1-3 экрана | Барьер низкий, не надо убеждать долго |
| Low-ticket + теплый | Короткий-средний | Уже знают эксперта |
| Mid-ticket + теплый | Средний (10 блоков) | Нужен proof + value stack |
| High-ticket + холодный | Длинный (14+ блоков) | Каждое возражение закрыть. Кейс: 10x длина = +363% |

## Специфика русского рынка (2025-2026)

### Каноническая структура на GetCourse (10 блоков)
1. Оффер (первый экран) -- название, тип, дата, CTA, фото эксперта
2. Для кого -- сегменты с выгодами
3. Проблемы -- список болей
4. Результаты -- что получит
5. Программа по модулям -- с результатом после каждого
6. Формат работы -- скриншоты ЛК
7. Отзывы/кейсы -- имя, фото, город, видео
8. Тарифы -- 2-3 варианта, рекомендуемый выделен
9. Гарантии -- условия возврата
10. Об эксперте -- достижения, фото

### Отличия Рунет vs Запад

| | Рунет | Запад |
|---|-------|-------|
| Канал лидов | Telegram + VK | Email + Instagram |
| Рассрочки | Обязательно (25% выручки) | Опционально |
| Платформа | GetCourse | Kajabi / ClickFunnels |
| Доверие | Личность эксперта + отзывы | Бренд + сертификации |
| Регистрация | Telegram-бот (минимальное трение) | Email capture |

### Конкурент: Jataka (jataka.expert)
- 12 300+ учеников, 12 лет
- Лид-магнит: 3-дневный бесплатный курс через Telegram
- Структура лендинга: слайдер -> соцдоказательство (числа) -> 4 блока ценности -> преподаватели -> видео-отзывы -> гарантия 100% -> FAQ (8 вопросов)
- Регистрация через Telegram (не email!)

### Кейс школы астрологии (vc.ru) -- конкретные данные
- 10 000 лидов/мес при 250 руб./лид
- Переориентировались с "интересующихся астрологией" на "женщины ищущие смену профессии / мамы в декрете"
- Отдельные лендинги для каждого сегмента с релевантным фото и заголовком
- CR лендинга вырос **с 3% до 6%** после сегментации

### Бенчмарки конверсий (Рунет, инфобизнес)

| Этап | CR |
|------|----|
| Клик -> регистрация (подписная/бесплатник) | ~50% |
| Клик -> покупка (продающий лендинг) | 3-6% |
| Регистрация -> пришел на вебинар | 25% |
| Вебинар -> заявка | 7-20% |
| Заявка -> оплата (теплый) | 50-55% |
| Заявка -> оплата (холодный) | 10% |

### Рынок эзотерики
- Оборот: 21.6 -> 18.3 млрд руб. (-15%) в 2025
- Нужна дифференциация от массового рынка
- Ценность: авторская позиция + реальные кейсы (на фоне AI-контента)

### Дизайн в нише астрологии
- Темные фоны (черный/темно-синий) -- космос
- Градиенты фиолетовый-розовый -- мистицизм
- Золотые акценты -- премиальность
- Астрологическая символика
- Заголовки: судьба/предназначение + контроль/влияние
