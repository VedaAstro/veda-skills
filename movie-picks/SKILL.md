## SOLO — профиль вкуса

### Что работает
- Давление на героя (деньги / власть / семья / мораль)
- Системные конфликты — герой vs система, не герой vs злодей
- Последствия решений — причинно-следственная логика, не случайность
- Развитие сюжета — каждая серия двигает вперёд
- Сложные персонажи без однозначного "хорошего"
- Умный юмор в комедиях (ситуативный, не шутки)

### Что не работает
- Мрак ради мрака (Гоморра, ZeroZeroZero)
- Жестокость без смысла
- Отсутствие движения сюжета (Манчестер у моря)
- Подростковый стиль (Евротур)
- Слабый юмор / водевиль
- Драма без инсайтов (Потомки — 6.5/10)

## WITH_WIFE — профиль вкуса

### Что работает
- Легко смотреть, не требует полной концентрации
- Есть сюжет и идея (не фоновое видео)
- Умная комедия или лёгкая драма с теплом
- Эмоционально не тяжело
- Финал не оставляет в депрессии

### Что не работает
- Тяжёлая / давящая драма
- Затянутость / нудность
- Слабый юмор
- Подростковый вайб
- Избыточная жёсткость

## Алгоритм рекомендации

### Шаг 1: Определи режим (SOLO / WITH_WIFE)

### Шаг 2: Определи жанр
Если не указан — спроси: "Кино или сериал? Комедия, драма или без разницы?"

### Шаг 3: Примень фильтры

**SOLO:**
```
ТРЕБУЕТСЯ хотя бы 2 из:
  + давление/ставки на герое
  + системный конфликт
  + последствия решений
  + сложные персонажи

ЗАПРЕЩЕНО:
  - мрак ради мрака
  - нет движения сюжета
  - подростковый стиль
```

**WITH_WIFE:**
```
ТРЕБУЕТСЯ все 3:
  + смотрится легко (не давит)
  + есть сюжет/идея
  + эмоционально не тяжело

ЗАПРЕЩЕНО:
  - тяжёлая давящая драма
  - затянутость
  - слабый юмор
  - жёсткость без смысла
```

### Шаг 4: Проверь "уже смотрел"
Перед рекомендацией спроси или проверь по спискам в `reference/watched.md`.

### Шаг 5: Дай 3–5 вариантов с обоснованием
Формат:
```
**Название** (год, жанр)
Почему подойдёт: [1-2 предложения через призму профиля]
Похоже на: [из эталонов]
Где смотреть: Netflix / Кинопоиск / Apple TV / etc
```

### Шаг 6: Рейтинг уверенности
После каждой рекомендации — звёздочка попадания:
- ★★★ — очень похоже на эталон
- ★★ — должно зайти, но не 100%
- ★ — менее уверен, но стоит попробовать
