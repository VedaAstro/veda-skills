# Шаблоны бот-воронок

## 1. Welcome-цепочка (Onboarding)

**Задача:** довести до "aha moment" за < 60 секунд, собрать данные через micro-commitments.

```
/start
  → Приветствие (1 сообщение: кто бот, что даёт, НЕ что умеет)
  → "Как тебя зовут?" (свободный ввод)
  → "Когда родился(ась)?" (дата → inline calendar или текст)
  → "Во сколько?" (время → если не знает, предложи "примерно")
  → "В каком городе?" (текст → автокомплит)
  → [Пауза 2-3 сек: "Рассчитываю карту..."]
  → РЕЗУЛЬТАТ: картинка карты + персональный инсайт + 1 кнопка CTA
```

**Принципы:**
- Каждый шаг — одно поле. Не проси всё сразу в одной форме
- Прогресс: "Шаг 3 из 5" или эмодзи-индикатор ⬛⬛⬛⬜⬜
- Если юзер бросил на середине — через 1ч мягкий remind: "Осталось 2 шага до твоей карты"
- НЕТ меню на этапе онбординга. Только линейный поток
- Финал = ценность + единственный CTA (не "вот меню из 8 пунктов")

**VEDA-специфика:**
- "Aha moment" = натальная карта + атмакарака ("планета твоей души") + 1 инсайт
- Подчеркнуть уникальность: "Расчёт по Джйотиш с точностью до минуты" (не гороскоп)
- Солнце/Луна/Лагна одной строкой — быстрый персональный якорь

## 2. Nurture Drip-цепочка (Retention)

**Задача:** сформировать привычку возвращаться, постепенно углублять вовлечение.

```
D0 (сразу после онбординга):
  → "Твой прогноз на сегодня" (первый контент, подтверждение ценности)

D1:
  → "Луна сегодня в [знак] — что это значит для тебя" (астро-контент)
  → Кнопка: "Хочу подробнее" (углубление)

D3:
  → "Знаешь, что твоя атмакарака [планета] говорит о твоём призвании?"
  → Кнопка: "Расскажи" (вовлечение через любопытство)

D5:
  → Транзитный инсайт: "Через 3 дня Юпитер входит в [дом] — удачное время для [тема]"
  → Кнопка: "Напомни в этот день"

D7:
  → Social proof: "Мария, [знак] как у тебя, прошла диагностику и вот что узнала..."
  → Кнопка: "Узнать про диагностику" (первое касание с платным)

D14:
  → Оффер: "Персональная диагностическая сессия — разберём твою карту 1 на 1"
  → Scarcity: "На этой неделе осталось 3 слота"
```

**Принципы:**
- Каждое сообщение = ценность + 1 CTA. Никогда просто "привет, как дела?"
- Формат варьируется: текст → картинка → голосовое → текст
- Частота: D0-D3 ежедневно, D3-D14 через день, после D14 — 2-3 раза в неделю
- Персонализация обязательна: имя, знак, планеты из карты
- НЕ отправляй если юзер уже взаимодействовал сегодня (не дублируй)

## 3. Re-engagement (Win-back)

**Задача:** вернуть замолчавшего юзера через ценность, не через guilt-trip.

```
Молчит 3 дня:
  → Ценность: "У тебя интересный транзит на этой неделе. Показать?"
  → 1 кнопка: "Покажи"

Молчит 7 дней:
  → Любопытство: "Помнишь свою атмакараку [планета]? Есть кое-что новое о ней"
  → 1 кнопка: "Что нового?"

Молчит 14 дней:
  → Лёгкий FOMO: "Пока тебя не было, я рассчитал 3 транзита для тебя. Хочешь посмотреть?"
  → 1 кнопка: "Посмотреть"

Молчит 30 дней:
  → Последнее касание: "Я всё ещё храню твою карту. Если захочешь вернуться — нажми кнопку"
  → 1 кнопка: "Я здесь"
  → После этого — НЕ писать. Юзер ушёл. Respect.
```

**Принципы:**
- НИКОГДА: "Куда пропал?", "Ты забыл про меня?", guilt-tripping
- ВСЕГДА: персональная ценность как повод вернуться
- Максимум 4 попытки за 30 дней
- Если юзер ответил — возвращай в nurture drip с текущего дня, не с начала
- Не считай re-engagement "спамом" — это ценный контент, который юзер пропустил

## 4. Referral Loop

**Задача:** превратить довольных юзеров в источник новых юзеров.

```
Триггер: юзер получил 5+ прогнозов И нажимал "Хочу подробнее" хотя бы 2 раза
  → "Знаешь кого-то, кому интересна Джйотиш? Отправь им свою реферальную ссылку"
  → "За каждого друга с картой — бонусный расширенный прогноз"

После реферала:
  → Уведомление: "[Имя] присоединился по твоей ссылке! Твой бонус: расширенный прогноз [тема]"
  → Кнопка: "Получить прогноз"
```

**Принципы:**
- Проси реферал ТОЛЬКО у вовлечённых юзеров (не у всех подряд)
- Награда = контент (расширенный прогноз), не деньги
- Награждай ОБЕ стороны: реферер получает бонус, приглашённый получает "персональный бонус от друга"
- Deep link: `/start?ref=USER_ID` → бот знает кто привёл

## 5. Value Ladder (бесплатное → платное)

**Задача:** плавная эскалация от бесплатного контента к платному продукту.

```
Уровень 0 — БЕСПЛАТНО (бот):
  Ежедневный прогноз, натальная карта, базовые инсайты
  Цель: привычка + доверие

Уровень 1 — БЕСПЛАТНО+ (бот, за активность/реферал):
  Расширенный разбор планеты, совместимость, даша-прогноз
  Триггер: 7+ дней активности ИЛИ 1 реферал
  Цель: показать глубину, создать желание "хочу больше"

Уровень 2 — ПЛАТНО (диагностика):
  Персональная сессия 1-на-1 с диагностом (60 мин)
  Триггер: юзер нажал "Узнать про диагностику" + D7+ в боте
  Оффер: "Разберём твою карту лично. 3 слота на этой неделе"
  Цель: личный контакт + продажа курса

Уровень 3 — ПЛАТНО (курс 7Н/5S):
  Полная программа обучения
  Триггер: после диагностики
  Цель: максимальный LTV
```

**Принципы:**
- Каждый уровень даёт БОЛЬШЕ ценности, чем предыдущий (не просто "плати за то же самое")
- Переход между уровнями = естественное желание, не навязывание
- Scarcity работает только на уровнях 2-3 (на бесплатном — не ограничивай)
- Оффер появляется когда юзер ГОТОВ (по поведению), не по таймеру
