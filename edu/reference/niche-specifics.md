# Нишевая специфика: коучинг, астро-коучинг, психология

## Трансформационные программы (общие принципы)

**Архитектура трансформации:**
```
ОСОЗНАНИЕ → ПРИНЯТИЕ → РЕШЕНИЕ → ДЕЙСТВИЕ → ИНТЕГРАЦИЯ → ЗАКРЕПЛЕНИЕ
```

- Трансформация требует времени — не обещай «изменение за 7 дней» в глубинных темах
- Психологическая безопасность = абсолютный приоритет
- Сопровождение важнее контента (куратор, группа, поддержка)
- Личные истории ведущего = мощнейший инструмент доверия

## Онлайн-курсы по коучингу

**Специфические принципы:**
- Учи через моделирование: показывай сессии, а не только объясняй теорию
- Практика с реальными клиентами — обязательный компонент
- Супервизия — критически важна для навыковых курсов
- Этические аспекты — отдельный модуль (не опциональный)
- Самокоучинг — основа для понимания процесса изнутри

**Структура коуч-программы:**
```
Блок 1: Основы + философия
Блок 2: Базовые инструменты (практика на себе)
Блок 3: Продвинутые техники (практика с партнёром)
Блок 4: Работа с реальными клиентами (супервизия)
Блок 5: Бизнес-аспекты коучинга
Блок 6: Интеграция и сертификация
```

## Астро-коучинг и ведическая астрология

**Специфика аудитории:**
- Часто нет базовых знаний — начинай с основ натальной карты
- Скептики и верующие в одной группе → удерживай нейтральный академический тон
- Практика (консультации) важнее теории — расчёты без применения не работают

**Архитектура программы:**
```
Фундамент: планеты + знаки + дома + аспекты (отдельно каждый элемент)
Интерпретация: синтез элементов + ключевые позиции (обязательны кейсы)
Транзиты: текущие влияния + прогнозирование
Специализация: работа с клиентами, этика прогнозов
```

**Критические моменты дизайна:**
- Один элемент полностью → потом следующий (не сваливай в одну лекцию)
- Интерпретация = искусство, не алгоритм → учи мышлению, не шаблонам
- Кейсы реальных карт — обязательны в каждом блоке
- Этика прогнозов — отдельный акцент: не «ты потеряешь работу», а «период трансформации карьеры»

**Языковые правила для астро-контента:**
- Избегай категоричности («ты будешь...» → «тенденция к...», «период, когда...»)
- Давай выбор в интерпретации («возможные проявления: А или Б»)
- Связывай астрологию с психологическими концептами (доступнее для скептиков)

## Психологические программы

**Специфические требования:**
- Чёткие границы: «это образовательная программа, не терапия»
- Предупреждения при работе с травмой
- Доступ к специалисту при необходимости
- Темп студента важнее темпа программы
- Группа = мощный терапевтический фактор — используй осознанно

**Форматы для психологических тем:**
- Психоэдукация (информация о процессах психики)
- Практики осознанности
- Работа с телом (соматика)
- Дневники и рефлексия
- Групповые шеринги (с правилами конфиденциальности)

**Чего избегать:**
- Не диагностируй (даже если очевидно)
- Не давай советы там, где нужна терапия
- Не форсируй темп — особенно при работе с травмой

## Марафоны (мягкие ниши, коучинг, астрология)

**7-дневный марафон — оптимальная структура:**
```
День 1: Вход + контекст + первая практика (лёгкая победа)
День 2–3: Основной контент + практики
День 4: Углубление + сложная практика (кризис марафона)
День 5–6: Интеграция + применение
День 7: Завершение + путь вперёд + оффер
```

**21-дневный марафон:**
- Неделя 1: Основа и привычка (формирование нейронных паттернов)
- Неделя 2: Углубление и преодоление (критическая точка — поддержка максимальная)
- Неделя 3: Интеграция и закрепление (результаты видны, мотивация растёт)

**Ключевые правила марафона:**
- Ежедневный контакт обязателен (даже короткий — 5 минут)
- Геймификация: прогресс-бар, бейджи, публичные коммитменты
- Группа / чат = основа вовлечения
- День 4–5 = точка отсева — усиль поддержку

## Вебинарные курсы (живые встречи + материалы)

**Структура живого вебинара:**
```
00:00 — Открытие (5 мин): связь, «зачем вы сегодня здесь»
00:05 — Контекст (10 мин): почему важно именно сейчас
00:15 — Блок 1: [тема] + практика/вопрос (20 мин)
00:35 — Блок 2: [тема] + практика/вопрос (20 мин)
00:55 — Q&A / обсуждение (15 мин)
01:10 — Блок 3: [тема] + практика (15 мин)
01:25 — Интеграция и следующие шаги (10 мин)
01:35 — Закрытие / оффер (10 мин)
```

**Паузы и вовлечение в прямом эфире:**
- Вопрос в чат каждые 8–10 минут
- Опрос / голосование 1–2 раза за сессию
- Практика «делай вместе» (не только «смотри как я»)
- Персональные обращения к участникам по имени
