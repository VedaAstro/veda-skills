# Андрагогика — обучение взрослых

## Шесть принципов Knowles (Andragogy Model)

| Принцип | Суть | Практическое применение |
|---------|------|------------------------|
| Need to Know | Зачем мне это нужно ДО начала | Начинай каждый модуль с «что вы получите и зачем это важно» |
| Self-Concept | Я — самоуправляемая личность, не ученик | Давай выбор темпа, порядка, глубины. Без «должны/обязательно» |
| Prior Experience | Мой опыт — богатейший ресурс | Рефлексивные вопросы: «Вспомните случай...», «Как это перекликается с вашим опытом?» |
| Readiness to Learn | Готов учиться тому, что решает реальную проблему | Материал = решение актуальной боли, а не академическая теория |
| Orientation to Learning | Ориентируюсь на жизнь, а не на предмет | Названия модулей — результат, не тема. «Как работать с ретро-Меркурием» > «Ретроградный Меркурий» |
| Motivation | Внутренние мотиваторы сильнее внешних | Апеллируй к самооценке, качеству жизни, удовлетворённости — не только к сертификату |

## Пять допущений Линдемана

1. Взрослые мотивированы учиться, когда потребности удовлетворяются через обучение
2. Ориентация — на жизнь, а не на предмет → практические кейсы > академические примеры
3. Опыт — богатейший ресурс → активируй через рефлексию и обсуждения
4. Глубокая потребность в самоуправлении → давай автономию
5. Индивидуальные различия растут с возрастом → несколько путей через материал

## Принципы Роджерса (студентоцентрированное обучение)

1. Нельзя напрямую научить — можно только создать условия
2. Человек учится значимому, только когда сам участвует в принятии решений
3. Обучение, угрожающее самооценке, воспринимается с трудом → психологическая безопасность = приоритет
4. Обучение происходит, когда угроза самооценке минимальна
5. Значительная часть обучения — через действие

**Роль куратора/тренера = фасилитатор, не лектор.**

## Maslow в контексте обучения

```
САМОАКТУАЛИЗАЦИЯ → трансформационные программы
УВАЖЕНИЕ → сертификаты, признание, прогресс
ПРИНАДЛЕЖНОСТЬ → группы, комьюнити, мастермайнды
БЕЗОПАСНОСТЬ → структура, чёткий план, поддержка
ФИЗИОЛОГИЧЕСКИЕ → удобство доступа, технические требования
```

**Правило**: убедись, что нижние уровни закрыты перед работой с высшими.
Пример: если студент технически не может войти на платформу (физиологический уровень), он не будет трансформироваться.

## Теория самодетерминации (Deci & Ryan)

Три базовые потребности для внутренней мотивации:
- **Автономия**: «я сам выбираю»
- **Компетентность**: «я справляюсь»
- **Связность**: «я часть чего-то большего»

Применение: давай выбор (автономия), ранние победы (компетентность), группу/комьюнити (связность).

## Эмоции и обучение

- Эмоционально насыщенные переживания запоминаются лучше
- Позитивные эмоции расширяют когнитивные ресурсы (открытость к обучению)
- Страх и тревога сужают фокус и блокируют усвоение
- **Для коучинга/психологии**: психологическая безопасность = обязательное условие трансформации
