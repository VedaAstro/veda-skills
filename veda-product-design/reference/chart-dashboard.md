### Проблемы

1. **Grid `[1fr, 320px]`** — чарт центрирован в слишком широкой колонке, 280px пустоты
2. **Key Info Panel не заполняет высоту** — 250px контента vs 440px чарта, 190px дыра внизу
3. **PlanetCards дублирует Key Info** — те же планеты, знаки, дома в другом формате
4. **Yogas оторваны от чарта** — интерпретация карты в подвале страницы, новичок не доскроллит
5. **Varna на overview** — низкая ценность для обзорного экрана, лучше на отдельном табе

### Дублирование данных

| Данные | Где появляется 1-й раз | Где дублируется |
|--------|----------------------|-----------------|
| Солнце: знак, дом | Key Info Panel | PlanetCards |
| Луна: знак, дом | Key Info Panel | PlanetCards |
| Раху/Кету: знак, дом | Key Info Panel | PlanetCards |
| Лагна | Key Info Panel | (уникально) |
| Текущая даша | Key Info Panel | (уникально) |
| Марс-Сатурн: позиции | PlanetCards | (уникально — нет в Key Info) |

Key Info показывает 5 планет. PlanetCards показывает 9. Пересечение — 4 из 5. Дублирование ~80% Key Info.

## Решение

### Grid: `[440px, 1fr]`

Чарт получает ровно свой размер. Правая колонка = ~560px, заполняет остаток.

### Правая колонка: Key Info + Даша + Йоги (стек)

Три блока вертикально: ~250 + ~100 + ~70 = ~420px ≈ высота чарта (440px).

### PlanetCards → compact planet row

Горизонтальная таблица-лента: 9 колонок × 3 строки (имя, знак, дом). Full width. Убирает дублирование с Key Info.

### Varna → убрать с overview или оставить last

Наименее ценный блок. Либо только на табе "Варна", либо last element на overview.

## Simple mode (отдельная задача)

Simple mode должен быть переосмыслен как ДРУГОЙ экран:
- Чарт + 3 ключевых факта (лагна, солнце, луна) с пояснением
- Текущий период одной фразой
- Йоги как "особенности вашей карты"
- Без градусов, накшатр, домов, варны
- Будущее: AI-описания ("Ваш асцендент в Тельце — это значит...")
