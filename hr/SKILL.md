# HR — Найм

## Когда использовать

- Найм, подбор, интервью, оценка кандидатов
- Лист целей / scorecard для новой роли
- Тестовое задание или work sample
- Компенсации и оффер
- Вакансии на hh.ru / Telegram
- Onboarding, 30/60/90
- Разбор, почему найм сломался

## Два трека

| | EXPRESS (МП, таргетолог) | FULL (РОП, Head) |
|---|---|---|
| Лист целей | Да | Да |
| Скрининг | 15 мин, 4 вопроса | 20-30 мин, 4 вопроса |
| Глубокое интервью | Нет | 60-90 мин |
| Тестовое | Обязательно | Обязательно |
| Reference checks | 2 звонка | 3-4 звонка |
| Срок | 5-7 дней | 10-14 дней |

## Быстрый маршрут

1. Составь **Лист целей** с проверяемыми компетенциями → `hiring-playbook.md §2`
2. Опубликуй на **hh.ru + 5 Telegram-каналов** → `vacancy-templates.md` + `hiring-playbook.md §3`
3. **Скрининг** 15-20 мин по 4 вопросам → `§4`
4. **Тестовое** с rubric → `work-samples.md`
5. Для FULL трека: **глубокое интервью** → `§5`
6. **Reference checks** → `§7`
7. **Close** через задачу + деньги + свободу → `§8`
8. **Компенсация** → `compensation-models.md`
9. **Onboarding** → `§10`

## Ключевые правила

- Нет Листа целей → нет найма
- Подход «красного света»: ищи причину отказать, не причину взять
- Тестовое обязательно для ВСЕХ ролей, до собеседования
- Не озвучивай ЗП по телефону
- Правило «2 на 1»: 2 человека на ИС на 1 вакансию
- Презентация вакансии ПОСЛЕ выявления мотива кандидата
