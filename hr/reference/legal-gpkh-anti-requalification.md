# Анти-переквалификация ГПХ → трудовой (юридический фильтр для текстов вакансий)

## Зачем это
ФНС и суды периодически просматривают объявления о вакансиях и переквалифицируют отношения с самозанятыми/ИП в трудовые, доначисляя НДФЛ + страховые взносы + штрафы (до 40% от доначислений). Юристы Vedantica подтверждают практику. Этот фильтр обязателен для любой публикуемой вакансии под формат «договор с самозанятым/ИП».

## 11 признаков, по которым переквалифицируют (стоп-маркеры в тексте)

### 🔴 Красные (запрещены без исключения)

1. **«Испытательный срок»** — атрибут ТД (ст. 70 ТК), в ГПХ не существует. Заменять на «первые N месяцев — стоимость работ Y» или «адаптационный период оплачивается по фиксу».
2. **«Работодатель / работник / трудоустройство / приём на работу»** — лексика ТК. Используй «заказчик / исполнитель / сотрудничество / договор оказания услуг».
3. **«Полная занятость, на связи с 9:00 МСК»** — режим рабочего времени (ст. 91 ТК). Заменять на «координационные созвоны 2-3 раза в неделю в окне X-Y» или «гибкий график при условии участия в регулярных созвонах команды».
4. **«Карьерный рост через N месяцев в [должность] со ставкой Y»** — продвижение по службе = долгосрочные ТД-отношения. Заменять на «возможность расширения объёма и стоимости работ».
5. **«Оплачиваемый отпуск / больничный / соцпакет / ДМС / ученический отпуск»** — соц.гарантии ТК.

### 🟡 Жёлтые (смягчать формулировку)

6. **«Что нам важно»** + список личных качеств (проактивность, аккуратность, коммуникабельность) — это подбор по личности, как для штатной должности. Заменять на **«Компетенции для выполнения работ»** + конкретные навыки.
7. **«Бюджет на обучение / оплачиваем курсы»** — обучение работника за счёт работодателя признак ТД. Смягчать: «компенсируем профильные инструменты и курсы, необходимые для выполнения работ».
8. **«Что будешь делать»** + длинный список постоянных рутинных задач — системная деятельность, не разовые услуги. Заменять заголовок на **«Объём работ»**, формулировки в виде задач и предметов сопровождения.
9. **«Подчиняешься руководителю / отчитываешься перед...»** — иерархия. Заменять на «согласовываешь с заказчиком», «отправляешь отчёт по результатам».
10. **«Должность» = название штатной позиции** в названии вакансии (например «Помощник руководителя») — атрибут штата. Лучше формулировать через функцию: «Услуги по координации проектов», «Менеджер проектов» — допустимо но в связке с правильным форматом сотрудничества.
11. **«Контроль рабочего процесса / трекинг времени»** — управление трудовым процессом = ТД. Указывать что заказчик оценивает результат, а не процесс.

## Обязательный блок в каждой вакансии под ГПХ

В конце текста (перед "Следующий шаг") добавить:

```
<h3>Формат сотрудничества</h3>
<p>Договор оказания услуг с самозанятым или ИП. Заказчик ставит цели и
оценивает результат по согласованным KPI и срокам, не контролируя рабочий процесс.
Исполнитель самостоятельно распределяет своё время и выбирает инструменты.</p>
```

Этот блок — основная защитная плашка. Его читают и кандидаты, и налоговая.

## Чек-лист перед публикацией любой вакансии

Перед коммитом текста в `vacancy_content.py` или нажатием «Опубликовать» в HH:

- [ ] Нет слова «испытательный срок» (заменено на «первые N месяцев»)
- [ ] Нет жёсткого графика «с XX:XX до YY:YY» (заменено на «созвоны в окне» или «гибкий график»)
- [ ] Нет обещаний карьерного роста с привязкой к месяцам и должностям
- [ ] Нет «оплачиваемого отпуска / больничного / ДМС / соцпакета»
- [ ] Раздел «Что нам важно» переформулирован в «Компетенции для выполнения работ»
- [ ] Раздел «Что будешь делать» переименован в «Объём работ» (или аналог)
- [ ] Бюджет на обучение → «компенсируем профильные инструменты и курсы»
- [ ] **Есть финальный блок «Формат сотрудничества»** с явным указанием ГПХ
- [ ] Лексика: «исполнитель», «заказчик», «оказание услуг», «KPI», «согласовать»
- [ ] Нет: «работник», «работодатель», «приём», «трудоустройство»

## Что делать со штатным форматом

Если по факту человек должен быть штатным (full-time, фикс, единственный работодатель, контроль времени) — оформляй ТД, не пытайся скрыть это под ГПХ. Маскировка опаснее, чем честный ТД.

Гибридная модель «фикс + KPI» допустима через ГПХ только при условии что:
- Оплата по факту выполненных работ (акт ежемесячно)
- Исполнитель не интегрирован в штатное расписание
- Есть свобода выбора инструментов и времени
- Может работать с другими заказчиками параллельно
