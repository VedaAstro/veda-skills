## 2.7 Чара-каракатвы (Джаймини)

| Ранг | Карака | Санскрит | Код | Значение |
|------|--------|----------|-----|----------|
| 1 (max°) | Атмакарака | Ātmakāraka | AK | Душа, главная тема жизни |
| 2 | Аматьякарака | Amātyakāraka | AmK | Карьера, советник |
| 3 | Бхратрикарака | Bhrātṛkāraka | BK | Братья/сёстры |
| 4 | Матрикарака | Mātṛkāraka | MK | Мать |
| 5 | Путракарака | Putrakāraka | PK | Дети |
| 6 | Гнатикарака | Gnātikāraka | GK | Враги, болезни |
| 7 (min°) | Даракарака | Dārakāraka | DK | Супруг |

```
АЛГОРИТМ (7-карака):
1. Взять долготу {SU,MO,MA,ME,JU,VE,SA} в пределах знака (0°–30°)
2. Исключить RA/KE
3. Сортировать по убыванию → присвоить ранги AK..DK

8-КАРАКА (с Раху):
  Долгота Раху = 30° - rahu_degree
  8-я карака = Питрикарака (PiK) — отец
  Порядок: AK → AmK → BK → MK → PK → GK → PiK → DK
```

## 2.9 Ретроградность (Вакри)

| Категория | Планеты |
|-----------|---------|
| Никогда не ретроградны | SU, MO |
| Всегда ретроградны | RA, KE |
| Могут быть ретроградны | MA, ME, JU, VE, SA |

| Планета | Частота | Длительность |
|---------|---------|-------------|
| ME | ~3 р/год | ~24 дн |
| VE | ~1 р/18 мес | ~42 дн |
| MA | ~1 р/2 года | ~72 дн |
| JU | ~1 р/год | ~121 дн |
| SA | ~1 р/год | ~138 дн |

```
IF planet IN {SU, MO} AND is_retro → ОШИБКА ДАННЫХ
IF planet IN {RA, KE} → is_retro = True ВСЕГДА
IF is_retro → cheshta_bala = 60 virupas (max)
IF is_retro AND in_exaltation → флаг [VAKRI_UCCHA], НЕ автоинверсия
IF is_retro AND in_debilitation → флаг [VAKRI_NEECHA], НЕ автоинверсия
```

## 2.10 Верификация

```python
def is_retrograde_valid(planet_id: str) -> bool:
    NEVER_RETRO = {"SU", "MO"}
    ALWAYS_RETRO = {"RA", "KE"}
    CAN_BE_RETRO = {"MA", "ME", "JU", "VE", "SA"}
    if planet_id in NEVER_RETRO: return False
    if planet_id in ALWAYS_RETRO: return True
    return planet_id in CAN_BE_RETRO
```
