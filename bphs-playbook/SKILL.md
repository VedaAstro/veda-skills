# БПХШ Плейбук — Верификация астрорасчётов

## Когда использовать

- Проверка корректности данных из AstroGuru API
- Валидация таблиц трактовок в PostgreSQL (planet_in_house, planet_in_sign, functional_nature и др.)
- Верификация расчётов: навамша, варги, шадбала, аштакаварга
- Проверка йог (Панча Махапуруша, Раджа, Гаджакешари и др.)
- Определение функциональной природы планет для конкретной лагны
- Расчёт и проверка периодов Вимшоттари даша
- Любые вопросы по правилам Джйотиш из первоисточника (БПХШ)

## Связь с другими скиллами

| Скилл | Роль | Этот плейбук |
|-------|------|-------------|
| `/astro` (vedic-astro-knowledge) | Инженерный: API chain, Redis, resolvers | Астрономический: правила, таблицы, формулы |
| `/db` (veda-database-schema) | Схема БД, 48 таблиц | Эталонные данные для таблиц трактовок |
| `/diagnostic` | Диагностическая сессия | Знания для интерпретации карты |
