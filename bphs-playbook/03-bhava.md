## 4. Правила определения функциональной природы

### 4.1 Фундаментальные правила

| # | Правило | Источник |
|---|---------|----------|
| R1 | Правители Тикон (1, 5, 9) — всегда функциональные бенефики | 34.2 |
| R2 | Правители Дустхан (3, 6, 8, 11) — всегда функциональные малефики | 34.3 |
| R3 | Малефик + Кендра → функциональный бенефик (Кендра-адхипати-йога) | 34.4 |
| R4 | Бенефик + Кендра → теряет благость (Кендра-адхипати-доша) | 34.5 |
| R5 | Кендра + Тикона → Йогакарака | 34.6 |
| R6 | Правители 2/12 → результат по второму дому | 34.7 |
| R7 | Двойное владение → побеждает сильнейший: Тикона > Кендра > Упачая > Дустхана | 34.8 |
| R8 | Правитель 8-го злой, кроме случая = Лагнеша | 34.9 |
| R9 | Лагнеша — всегда благотворен | 34.10 |

### 4.2 Дополнительные правила

| # | Правило | Источник |
|---|---------|----------|
| R10 | Связь правителей Кендры и Триконы → Раджа-йога | 34.11 |
| R11 | То же + правитель Дустханы → Раджа-йога ослабляется | 34.12 |
| R12 | 11-й правитель — функциональный малефик | 34.13 |
| R13 | Бенефик в Кендре без связи с Триконой → нейтрален | 34.14 |

### 4.3 Алгоритм определения

```
FUNCTION get_functional_nature(graha, lagna):
  H1, H2 = houses_owned(graha, lagna)
  IF H1 == 1: RETURN "бенефик"                    # R9
  score = 0
  FOR Hi IN [H1, H2]:
    IF Hi IN {5, 9}: score += 3                    # Тикона
    IF Hi IN {4, 7, 10}:
      IF graha.is_natural_malefic: score += 1      # R3
      ELSE: score -= 1                             # R4
    IF Hi IN {6, 8, 12}: score -= 2                # Дустхана
    IF Hi == 3: score -= 1
    IF Hi == 11: score -= 2
    IF Hi == 2: score += 0                         # нейтрален
  IF owns_kendra(graha) AND owns_trikona(graha):
    RETURN "ЙОГАКАРАКА"                            # R5
  IF score > 0: RETURN "бенефик"
  IF score < 0: RETURN "малефик"
  RETURN "нейтрален"
```

FUNCTION get_houses(planet, lagna_rashi):
  owned_rashis = RASHI_LORDS[planet]  // → 01-rashi
  houses = []
  FOR rashi IN owned_rashis:

  RETURN houses

## 10. Аруда-пады (Bhāva Āruḍha Pada)

> Источник: БПХШ, глава 29

### 10.1 Расчёт

```
FUNCTION arudha_pada(house_num, lord_rashi, lagna):
  house_cusp = (lagna + house_num - 1) % 12
  distance = (lord_rashi - house_cusp) % 12
  arudha = (lord_rashi + distance) % 12
  IF arudha == house_cusp:
    arudha = (house_cusp + 10 - 1) % 12
  IF arudha == (house_cusp + 6) % 12:
    arudha = (house_cusp + 4 - 1) % 12
  RETURN arudha
```

    assert any(h in {1,4,7,10} for h in houses) or any(h in {5,9} for h in houses)

def verify_dikbala(graha, house):
    """Проверка дик-балы по §8."""
    DIKBALA_MAX = {'Surya':10, 'Chandra':4, 'Mangala':10, 'Budha':1, 'Guru':1, 'Shukra':4, 'Shani':7}
    if house == DIKBALA_MAX[graha]:
        return "MAX"
    if house == (DIKBALA_MAX[graha] + 6 - 1) % 12 + 1:
        return "MIN"
    return "PARTIAL"
