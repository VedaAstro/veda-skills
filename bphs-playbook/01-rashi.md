## 1.2 Формулы классификаций

```
element(n)   = [Агни, Притхви, Ваю, Джала][(n-1) % 4]
modality(n)  = [Чара, Стхира, Двисвабхава][(n-1) % 3]
gender(n)    = IF n % 2 == 1 → М ELSE → Ж
direction(n) = SWITCH element(n):
                 Агни    → Пурва (Восток)
                 Притхви → Дакшина (Юг)
                 Ваю     → Пашчима (Запад)
                 Джала   → Уттара (Север)
```

```
ИЕРАРХИЯ ДОСТОИНСТВА (по убыванию):
  1. Парама-Уччха   — точный градус экзальтации
  2. Уччха          — знак экзальтации
  3. Мулатрикона    — в диапазоне мулатрикона
  4. Свакшетра      — собственный знак, вне мулатрикона
  5. Митра-кшетра   — знак друга
  6. Сама-кшетра    — знак нейтрала
  7. Шатру-кшетра   — знак врага
  8. Нича            — знак дебилитации
  9. Парама-Нича     — точный градус дебилитации

ОСОБЫЕ СЛУЧАИ:
  IF Будха в Деве:
    0°–15°     → Уччха (пик 15°)
    16°–20°    → Мулатрикона
    20°01'–30° → Свакшетра
  IF Чандра в Тельце:
    0°–3°      → Уччха (пик 3°)

  Раху/Кету: нет мулатрикона (нет собственных знаков по БПХШ)
```

## 1.9 Верификация

```python
def get_dignity(planet, sign, degree):
    if sign == planet.exalt_sign:
        return "PARAMA_UCHCHA" if degree == planet.exalt_degree else "UCHCHA"
    if sign == planet.debil_sign:
        return "PARAMA_NEECHA" if degree == planet.debil_degree else "NEECHA"
    if sign == planet.moola_sign and planet.moola_start <= degree <= planet.moola_end:
        return "MOOLATRIKONA"
    if sign in planet.own_signs:
        return "SWAKSHETRA"
    return get_relationship(planet, get_lord(sign))  # MITRA/SAMA/SHATRU/ADHI_*
```

```

ASSERT: Sun + Moon = по 1 знаку, остальные 5 планет = по 2 знака → 1+1+5*2 = 12
```
