# 20 признаков любительского UI

> Каждый из этих паттернов — моментальный downgrade от premium к «сделано на коленке».
> Если нашёл хотя бы один — стоп, исправить ДО деплоя.

---

## TYPOGRAPHY

### 1. Все заголовки одного размера
**Проблема:** нет reading order, глаз не знает куда смотреть
**Фикс:** минимум 1.5× разница между уровнями (24px body → 36px h2 → 54px h1)

### 2. Line-height 1.0 или 2.0
**Проблема:** 1.0 = строки слипаются. 2.0 = текст «плавает», нет cohesion
**Фикс:** headlines 1.1-1.2, body 1.5-1.6. Всегда

### 3. Строка длиннее 80 символов
**Проблема:** глаз теряет строку при возврате к началу
**Фикс:** `max-w-prose` (65ch) или `max-w-xl` для body text

### 4. Центрированный body text
**Проблема:** нечитаемо дольше 2 строк
**Фикс:** center только для headlines и single-line captions. Body = left-aligned

---

## SPACING

### 5. Одинаковый spacing везде
**Проблема:** нет visual grouping (Gestalt proximity). Всё сливается в кашу
**Фикс:** правило 1:3 — между группами ≥ 3× внутри группы

### 6. Spacing не из шкалы (17px, 23px, 37px)
**Проблема:** inconsistency, «на глаз». Каждый экран чуть по-разному
**Фикс:** только из шкалы: 4/8/12/16/24/32/48/64/96

### 7. Нет breathing room вокруг hero
**Проблема:** hero headline упирается в nav или в следующую секцию
**Фикс:** py-24 (96px) desktop, py-16 mobile. Apple делает 120px+

---

## COLOR

### 8. Больше 3 хроматических цветов
**Проблема:** цирк, не интерфейс. Глаз не понимает что важно
**Фикс:** 1 brand + 1 accent + neutral ramp. Всё. Больше = обоснуй

### 9. Серый текст на цветном фоне
**Проблема:** контраст < 3:1, нечитаемо, WCAG fail
**Фикс:** на цветном фоне только white или dark text. Проверить contrast checker

### 10. Чистый #000 на #FFF
**Проблема:** слишком жёсткий контраст, утомляет глаз
**Фикс:** off-black #1A1917 + off-white #F7F3EC (warm) или #111 + #FAFAFA (cool)

---

## IMAGES

### 11. Приплюснутые / растянутые изображения
**Проблема:** aspect ratio сломан, выглядит сломанно
**Фикс:** `object-fit: cover` + контейнер с фиксированным aspect-ratio. Всегда `width` + `height`

### 12. Огромные нескомпрессованные фото (5MB+)
**Проблема:** LCP > 3s, пользователь уходит
**Фикс:** AVIF/WebP, srcset с sizes, max 200KB для hero image

### 13. Stock-фото с «бизнес-людьми»
**Проблема:** мгновенный downgrade доверия. Все видели эти фото
**Фикс:** свои фото или вообще без фото (типографика + цвет)

---

## COMPONENTS

### 14. Карточки внутри карточек
**Проблема:** двойная elevation, путаница иерархии, визуальный шум
**Фикс:** flat list внутри карточки или divider. Один уровень elevation

### 15. Кастомные компоненты вместо библиотечных
**Проблема:** inconsistency, баги accessibility, time waste
**Фикс:** Untitled UI component → MCP search → только потом кастом

### 16. Emoji в UI-элементах
**Проблема:** выглядят по-разному на разных OS, не масштабируются, дёшево
**Фикс:** иконки из @untitledui/icons или 3D WebP из /public/

---

## MOTION

### 17. Bounce / elastic анимации
**Проблема:** игриво, несерьёзно, не для premium бренда
**Фикс:** ease-out для появления, ease-in для ухода. Без пружин

### 18. Все элементы появляются одновременно
**Проблема:** нет reading order в motion, каша
**Фикс:** stagger 60-100ms между элементами (eyebrow → headline → body → CTA)

---

## LAYOUT

### 19. Нет max-width на контейнере
**Проблема:** на ultra-wide экране текст растягивается на 2000px
**Фикс:** max-w-7xl (1280px) + mx-auto для основного контента. Hero может быть full-width

### 20. Кнопки разных размеров на одном экране
**Проблема:** нет system, каждая кнопка «по настроению»
**Фикс:** один size per context. Все primary buttons на странице = одинаковый size

---

## Быстрый self-check

Прежде чем показать результат — пройди этот список за 30 секунд:

```
□ Squint test: прищурился — hierarchy видна?
□ 3-color rule: не больше 3 хроматических цветов?
□ Spacing consistency: всё из шкалы, правило 1:3?
□ Typography: max 3 уровня, line-height correct?
□ No cards in cards?
□ No emoji?
□ No stretched images?
□ No bounce animations?
□ Body text left-aligned, max 65ch?
□ Touch targets ≥ 44px?
```

Если ответ «нет» на любой пункт — исправь. Не деплой.
