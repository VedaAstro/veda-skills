# E-E-A-T Playbook

## Когда открывать

- Нужно оценить экспертность и trust страницы
- Есть сомнение, что контент выглядит AI-thin или без автора
- Страница коммерческая, экспертная или чувствительная по доверию

## Модель оценки

| Блок | Что проверять |
|------|---------------|
| Experience | реальные кейсы, first-hand signals, конкретные примеры |
| Expertise | биография автора, специализация, корректность терминов |
| Authoritativeness | публикации, упоминания, профиль автора/бренда |
| Trustworthiness | контакты, прозрачность, цены, условия, политика, безопасность |

## Чеклист

```text
□ автор или бренд-носитель знания виден на странице
□ дата публикации/обновления не спрятана
□ есть конкретика, а не только общие слова
□ есть proof: кейсы, отзывы, скриншоты, методология, примеры
□ коммерческий следующий шаг прозрачен
□ нет следов batch-template текста без самостоятельной ценности
```

## Для VEDA

- На экспертных страницах усиливают trust:
  - профиль астролога/автора
  - методика разбора
  - примеры трактовок
  - реальные отзывы и кейсы
- На коммерческих страницах обязательны:
  - понятные услуги
  - стоимость или модель записи
  - прозрачные контакты/бренд

## Что не делать

- Не сводить E-E-A-T к длине текста
- Не называть страницу “экспертной” без автора, кейсов и методологии
