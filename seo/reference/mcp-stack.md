## Главные блокеры для "боевого" SEO-режима

1. Нет доступов к нужному домену в Google Search Console (`sites: 0`).
2. В Yandex Webmaster подтверждён не целевой домен (только `vedantica.tech`).
3. Метрика может быть подключена, но нужен счётчик именно на целевом домене, иначе аналитика не про тот продукт.

## Минимальный путь до полноценно работающего стека

1. Назначить целевой домен (один приоритетный).
2. Выдать этому же Google-аккаунту права Owner/Full в Search Console на этот домен.
3. Подтвердить этот домен в Yandex Webmaster под тем же токеном.
4. Проверить/создать Yandex Metrika counter для этого домена.
5. Перезапустить MCP-сессию и выполнить smoke-check:
   - `search-console.sites_list` -> домен виден;
   - `yandex-webmaster.list-hosts` -> домен виден;
   - `yandex-metrika.get-counters` -> счётчик доступен;
   - `yandex-wordstat.get-regions-tree` + `brave-search.web_search` -> отвечают.

## Инструменты мониторинга для programmatic SEO

### Обязательные (бесплатные)

| Инструмент | Зачем | Статус |
|-----------|-------|--------|
| Яндекс Webmaster MCP | Индексация, ошибки, подача sitemap, внешние ссылки | Настроен (yandex-webmaster) |
| Яндекс Метрика MCP | Поведенческие факторы, цели на CTA | Настроен (yandex-metrika) |
| IndexNow API | Мгновенное уведомление Яндекса о новых URL | Нужен скрипт `notify-indexnow.ts` |
| Яндекс Wordstat MCP | Проверка спроса по ключевикам для новых кластеров | Настроен (yandex-wordstat) |

### Рекомендуемые (платные)

| Инструмент | Зачем | Цена |
|-----------|-------|------|
| **Topvisor** | Трекинг позиций 400+ URL в Яндексе и Google | от 2 490 р/мес |
