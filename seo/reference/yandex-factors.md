# Факторы ранжирования Яндекса (из утечки кода 2023)

> Загружай on-demand: когда нужны детали по конкретным факторам

## Обзор

Утечка исходного кода Яндекса в январе 2023 подтвердила **1922 фактора ранжирования**.
Четыре категории запросов с разными весами факторов.

## Веса по типам запросов

### Навигационные запросы ("Вконтакте", "Сбербанк онлайн")
- 50% — авторитетность бренда (упоминания, прямые заходы)
- 50% — точное совпадение URL/домена с запросом

### Коммерческие запросы ("купить телефон", "консультация астролога")
- **65% — поведенческие факторы**
  - CTR в выдаче
  - Время на сайте (long click vs short click)
  - Bounce rate
  - Возвраты в выдачу (pogo-sticking)
  - Глубина просмотра
- 35% — контентные + коммерческие факторы
  - Цены, ассортимент, контакты
  - Соответствие запросу

### Информационные запросы ("Марс в 7 доме", "что такое натальная карта")
- 45% — глубина и полнота контента
  - Тематическая полнота (coverage)
  - Уникальность
  - Длина (но не спам)
  - Мультимедиа (изображения, видео, таблицы)
- 55% — авторитетность источника
  - ИКС
  - Ссылочный профиль (качество)
  - Возраст домена
  - Поведенческие метрики агрегированно

### Региональные/локальные запросы ("астролог Москва")
- **70% — географическая близость**
  - Яндекс.Бизнес карточка
  - Адрес на сайте
  - Региональная привязка в Вебмастере
  - Локальный телефонный код
- 30% — отзывы и рейтинг
  - Яндекс.Карты отзывы
  - Агрегированный рейтинг

## Ключевые подтверждённые факторы

### Поведенческие (из утечки)
| Фактор | Описание | Вес |
|--------|----------|-----|
| `QueryDwellTime` | Время от клика до возврата в выдачу | Высокий |
| `IsLongClick` | Клик без возврата >30 сек | Высокий |
| `IsLastClick` | Последний клик в сессии поиска | Очень высокий |
| `AvgTimeOnPage` | Среднее время на странице | Средний |
| `BounceRate` | Процент отказов | Средний (негативный) |
| `PageDepth` | Глубина просмотра в сессии | Средний |
| `DirectTraffic` | Прямые заходы на домен | Высокий (для бренда) |

### Текстовые (из утечки)
| Фактор | Описание | Вес |
|--------|----------|-----|
| `TextRelevance` | TF-IDF + BM25 релевантность | Базовый |
| `TitleMatch` | Совпадение ключа с title | Высокий |
| `HeaderMatch` | Совпадение с H1-H3 | Средний |
| `TextLength` | Длина контента | Средний (до оптимума, затем нет) |
| `UniqueWords` | Лексическое разнообразие | Средний |
| `SpamScore` | Переоптимизация (Баден-Баден) | Негативный |

### Ссылочные (из утечки)
| Фактор | Описание | Вес |
|--------|----------|-----|
| `LinkAge` | Возраст ссылки | Средний |
| `DonorQuality` | Качество ссылающегося сайта | Высокий |
| `AnchorRelevance` | Релевантность анкора | Средний |
| `LinkDiversity` | Разнообразие доменов-доноров | Средний |
| `LinkSpamScore` | Биржевые/спамные ссылки | Негативный (Минусинск) |

### Коммерческие (из утечки)
| Фактор | Описание | Вес |
|--------|----------|-----|
| `HasPrices` | Наличие цен | Высокий |
| `HasContacts` | Телефон, email, адрес | Высокий |
| `HasDelivery` | Условия доставки | Средний |
| `HasReviews` | Отзывы на сайте | Средний |
| `HasLegalInfo` | ИНН, ОГРН, юрлицо | Средний |
| `AssortmentSize` | Количество товаров/услуг | Средний |

## Алгоритмы-фильтры Яндекса

| Алгоритм | Наказывает за | Как избежать |
|----------|---------------|-------------|
| **Баден-Баден** | Переоптимизация текстов, keyword stuffing | Писать для людей, не роботов |
| **Минусинск** | Покупные ссылки с бирж | Не покупать ссылки. Органический linkbuilding |
| **АГС** | Некачественные сайты-сателлиты | Уникальный полезный контент |
| **Накрутка ПФ** | Искусственные поведенческие | Реальная ценность → реальное поведение |

## Яндекс vs Google: сводная таблица

| Фактор | Яндекс | Google |
|--------|--------|--------|
| Поведенческие | **Критично** (подтверждено утечкой) | Важно, но не доминирует |
| Ссылки | Средне (после Минусинска) | **Критично** (топ-3 фактор) |
| Контент | Морфология + полнота | E-E-A-T + полнота |
| Региональность | **Сильная** сегментация | Умеренная |
| Коммерция | **Отдельная категория** | Через E-E-A-T |
| Индексация | Медленнее | Быстрее, лучше с JS |
| Возраст домена | Больший вес | Меньший вес |
| Mobile | Важно | **Core ranking factor** |
| Core Web Vitals | Учитывает | **Ranking signal** |
| AI-контент | Не наказывает AI per se, наказывает спам | HCU — отдельный сигнал |
