# VEDA SEO Content Templates

## Когда открывать

- Нужно спроектировать SEO-страницу
- Нужно проверить, хватает ли контента и внутренних связей
- Нужно быстро выбрать шаблон под programmatic, UGC или профиль эксперта

## 1. Programmatic SEO страница

Пример:

```text
URL: /planeta/mars-v-7-dome
Источник: astro_knowledge / интерпретации + remedies + AI enrichment
```

Структура:
- `title`: ключ в начале, 50-60 символов
- `description`: 140-160 символов, польза + CTA
- `h1`: один, близок к title
- Блок 1: прямой ответ / общее значение
- Блок 2: влияние на ключевые сферы жизни
- Блок 3: примеры / частые сценарии / вопросы
- Блок 4: рекомендации и коррекции
- Блок 5: FAQ
- CTA на калькулятор или диагностику
- Связанные страницы: 5-10 внутренних ссылок

Минимум:
- 1500+ слов
- canonical self-reference
- `Article` + `FAQPage` + `BreadcrumbList`
- WebP/alt/width/height у изображений

Quality gates:
- не публиковать, если страница отличается от соседних только названием планеты/дома
- не публиковать батчами без внутренней перелинковки и sitemap coverage
- у каждой страницы должен быть собственный answer-first блок, а не только общий шаблон

## 2. Community / UGC тред

Пример:

```text
URL: /community/mars-v-7-dome-opyt
Schema: DiscussionForumPosting
```

Правила:
- SSR обязателен
- title и description собираются из заголовка и первых строк поста
- noindex для пустых тредов, спама, черновиков, тредов без ответов
- индексировать только треды, где уже есть плотность ответа и польза
- обязателен breadcrumb

Минимум для индексации:
- 800+ слов суммарно
- 3+ осмысленных ответа
- 1+ связанный материал или CTA

## 3. Профиль астролога

Пример:

```text
URL: /astrolog/ivanova-maria
Schema: Person + AggregateRating + Offer
```

Обязательные блоки:
- имя, специализация, опыт, город
- реальное фото
- отзывы и рейтинг
- услуги и цены
- статьи автора / публикации / кейсы
- понятный CTA на запись

Минимум:
- 500+ слов
- 3+ отзыва
- 1+ offer с ценой

## Тип запроса -> тип страницы

| Тип запроса | Пример | Что даём |
|-------------|--------|----------|
| Информационный | `марс в 7 доме значение` | programmatic article |
| Коммерческий | `консультация астролога онлайн` | marketplace/service page |
| Транзакционный | `натальная карта рассчитать` | calculator landing |
| Социальное доказательство | `отзывы астролог мария иванова` | author/profile page |
| UGC / опыт | `марс в 7 доме форум` | community thread |

## Publication Checklist

```text
□ title уникальный, ключ ближе к началу
□ description уникальный, с пользой и CTA
□ h1 один
□ контент не thin
□ 5-10 внутренних ссылок
□ есть canonical
□ есть schema
□ изображения в WebP, есть alt, width, height
□ URL добавлен в sitemap
□ страница не блокируется robots/noindex
□ есть понятный следующий шаг: CTA / related pages / calculator
□ страница проходит anti-abuse sanity check: это не thin template page и не scaled filler
```
