### Планеты в падении: трактовка и подход

<rules>
ПРАВИЛО ТРАКТОВКИ ПАДЕНИЙ:
- Планета в падении ≠ отсутствие результатов
- Планета в падении = энергия идёт через «узкое горлышко», не льётся свободным потоком
- В Кали-югу планеты в падении НЕ ДАЮТ УСНУТЬ, создают ресурс для развития
- Окончательный вывод ВСЕГДА делать по Шадбале
- Стхана Бала (положение в знаке) = 30-60% от общей силы

АЛГОРИТМ РАБОТЫ С ПАДЕНИЯМИ:
1. Определить внешние проявления (знак) — ограничения
2. Проверить Шадбалу — внутренние ресурсы
3. Проверить отмены падения (Нича-бханга)
4. Проверить Навамшу — улучшение/двойственный знак помогает укорениться
5. Дать коррекции через мантры и осознанность
</rules>

### Сильная планета + ретроградность

<logic>
ЕСЛИ планета сильная по знаку НО ретроградная:
  → Есть накопленная позитивная карма + недоделанные задачи из прошлого
  → Внешне потенциал есть, но проявляется через трудности
  → Нужны нетривиальные решения
  → Человек «пересдаёт экзамены» с более осознанным подходом

ЕСЛИ планета сильная по знаку НО слабая по Шадбале:
  → Внешние условия хорошие, внутренний настрой слабый
  → Внешний успех + внутренние сомнения
  → Нужна профилактика и подпитка по показателям планеты
</logic>

### Совмещение несовместимых влияний

<rules>
АЛГОРИТМ ТРАКТОВКИ ПРОТИВОРЕЧИЙ:
1. Понять природу каждой планеты отдельно
2. Учесть контекст ЗНАКА (кто сильнее в данном знаке?)
3. Учесть контекст ДОМА (в какой сфере проявляется)
4. Учесть дополнительные аспекты и соединения (Юпитер смягчает)
5. НЕ пытаться «исправить» напряжение — использовать как ресурс роста
6. Показать клиенту РАЗГОН КАРМЫ от минуса к плюсу

Типичные «конфликтные» пары:
- Солнце + Сатурн → эго vs ограничения → дисциплинированное лидерство
- Луна + Раху → эмоции vs иллюзии → нестандартное восприятие
- Меркурий + Кету → общение vs уход в себя → глубокая интуитивная коммуникация
</rules>

### Работа со стыдом и страхом проявления

<rules>
СТЫД vs СОВЕСТЬ:
- Стыд = эмоция, возникающая в обществе, связанная с оценкой других
- Совесть = глубокое внутреннее ощущение, не зависящее от людей

КАК ПРОВЕРИТЬ:
- «Я не делаю это, потому что боюсь осуждения?» → СТЫД
- «Я не делаю это, потому что чувствую внутренне, что это неправильно?» → СОВЕСТЬ

СТЫД ОСЛАБЛЯЕТ ЭГО → безвольное состояние → легко управлять человеком.

КАК ОСВОБОДИТЬСЯ:
1. Уважение к себе — честность и искренность
2. Слушать свои желания — осознанный выбор
3. Позволять себе ошибаться — каждый шаг = опыт
4. Не осуждать других — меньше оценок = легче себе
</rules>
