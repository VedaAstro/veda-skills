# Voice & Quality — тексты бота VEDA

> Два режима: **шаблоны** (пишем мы) и **диалог** (пишет Sonnet).
> Для шаблонов — формулы и примеры. Для диалога — контекстный слой.

---

## ЧАСТЬ 1: ШАБЛОНЫ (хардкод, пишем мы)

Шаблоны = welcome, caption к карте, CTA, daily intro, re-engagement.
Здесь копирайт-формулы уместны — текст статичный, LLM не участвует.

### 1.1 Формулы коротких текстов

**Hook-Insight-CTA** (для welcome после карты):
```
[Факт из карты — конкретный, личный]
[Что это значит для жизни — 1 предложение]
[Открытый шаг → CTA]
```

**BAB — Before-After-Bridge** (для re-engagement):
```
[Как сейчас — узнавание боли]
[Как может быть — конкретная картина]
[Мост: что сделать прямо сейчас]
```

**Open Loop** (для daily, nurture):
```
[Незаконченная мысль, которая цепляет]
[Частичный ответ — ценность]
[Обрыв → "завтра покажу..."]
```

**PAS — Problem-Agitate-Solve** (для CTA-блоков):
```
[Проблема на языке клиента]
[Усиление через конкретику, не страх]
[Решение = следующий шаг в боте]
```

### 1.2 Welcome после карты — Good vs Bad

**ПЛОХО (сейчас):**
```
✨ Анна, твоя натальная карта готова

Атмакарака — ♄ Сатурн
Твоя душа учится терпению. Ты здесь, чтобы через ограничения
найти настоящую свободу.

☀️ Близнецы · 🌙 Весы · ⬆️ Дева

Каждый день — персональный прогноз по твоей карте:
какие сферы усилить и когда лучше действовать.
```
Проблемы: "учится терпению" — банально. "Через ограничения найти свободу" — клише.
"Какие сферы усилить" — абстракция. Не цепляет JTBD, не вызывает "это про меня".

**ХОРОШО (цель):**
```
✨ Анна, твоя карта готова

Планета твоей души — ♄ Сатурн.

Ты из тех, кто берёт на себя больше всех — и удивляется,
почему другим всё даётся легче. Это не невезение.
Сатурн даёт медленно, но навсегда.

☀️ Близнецы · 🌙 Весы · ⬆️ Дева

Хочешь увидеть, что планеты говорят тебе сегодня?
```
Почему лучше: конкретное наблюдение ("берёт на себя больше всех") → узнавание.
Рефрейм ("не невезение, а навсегда") → надежда. CTA → конкретный следующий шаг.

### 1.3 ATMAKARAKA_INSIGHTS — принципы

Каждый инсайт по Атмакараке должен содержать 3 элемента:

| # | Элемент | Что делает | Пример (Сатурн) |
|---|---------|-----------|-----------------|
| 1 | **Узнавание** | "Это про меня!" | "Ты из тех, кто берёт на себя больше всех" |
| 2 | **Рефрейм** | Боль → сила | "Это не невезение. Сатурн даёт медленно, но навсегда" |
| 3 | **Крючок** | Желание узнать больше | "И сегодня у него для тебя кое-что важное" |

**Антипаттерны для инсайтов:**
- "Твоя душа учится X" — банально, подходит всем
- "Ты здесь чтобы Y" — пафос без конкретики
- Любая формулировка, которая подходит любой планете — значит она пустая
- Негатив без выхода ("тебе будет трудно")

**Тест качества:** если заменить планету на любую другую и текст всё ещё работает — текст плохой.

### 1.4 CTA — Good vs Bad

**ПЛОХО:**
```
Хочешь узнать, что говорят планеты сегодня?
🔮 Мой прогноз на сегодня
```
Проблема: generic. "Что говорят планеты" = гороскоп из журнала.

**ХОРОШО:**
```
Хочешь увидеть, что Сатурн приготовил тебе на сегодня?
→ Мой день
```
Почему лучше: персонализация (Сатурн, не "планеты"). Глагол "приготовил" — интрига.
Кнопка короче — ниже трение.

### 1.5 Re-engagement — Good vs Bad

**ПЛОХО:**
```
Привет! Давно не заходила. У меня есть для тебя прогноз 🌟
```
Проблема: guilt-trip + generic.

**ХОРОШО (BAB):**
```
Последние 3 дня Венера была в падении — сложное время для отношений.

С завтрашнего дня она набирает силу. Хочешь узнать, что это значит для тебя?
```
Почему лучше: конкретный астро-факт (value-based причина вернуться) + open loop.

### 1.6 Daily forecast intro — Good vs Bad

**ПЛОХО:**
```
Доброе утро! Вот твой персональный прогноз на сегодня ☀️
```
Проблема: пустые слова перед контентом.

**ХОРОШО (варианты чередования):**
```
A: Луна сегодня в Козероге — день для дел, которые ты давно откладывала.
B: Твой Юпитер сейчас в 10-м доме. Это значит...
C: Помнишь, я говорила про Сатурн? Сегодня он кое-что меняет.
```
Почему лучше: сразу факт + интрига. Нет "обёрточных" фраз.
Варианты чередуются — variable reward (не знаешь какой формат будет).

## ЧАСТЬ 2: ДИАЛОГ (Sonnet 4.6, LLM)

Системный промпт Веды (base.py) уже задаёт voice.
Здесь — контекстный слой, который подставляется per-user.
Sonnet сам решает КАК писать. Мы говорим ему КОМУ и ЗАЧЕМ.

### 2.1 Контекстный блок (подставляется в промпт)

```xml
<user_intent>
  Сегмент: {A1|A2|A3|B2} — из /buyer
  JTBD: {одна фраза — главная задача}
  Schwartz: {unaware|problem|solution|product|most_aware}
  Этап: {onboarding|first_day|active|cooling|dormant}
</user_intent>
```

### 2.2 Schwartz-уровни → что бот делает

НЕ шаблон. Направление для Sonnet.

| Уровень | Юзер думает | Бот делает |
|---------|------------|-----------|
| **Unaware** | "Просто попробую" | Дать wow-факт из карты → "это про меня" |
| **Problem-aware** | "У меня проблема, но не знаю причину" | Связать проблему с картой → "вот почему так" |
| **Solution-aware** | "Астрология может помочь, но как?" | Показать конкретный инструмент → daily, разбор сферы |
| **Product-aware** | "Знаю про курс, но дорого/страшно" | Дать опыт ценности → "я уже получаю пользу бесплатно" |
| **Most aware** | "Хочу, но жду момент" | Мягкий nudge через карту → "сейчас хороший период для X" |

### 2.3 Сегмент → приоритеты карты

Источник: `/buyer` playbook (9 520 анкет).

#### Основные сегменты

| Сегмент | Доля | JTBD | Из карты | Избегать | Язык клиента |
|---------|------|------|----------|----------|-------------|
| **A1** "Потерявшая себя" | 40% | Понять себя, найти предназначение | Атмакарака, 10-й дом, Кету | Отношения | "потеряла себя", "день сурка", "загораюсь и бросаю" |
| **A2** "Сложные отношения" | 20% | Понять причину, принять решение | 7-й дом, Венера, Дара-карака | Карьера | "живём как соседи", "кому я нужна в 44" |
| **B1** "Тревожная мама" | 10% | Понять ребёнка, направить | 5-й дом, карта ребёнка | Личные проблемы мамы | "закрылся", "чужой человек", "пока не поздно" |
| **B2** "После развода" | 5%+ | Начать заново, финансовая независимость | 1-й дом, транзиты (когда полегчает) | Партнёрство (рана свежая) | "одна с детьми", "циклические отношения", "долги" |

#### Кросс-сегментные темы (часто пересекаются с основными)

| Тема | Доля | JTBD | Из карты | Язык клиента |
|------|------|------|----------|-------------|
| **Деньги / заработок** | 37% мотивация | Закрыть кредиты, уйти из найма, доход на себя | 2-й дом (деньги), 11-й (доход), Юпитер, 10-й (карьера) | "погрязла в кредитах", "единственная рабочая сила", "зависит от мужа финансово" |
| **Выгорание / карьера** | 5% + профессии ниже | Сменить профессию, найти реализацию | 10-й дом, Атмакарака, Даши (когда менять) | "день сурка", "нет реализации", "устала от работы" |
| **Одинокие** | 5% | Найти партнёра, понять почему одна | 7-й дом, Венера, Упапада-лагна | "5 лет одна", "мимолётные встречи с женатыми" |

#### Профессии (определяют карьерно-финансовую боль)

| Профессия | Доля | Скрытая боль | Что из карты |
|-----------|------|-------------|-------------|
| Декрет / домохозяйка | 16% | Зависимость от мужа, "отстала от жизни" | 10-й дом (реализация), 2-й (собственный доход) |
| Бухгалтер / финансы | 8% | Устала от рутины, хочет уйти из найма | Атмакарака (что душе нужно), Даши |
| Офис / менеджмент | 8% | "День сурка", нет реализации | 10-й дом, Раху (амбиции) |
| Образование | 7% | Выгорание, ищет новое | Кету (таланты прошлого), 9-й дом (учительство) |
| Бьюти | 6% | Хочет масштабировать | Венера (красота), 11-й дом (доход) |
| Медицина | 6% | Выгорание, тяжёлая работа | 12-й дом (служение), Кету |
| ИП | 5% | Новый источник дохода | 11-й дом, Юпитер (расширение) |

> **Ключевой инсайт:** 37% мотивированы деньгами/заработком — это КРОСС-СЕГМЕНТНАЯ тема. A1 хочет "найти себя" → но часто подразумевает "и зарабатывать на этом". B2 хочет "начать заново" → "и стать финансово независимой". Бот должен вплетать 2-й/10-й/11-й дома не только для "карьерных" сегментов.

### 2.4 Антипаттерны диалога (добавить в self_check)

Бот НЕ должен:

| # | Антипаттерн | Пример | Почему плохо |
|---|------------|--------|-------------|
| 1 | **Вода** — фраза без информации | "Астрология помогает лучше понять себя" | Подходит всем = никому |
| 2 | **Лекция** — объяснение термина | "Атмакарака — это планета с наибольшим градусом, она отвечает за..." | Человек не спрашивал определение |
| 3 | **Универсальный комплимент** | "Ты сильная и глубокая натура" | Подходит любому знаку |
| 4 | **Страх без выхода** | "Сатурн в 7-м — отношения будут сложными" | Приговор, нет рефрейма |
| 5 | **Перегруз** | "Луна в Козероге, Солнце в Близнецах, Раху в 5-м..." | Много данных = каша |
| 6 | **Дежурный вопрос** | "А как ты себя чувствуешь?" | Терапевт, не подруга |
| 7 | **Копирайтерский штамп** | "Готова узнать секрет своей карты?" | Манипуляция, чувствуется фальшь |

### 2.5 Паттерны сильного текста

Бот ДОЛЖЕН:

| # | Паттерн | Пример | Почему работает |
|---|---------|--------|----------------|
| 1 | **Конкретное наблюдение** | "С твоей Луной в Козероге — тебе важно чувствовать что день прошёл не зря" | Узнавание: "откуда она знает?" |
| 2 | **Рефрейм боли** | "Сатурн не наказывает. Он просто не даёт срезать путь" | Та же проблема, но с другой стороны |
| 3 | **Конкретика вместо абстракции** | "Ближайшие 2 недели Юпитер усиливает твой 10-й дом — хорошо для рабочих решений" | Можно проверить, можно применить |
| 4 | **Одна мысль** | Одно наблюдение на сообщение | Легко усвоить, хочется ещё |
| 5 | **Открытый конец** | "...и в четверг кое-что сдвинется. Напомню" | Open loop → возвращение |
| 6 | **Язык клиента** | "Знакомо ощущение — делаешь всё правильно, а результата нет?" | Из /buyer, не из учебника |

## ЧАСТЬ 3: МАТРИЦА JTBD × АТМАКАРАКА

Мост между астро-данными и текстом.

> **СТАТУС:** Темы и боли — из ведической традиции (Парашара, Джаймини).
> Hooks и рефреймы — **ЧЕРНОВИКИ**, требуют A/B тестирования на реальных юзерах.
> Язык болей привязан к /buyer (9 520 анкет), но привязка "планета → сегмент"
> — гипотеза, не факт. Не хардкодить без проверки.

### Матрица

| АК | Тема (традиция) | Какой сегмент чаще резонирует | Боль на языке клиента | Рефрейм | Hook (черновик) |
|----|-----------------|-------------------------------|----------------------|---------|----------------|
| ☀️ Солнце | Самовыражение, лидерство | A1 "потерявшая себя" | "не знаю куда себя деть", "день сурка" | Тебе нужна не рутина, а место где ты — это ты | "Ты из тех, кого замечают — даже когда ты молчишь" |
| 🌙 Луна | Эмоции, забота | B1 "тревожная мама", A2 | "устала отдавать", "пустота внутри", "нет сил" | Чувствительность — не слабость, а то, что делает тебя незаменимой | "Ты чувствуешь других лучше чем себя — и это не случайность" |
| ♂️ Марс | Действие, воля | Выгорание/карьера | "загораюсь и бросаю", "перегорела", "бросаю на полпути" | Энергия есть — ей нужно правильное русло | "Ты не ленивая. Просто тебе физически невозможно делать то, во что не веришь" |
| ☿ Меркурий | Ум, общение | A1, Бухгалтер/Офис | "слишком много в голове", "100 курсов, ничего не помогает" | Ум — инструмент, не враг | "У тебя в голове 10 вкладок — и ни одна не закрывается. Знакомо?" |
| ♃ Юпитер | Смысл, мудрость | A1 "предназначение" | "ищу смысл", "пробовала разное — всё не то" | Ты ищешь не занятие — ты ищешь правду о себе | "Тебе мало просто жить — тебе нужно понимать зачем" |
| ♀ Венера | Любовь, гармония | A2 "отношения", B2 | "живём как соседи", "устала стараться в одну сторону" | Ты ищешь не идеал — ты ищешь настоящее | "Ты отдаёшь в отношениях больше, чем получаешь — и устала от этого" |
| ♄ Сатурн | Терпение, структура | B2 "после развода", Деньги | "всё через усилие", "кредиты", "единственная рабочая сила" | Даёт медленно, но навсегда | "Ты берёшь на себя больше всех — и удивляешься, почему другим легче" |
| Раху | Амбиции, новое | ИП, Карьера | "хочу уйти из найма", "хочу больше", "не хватает" | Жажда нового — не жадность, а поиск себя | "Тебе всегда мало — не потому что жадная, а потому что ещё не нашла своё" |
| Кету | Отпускание, интуиция | Психологи, Духовность | "отстранённость", "не от мира сего" | Ты уже знаешь ответы — осталось им доверять | "Иногда кажется, что ты не отсюда. И это правда — но в хорошем смысле" |

### Как использовать

**Для шаблонов (onboarding_insights.py):**
Hook из матрицы → замена generic-фраз. Рефрейм → вторая строка инсайта.
Но: тестировать! Если открываемость не растёт — hook не работает.

**Для LLM (контекстный слой):**
НЕ подставлять hook буквально. Дать направление:
```xml
<atmakaraka_context>
  Планета: Сатурн
  Тема: терпение, структура, ответственность
  Боль клиента: "всё через усилие", "кредиты", "одна тащит"
  Рефрейм: даёт медленно, но навсегда
  Тон: конкретное наблюдение → "много берёт на себя"
</atmakaraka_context>
```

**Ограничения матрицы:**
- Привязка "планета → сегмент" — гипотеза. У A1 может быть любая АК.
- Hooks написаны, не проверены. Нужен A/B тест на реальных юзерах.
- Язык болей — из /buyer, но конкретные фразы могут не работать для всех АК.

## ЧАСТЬ 4: ЧЕКЛИСТ КАЧЕСТВА ТЕКСТА

Для проверки ЛЮБОГО текста бота — и шаблонов, и LLM-выхода.

### Тест на воду
- [ ] Убрать предложение → смысл потерялся? Если нет — предложение лишнее.
- [ ] Заменить имя на другое → текст всё ещё работает? Если да — мало персонализации.
- [ ] Заменить планету на другую → текст всё ещё работает? Если да — текст пустой.

### Тест на узнавание
- [ ] Есть конкретное наблюдение из карты? (не определение термина)
- [ ] Человек может сказать "это про меня"? (не "интересно")
- [ ] Используется язык клиента, а не язык астрологии?

### Тест на надежду
- [ ] Если есть проблема — есть рефрейм или выход?
- [ ] Нет ли "приговора" без шага вперёд?
- [ ] Финал сообщения открывает следующий шаг (не закрывает точкой)?

### Тест на трение
- [ ] CTA в одно действие? (одна кнопка, один клик)
- [ ] Текст до CTA ≤ 5 предложений?
- [ ] Кнопка ≤ 4 слова?
