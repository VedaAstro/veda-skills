## КРИТИЧЕСКИЕ ЮРИДИЧЕСКИЕ РИСКИ В ДОКУМЕНТАХ

### 1. Оговорка «без ЗоЗПП» в офертах — НЕМЕДЛЕННО ИСПРАВИТЬ

В oferta_1022 (и вероятно других):
> «...исключающее применение закона о защите прав потребителей»

**Статус по праву:** НИЧТОЖНО. ВС РФ, определение N72-КГ23-1-К8 (2023):
потребитель не может отказаться от прав по Закону о ЗоЗПП. Такое условие суд признаёт недействительным. Хуже — наличие такого пункта сигнализирует суду о недобросовестности компании, что увеличивает размер компенсации.

**Затронуто:** oferta_1022 используется в 7+ продуктах (Ректификация, НейроАстролог 1/2 ступень, Активация Денежного Магнита, Тест-драйв и др.)

### 2. Оферта «Освобождения» = оферта «Панчанги»

Продукт «Освобождение» использует `offerta_panchanga` — оферту другого продукта. Если предмет договора не совпадает — это не публичная оферта для этого продукта.

### 3. Оферты на Google Docs (2 продукта)

- ЛОТЫ в честь дня рождения: Google Docs оферта
- Ведический астролог (юбилей): 2 Google Docs оферты (ДО + ДПО)

Риск: документ может стать недоступным, Google может заблокировать. Суды принимают с оговорками.
