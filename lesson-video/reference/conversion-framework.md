# Конверсионная конструкция — lesson-video + PDF

## Враг = неосознанное проживание кармических задач

1. Задача не решена → ситуация повторяется, каждый раз жёстче
2. Привычные усилия УСУГУБЛЯЮТ (идут против задачи)
3. Чужой путь = яд (Юпитер во 2-м → расширяйся, Сатурн во 2-м → дисциплина)
4. Цикл: симптом → неверное объяснение → действие не туда → вина → потеря веры → повторение (хуже)

Кармическая задача = центральная ось всей воронки (уроки + PDF + диагностика).
Одновременно создаёт: Барнум (узнавание), внешний локус (не вина), потенциал (решение = ресурс),
открытую петлю (задача видна, КАК решить = на разборе).

## Алгоритм эмоционального сдвига

Порядок подачи материала (в уроке — по ключам, в PDF — по секциям) создаёт цепочку:

```
1. Любопытство   → "что в моей карте?"
2. Узнавание     → "откуда они знают?!" (Барнум)
3. Удивление     → "у меня есть ТАКОЕ?!" (потенциал)
4. Осознание     → "вот ПОЧЕМУ ничего не менялось" (кармическая задача)
5. Тревога       → "а я это не использую / иду не туда"
6. Надежда       → "но решение существует" (карта показывает путь)
7. Фрустрация    → "но тут не написано / не сказано КАК" (открытая петля)
8. Решимость     → "пойду на разбор / следующий урок" (CTA)
```

В уроке шаги 1-6 = ключи + Веда, шаг 7 = тизер следующего урока, шаг 8 = домашка + seed CTA.
В PDF шаги 1-6 = секции 2-7, шаг 7 = секция 8 «что видно и что скрыто», шаг 8 = секции 9-11.

## Барнум-формула

Механизм «откуда они знают»: позиция карты → узнаваемая бытовая ситуация.

```
Позиция карты → типичная ситуация → мягкое узнавание
```

Работает потому что астро-позиции коррелируют с широкими жизненными паттернами,
а конкретная привязка к карте создаёт иллюзию точного попадания.

Примеры:
- Сатурн(R) в 11-м: «Доход растёт медленно, но стабильно. Крупные суммы приходят с задержкой.»
- Венера в Козероге, 7-й: «Лёгкие отношения не цепляют — нужна глубина. Но глубина оборачивается тяжестью.»
- Раху в 12-м: «Ощущение что вы живёте не свою жизнь. Уходите в себя, когда нужно действовать.»

Формула «жизнь → метафора → термин» (из examples-and-voice.md) усиливает Барнум:
термин приходит ПОСЛЕДНИМ и звучит как «диагноз» — укрепляет доверие.

## Снятие вины: через контент, не через слова

**Принцип**: читатель приходит к выводу «это не моя вина» САМ, через факты карты.

ЗАПРЕЩЕНО: «Это не ваша вина», «вы не виноваты», любые прямые утешения.
Банальные утешения = ощущение манипуляции. Контент = ощущение открытия.

Механика:
1. Описать позицию карты (факт)
2. Описать цикл, который она создаёт (механизм)
3. Показать: привычное направление усилий НЕ совпадает с задачей
4. Читатель узнаёт → делает вывод сам

Применяется: в уроках (ключ #2 = «не я плохая — задача»), в PDF (секции 4-5).

## Оружие = навык читать задачи через карту

| Диагностика (GPS) | Курс (навигатор) |
|-------------------|-----------------|
| Точка: «вы здесь, вам туда» | Реальное время: «через 200м направо» |
| Один раз, текущий момент | Всегда, периоды меняются каждые 2-3 года |
| Точка входа | Навык на всю жизнь |

## 3 веры (Хормози) через 3 урока

| Урок | Вера | Через что |
|------|------|-----------|
| #1 | В метод | Карта показала КОНКРЕТНУЮ задачу → WOW |
| #2 | В себя | Не я плохая — решала не ту задачу |
| #3 | В условия | Школа даёт навигатор, не просто координату |

## Мост к диагностике

1. Ключ показал ЧТО (диагноз)
2. Зритель: «а что МНЕ делать?»
3. Ирина: «Урок — общая картина. Конкретные действия — разбор»
4. Что получите: план для периода, канал, коррекция блоков

## Обоснование бесплатного — 3 фрейма

**Фрейм 1 — Ценность инструмента:** «Карта — мощный инструмент. Без проводника остаётся картинкой.»
**Фрейм 2 — Миссия Ирины:** «Моя миссия — чтобы каждая знала свой путь. По-настоящему.»
**Фрейм 3 — Свобода выбора:** «Дальше — решение за вами. Путь свой вы будете знать.»
**Фрейм 4 (урок 3, мягко):** «Обучение предлагаем не всем — только кому подходит по карте.»

## Нарастающая сила CTA

```
Урок 1: лёгкий seed (1 упоминание в последнем ключе)
Урок 2: средний seed (1 упоминание)
Урок 3: полный CTA (все 3 фрейма + гарантия)
```

## Риторика Бредемайера

**5 правил Шнайдера:** простота, выделение, фокусировка, повторение (3 раза разными словами), контраст.

**Каскад КПД (возврат внимания каждые 2-3 мин):**
- К (коснуться) — персональный факт из карты
- П (повернуть) — переключить канал (Ирина → Веда → карта)
- Д (договорить) — инсайт с персональными данными

**Вопросы по типам:**
| Тип | Где | Пример |
|-----|-----|--------|
| Сократовские | Домашка | «Совпадает ли канал с тем как зарабатываете?» |
| Мотивационные | Предутилизация | «Вы же чувствуете, что масштаб ваш?» |
| Контролирующие | После ключа | «Откликается? Есть такое?» |
| Риторические | Усиление | «Продолжать экономить, когда карта говорит: расширяйся?» |

**Трёхканальная аргументация:** визуальный («посмотрите», «видите?») + аудиальный («звучит знакомо?») + кинестетический («чувствуете?», метафоры сейф/замок/ключ). Чередовать.
